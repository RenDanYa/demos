# 三个时间记录分析脚本功能说明文档

| 属性           | 脚本1: `1parse_md_file.py`                                    | 脚本2: `2parse_time_record_line.py`                       | 脚本3: `3parse_time_record_line_to_md.py`           |
| ------------ | ----------------------------------------------------------- | ------------------------------------------------------- | ------------------------------------------------- |
| **输入文件**     | 多个Markdown日记文件（通过对话框选择）                                     | 多个Markdown日记文件（通过对话框选择）                                 | 多个Markdown日记文件（通过对话框选择）                           |
| **输出文件**     | `时间记录统计.xlsx`（Excel文件）                                      | `工作时间统计.xlsx`（Excel文件）                                  | `工作时间统计.md`（Markdown文件）                           |
| **提取的数据**    | 1. Frontmatter中的数值字段（运动时间、体重、打字速度等）<br>2. 时间记录部分的时间段（按标签分类） | 1. 日期和星期几<br>2. 按时间段（上午、下午、晚上、深夜）统计的工作时长<br>3. 最晚深夜工作时间 | 1. 日期和星期几<br>2. 按时间段（上午、下午、晚上、深夜）统计的工作时长          |
| **时间记录格式要求** | 时间记录部分需包含标签：`#时间记录/播客`、`#时间记录/放松模式`、`#时间记录/运动`、`#时间记录/阅读`   | 时间记录行需包含工作时间标签，排除`#时间记录/休息`、`#时间记录/运动`、`#时间记录/机动`       | 时间记录行需包含工作时间标签，排除`#时间记录/休息`、`#时间记录/运动`、`#时间记录/机动` |

- 4parse_frontmatter.py
	- 解析状态，查看时长<8,8-10,10-12,>12的天数分布如何
- 5parse_read_time_range.py
	- 每本书的阅读和大纲整理分别多长时间
