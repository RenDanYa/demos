- 全球顶尖学者、CEO参与，讨论全球经济问题

- 会议流程：见面会 → 小组讨论 → 聚餐

- 正式会议 → 注意力全在演讲内容  
- 宴会 → 进食时不便交谈，只能与邻座交流  
- **见面会** → 最佳机会，尤其靠近吧台（每个人都会来喝点东西）  

### 5.3 准备策略

- 提前几个月认识会议主办方，获取参会者信息（虽不详细但足够）  
- 助理协助收集资料  
- **目标**：不仅有幸运邂逅，更能进行深层次对话  

### 5.4 案例一：与约翰·佩珀（耶鲁校友、宝洁前CEO）的交流

## 2. 成功案例：如何联系索尼公司

**背景**：杰夫·阿诺德（WebMD创始人）有一项新技术，想联系索尼公司但找不到决策人。

**步骤拆解**：

1. **寻找间接关系**：
   - 作者认识索尼CEO霍华德·斯金格，但未直接等待回复。
   - 搜索为索尼服务的公司 → 找到BrandBuzz公司（索尼的高级广告代理）。

2. **利用已有朋友**：
   - BrandBuzz的CEO约翰·帕提拉是作者的好朋友。

3. **给朋友提供双重价值**：
   - 介绍杰夫给约翰 → 新商机。
   - 向索尼提供新技术商机 → 促进合作关系。

4. **创造“巧合”**：
   - 请约翰提前发介绍邮件，打电话时抄送。
   - 让对方感觉一切像是巧合。

5. **坚持跟进**：
   - 邮件无回复 → 放下傲气，持续打电话或发邮件。
   - **最佳通话时间**：早上8点整或下午6:30（避开朝九晚五压力）。
   - 留言技巧：“我对见面非常激动，理解您很忙，期待回复。”

6. **最终成功**：在下午6点直拨直线电话，塞尔日亲自接听 → 一周后会面。

## 4. 详细策略与具体法则

### 4.1 寻找他人的推荐

**核心问题**：打冷电话时，你极度缺乏信誉。

**解决方式**：
- 提出“是××介绍来的”（私人或组织推荐）→ 瞬间改变对方态度。
- 即使不在大公司，也可从以下圈子寻找联系：
  - 朋友、家庭、客户、邻居
  - 同学、工作伙伴、教会会友

**现代工具**：
- **领英**：快速找到共同联系（平均分隔仅3人）。
- **Facebook**：分隔缩小到4.74人。
- **谷歌**：搜索对方学校、兴趣、董事职位、慈善活动、运动爱好等。

> 📌 只要通过3~5个人，就能联系到世界上几乎任何人。

### 4.2 表明你的价值

**核心原则**：销售的本质是解决他人的问题。

**具体做法**：
- 提前研究对方公司、行业、即将发布的产品。
- 了解对方的痛点与目标（如案例中塞尔日准备季度新品发布、假日旺季需求）。
- 让电话内容具有针对性，避免“电话冷场”。

> ✅ 记住：这一切都跟对方有关——你能为他们做什么？

### 4.3 说话少一些，对话多一些

**要点**：
- 表达急切与人方便的语气。
- 不要以“我们应该找个时间见面”结束。改用：
  - “我下周会去市里，周二一起吃个饭怎么样？”
- 不要单向推销，要留出时间让对方回应（“嗯哼”“没错”）。
- **主要目的不是完成交易，而是争取一次会面**，以便面对面深入交流。

> 🎯 控制时间，确保下一次能在办公室或餐桌上继续交谈。

  - 重要的是**聚在一起做什么**，而不是多久见一面

#### 1.3 激情的力量：如何让热爱成为社交催化剂

- **个人兴趣是最好的社交切入点**
  - 你最擅长的活动往往是你最感兴趣的领域
  - 作者举例：对食物和锻炼的热爱帮助他结识许多志同道合的朋友
  - 其他例子：集邮、棒球卡、政治话题、跳伞等

- **回馈社会的力量**
  - 作者组织年度之旅，前往安提瓜和危地马拉帮助“孤儿村”的孩子
  - 通过分享故事、照片、慈善捐赠，让更多人参与其中
  - 这种经历让人成为更优秀的领导、同事、父母

- **众筹平台的力量**
  - Kickstarter案例：前四年有370万民众为项目筹集5.45亿美元
  - 社交媒体让热情传播，人们愿意为自己的兴趣“砸钱”

- **激情的传染性**
  - 激情会让人放下防备，激发他人好奇心
  - 在生意场上，分享激情非常重要

#### 1.4 办公室外的社交：非传统场所的价值

- **为什么离开办公室更好**
  - 活动让人自然放松（如晚餐有酒相伴）
  - 更能看清对方的真实表现
  - 共同参与双方热爱的活动，能学到更多东西

- **具体场所案例**
  - **基督教青年会（YMCA）**：一位银行副总裁在早晨五六点的锻炼中，与企业家、客户交流，回答投资与贷款问题
  - **教堂**：作者带朋友去洛杉矶圣阿加莎教堂（非裔美国人为主的西班牙风格教堂），“另类仪式”中相互拥抱长达10分钟，被视为高度私人化的礼物

#### 1.5 打破公私界限：真诚与情感外露的价值

- **旧式 vs 新式商业观点**
  - 旧式：情感外露被视为弱点
  - 新式：情感是联系双方的纽带，关系越紧密，生意越成功

- **邦妮·蒂格瑞斯（Bonnie Digrius）的案例**
  - 每年给联系人发内部信件，写下个人故事（如父亲的去世如何改变生活）
  - 结果：收到大量回信，人们分享类似经历
  - 社交圈扩展至全美，获得成百上千人的信任和敬佩

#### 1.6 实用活动清单：保持联系的具体方式

| 活动类型 | 具体做法 | 特点 |
|---------|----------|------|
| **15分钟加咖啡** | 室外进行，轻松省时 | 作者最受欢迎的建议，需清楚说明对对方的价值 |
| **会议** | 在参会时联系当地认识的人，邀请主题演讲或共进晚餐 | 利用出差机会 |
| **分享锻炼或爱好** | 高尔夫、国际象棋、集邮、图书俱乐部等 | 共同兴趣破冰 |
| **简餐** | 早餐、午餐、下班喝一杯、晚餐 | 食物打破僵局 |
| **特别活动** | 剧院之夜、签书会、演唱会 | 让活动因邀请而更特别 |
| **在家招待客人** | 小型晚宴，只邀请1-2个不熟悉的人 | 让客人离开时认识新朋友 |
| **当志愿者** | 5-10人服务他人，慈善行走，打包午餐送餐 | 共同服务他人 |

#### 1.7 重要提醒：平衡关系，不忘身边人

- 需要腾出时间陪伴家人和朋友
- 确保不要忽视生活中最重要的关系
- 当每一天充满激情和有趣的朋友时，联系他人不再是挑战或负担

### 第二部分：要么跟进，要么失败——后续跟进的艺术

#### 2.1 为什么跟进至关重要：对抗遗忘曲线

- **问题现状**
  - 快节奏、数字化世界，信息过载
  - 大脑只能记住一小部分新认识的人
  - 多数人不得不淡忘新信息

- **核心观点**
  - 认识新朋友后，必须多做事情确保不被遗忘
  - **做好后续跟进能甩开95%的竞争者**
  - 坚持跟进是社交网络的“锤子和钉子”

- **生动案例**
  - 作者在兄弟会演讲后，给100多人递了名片
  - 第二天早上收件箱中，**第一个发邮件**的小学弟被他记得最清楚
  - 印象最深的礼物：花几美金表达情意的感谢信、邮件、贺卡

#### 2.2 跟进的最佳时机与方式

- **时间窗口**
  - 见面结束后 **12小时到一天内** 跟进
  - 飞机上遇见 → 当天晚些时候发邮件
  - 鸡尾酒聚会遇见 → 第二天早上发邮件

- **邮件内容要点**
  - “很高兴能遇见您！我们一定要保持联系！”
  - **提及当时对话中的特别话题**（共同爱好或商业利益），帮助对方想起你是谁

#### 2.3 善用数字工具：邮件、领英、Facebook

- **领英邀请**：跟进邮件的补充
- **Facebook邀请**：需谨慎，说明“如果您对新结交的朋友使用Facebook，我很乐意联系；如果您介意，没关系，我同样会保持联系”
- **联系人管理系统**
  - 会面结束时，将新朋友姓名、邮箱录入联系人名单
  - 加入工作计划，**一个月后再次发邮件**，仅仅为了保持联系

> 反问：如果不打算努力把这些人变成生活的一部分，何苦去认识新朋友？

#### 2.4 书面确认承诺：詹姆斯·克拉克的方法

- **做法**
  - 每次会面后重申每个人所做的承诺
  - 第二次跟进会议时询问大家
  - 留下书面信息，例如：
    > “昨天午餐时跟您聊天非常愉快！我想继续跟您聊聊我们讨论过的想法……下次我来时，希望您可以抽空跟我见5-10分钟。”

- **效果**
  - 对方十有八九会回信同意
  - 再次见面时，你可“手写”凭证证明对方曾承诺见面

- **重要原则**
  - 不要提醒对方能为你做什么
  - **专注于你能为他们做什么**——这给了他们继续见面的理由

#### 2.5 超越邮件：手写感谢信的力量

- **为什么手写信有效**
  - 极其吸引注意力（你上次收到手写信是什么时候？）
  - 寄到私人地址的信一定会被打开

- **感谢信的作用**
  - 提供继续联系的机会
  - 强化双方关系
  - 营造示好氛围
  - 可提及前次未提的重要信息，强调渴望再次见面和主动帮助

#### 2.6 后续跟进的10条黄金提示

| 序号 | 提示要点 | 说明 |
|------|----------|------|
| 1 | **时常表达感激之情** | 真诚感谢对方的时间与分享 |
| 2 | **包含有趣内容** | 一个笑话或幽默时刻 |
| 3 | **再次确认承诺** | 说明如何发展关系 |
| 4 | **简明扼要** | 不啰嗦，直击重点 |
| 5 | **书写记名感谢信** | 手写更佳 |
| 6 | **结合电子邮件+传统邮寄** | 增加私下接触机会 |
| 7 | **通过社交媒体发送联系请求** | 在邮件之后进行 |
| 8 | **时机很重要** | 会晤或采访结束后尽快发送 |
| 9 | **不要等到假日** | 平时及时联系更让人记住 |
| 10 | **感谢中间人** | 与充当纽带的人保持联系，告知进展并表达感恩 |

#### 2.7 养成习惯：让跟进成为自然反应

- **最终目标**
  - 将后续跟进内化为一种习惯
  - 一旦养成，疯狂记住他人姓名（以及被他人记住）的日子一去不复返
- **核心理念**：坚持做好跟进，你就能脱颖而出

## 1. 核心理念：会议的本质与价值

- **会议的主要意义只有一个**：结识志趣相投的人，帮助你完成工作目标。
- **不是获取知识的地方**：会议很少教给你真正的知识，而是拓宽社交圈、促成生意的绝佳场所。
- **社交优先于销售**：
  - 旧销售模式：80%时间推销，20%发展关系。
  - 新销售模式：80%时间建立社交圈，20%发展信任关系。
- **投资回报分析**：会前评估投入（时间、金钱）与预期回报，决定是否参会。

> 💡 **核心观点**：不要只做“与会者”，要做“突击先锋”——主动创造机会，而不是被动等待。

## 2. 会前准备：像作战一样规划会议

| 准备事项 | 具体做法 |
|---------|----------|
| 明确目标 | 列出你最想见的人（3-5位核心人物） |
| 研究背景 | 了解目标人物的身份、经历、成就、爱好、公司挑战 |
| 制定策略 | 确定如何接近、何时何地见面、聊什么内容 |
| 准备资料 | 将名单折好放进口袋，记录聊天要点和后续计划 |
| 资源盘点 | 你有哪些可贡献的资源（时间、创意、人脉） |

> 📌 **德勤案例亮点**：建立追踪系统，每位员工分配两位目标人物（一位潜在客户、一位可带来利益回报的人），会前提供详细资料和“办法清单”，会后报告进展。

## 3. 六大实战法则

### 3.1 援助会议组织者（最好成为组织者）

- **为什么**：成为“内部成员”可获得与会者名单、热点信息、VIP聚会入场资格。
- **怎么做**：
  - 提前几个月联系负责人：“我愿意贡献时间、创意或人脉，帮助办成最成功的会议。”
  - 了解谁参会、谁发言、社交论坛的最佳形式。
- **成果**：与迈克尔·哈默建立友谊，德勤业务空前增长。

### 3.2 与其听，不如说

- **演讲的价值**：
  - 表现自我、事业和想法的最简单高效方式。
  - 研究证明：钱包大小与演讲次数成正比。
  - 一旦站上台，你就获得公信力，成为听众眼里的名人。
- **如何成为发言人**：
  - 从小范围开始练习（如国际演讲协会）。
  - 帮助组织活动，换取演讲机会。
  - 即使不是发言人，也要**第一个举手提问**，介绍自己，提出深刻问题。
- **内向者对策**（苏珊·凯恩经验）：
  - 加入演讲俱乐部，低风险练习。
  - 请培训师反复训练，降低音调，学会腹式呼吸。
  - 带上让自己放松的物品（如装满书的包）。

### 3.3 游击战术：组织一场会中会

- **不拘泥于官方议程**：自己安排晚宴、非正式讨论、户外活动。
- **操作方法**：
  - 提前物色附近好餐馆，发出邀请。
  - 给接待酒店发传真，告知客人入住当晚有晚宴。
  - 邀请演讲者参加，提升晚宴吸引力。
- **其他形式**：私人滑雪之旅、博物馆参观、家中聚会（如米尔肯研究院全球会议前在家办晚宴）。
- **核心**：主动帮助他人搭桥牵线，让自己成为社交焦点。

### 3.4 以重要人物为核心

- **谁是“重要人物”**：会议主办方、发言人、知名品牌CEO、行业专家。
- **战术**：
  - 查看他们的会议日程，确保你能参加。
  - 提前到达演讲会场，站在入口或签名处。
  - **在发言人上台前**与他们聊天（早餐时他们最放松，还没披上“名人外套”）。
  - 请已成为朋友的主办方帮你引荐。

### 3.5 成为“消息通”

- **定义**：掌握关键信息并愿意分享的人。
- **需要知道的信息**：
  - 商业动态
  - 当地最佳饭店
  - 私人聚会信息
- **效果**：你是消息通，别人就乐于主动找你。

### 3.6 成为通讯员

- **实时报道**：使用推特等社交媒体，跟踪会议动态和主题标签（Hashtag）。
- **行动清单**：
  - 会前确认重要人物推了哪些参会者。
  - 会议期间给遇到的朋友拍照，发推或存进Facebook。
  - 整理成故事或照片集。
- **时效要求**：回家飞机上迅速整理，趁大家记忆犹新时发邮件或分享。

## 4. 关键技巧：深度攀谈

- **“攀谈”本质**：碰巧偶遇目标人物，花一两分钟聊天，目的是**发出再次联系的邀请**。
- **“深度攀谈”要点**：
  - 让对方感觉高效且有意义。
  - **不要推销产品**，而是让对方喜欢你、信任你。
  - 问题围绕对方的想法和困难展开。
- **成功案例**（与巴里·迪勒偶遇）：
  - 站在他必经之地。
  - 亮出对方熟悉且信任的名字（老板巴里·斯特恩利希特）。
  - 表达钦佩 + 主动提出想为对方做什么。
  - 结果：获得进入迪勒公司的工作机会。

## 5. 目标锁定：确定你要“邂逅”的人

- **写下来**：人名 + 聊天要点 + 后续联系计划。
- **找到他们常驻的位置**：请主办方告知目标人物所在的区域、座位（大多数人全程坐同一位置）。
- **提前踩点**：弄清楚他们上台的时间、路线，守在必经之路。

## 6. 休息时段就是工作时段

| 原则 | 具体做法 |
|------|----------|
| 站对地方 | 人们最常聚集的地方：餐桌、吧台、接待处 |
| 全神贯注 | 像基辛格那样：进入房间，走向右边，环视四周，让他人看到你 |
| 衣着精干 | 不要低估穿着得体的重要性 |
| 主动出击 | 把房间当作运动场，开始你的“偶遇”之旅 |

## 7. 后续跟进：善始善终

- **重要程度**：跟进 → 再跟进 → 继续跟进。
- **行动建议**：
  - 会议期间坐在后排，给休息时段遇到的人写邮件。
  - **每位聊过天的人都需收到一封邮件**，确认再次会谈事宜。
  - 即使没见到发言人，也可以写信。
- **邮件范例**（发给卡拉）：
  - 提及会议中的有趣细节（龙舌兰酒）。
  - 想继续讨论的话题（营销策略、忠诚策略）。
  - 询问方便通话的时间。
  - 真诚赞美对方（“您真是一位伟大的演讲家”）。

> ⚠️ **警告**：不要丢掉名片。丢了名片 = 丢了机会。

## 8. 焦点在大众，不在演讲者

- 很少觉得会议内容有特殊功效。
- 即使演讲很精彩，**谈论的焦点仍然是大众**（即与会者之间的互动和连接）。
- 例外：如迈克尔·哈默这样能把讲话变成生活课堂的演讲者。

## 9. 拒绝做的五种人

| 类型 | 特征 | 问题 |
|------|------|------|
| 不善言辞之人 | 握手无力，站墙角，不出风头 | 认为自己只是来看演讲的 |
| 如影随形之人 | 紧紧跟随第一个遇到的人 | 恐惧独立，错失多样化人脉 |
| 追踪名人之人 | 花一小时排队只为了跟名人握手 | 得不到真正的关系，忽略了屋内其他有价值的人 |
| 虚情假意之人 | 目光虚伪，缺乏真诚 | 立即招致反感；应像克林顿那样，30秒内充满温暖和真诚 |
| 发散或收集名片之人 | 狂热发名片或收集名片 | 除非得到再次会面的承诺，否则名片无用；沾沾自喜于数量是无效功 |

## 10. 经典案例：苏珊·凯恩的TED演讲

- **背景**：内向者，却做了TED历史上观看最多的演讲之一。
- **准备过程（“危险演讲的一年”）**：
  1. 加入国际演讲协会，定期在陌生人面前练习。
  2. TED培训师用2小时教她降低音调、从腹部呼吸。
  3. 演讲前一周，与第二位培训师整天练习，不断反馈。
  4. 上台时手提装满喜欢书籍的包，放松心情。

## 1. 核心概念：超级引荐人与弱联结

### 1.1 超级引荐人的定义与价值

- **定义**：那些似乎认识每一个人，而且每一个人都认识他的人。
- **分布领域**：猎头、说客、筹款人、政客、记者、公关专家等。
- **类比**：就像篮球界的迈克尔·乔丹、高尔夫界的泰格·伍兹一样重要。
- **价值**：一旦与一位超级引荐人成为朋友，你只需通过两个人便可认识成百上千的人。

### 1.2 弱联结的力量（格兰诺维特研究）

> 社会学家马克·格兰诺维特于1974年发起经典研究“找工作”（马萨诸塞州牛顿市）

| 求职方式 | 占比 |
|---------|------|
| 通过朋友介绍 | 56% |
| 传统方式（报纸、猎头等） | 19% |
| 招聘者直接指定 | 10% |

**关键发现（格拉德威尔总结为“弱联结的力量”）：**

- 真正让人获利的往往是**关系并不密切的熟人**（弱联结）
- 只有 **17%** 的研究对象经常与帮助过他们的引荐人联系
- **55%** 只是偶尔见面
- **28%** 几乎不怎么见面

**为什么弱联结更重要？**

| 强联结（好朋友） | 弱联结（熟人） |
|----------------|---------------|
| 同质化社交圈 | 接触不同世界 |
| 信息高度重叠 | 信息差异大 |
| 获取的新知有限 | 带来大量新鲜知识和机会 |

> 📌 格拉德威尔名言：“混脸熟代表着一种社交力量的来源。你认识的熟人越多，你的力量越强大。”

### 1.3 六度分隔理论与超级引荐人

- **米尔格拉姆实验（1967）**：随机信件平均经过**6次**传递到达目标
- **惊人发现**：大部分信件都经过**同样三个人**传递
- **结论**：如果你想认识更多熟人，多认识“超级引荐人”会大有帮助

### 2. 闲聊的目标与核心原则

  - 唯一重心是挖掘边缘社交的价值。
  - 为粉丝提供重大技术消息，实现双赢。
  - 风险投资家主动款待他，希望他参与创建新公司，因为他掌握了新兴公司和科技动向的信息流。

- **启示**：

## 6. 如何管理边缘社交

### 6.1 把音量调大——克服规模恐惧

- **常见误解**：交太多边缘朋友会影响真正的友谊。
- **事实**：
  - 网络人际关系容易批量管理，亲密关系需要一对一培养。
  - 做对的人既有一小群好朋友，又有一大张多样化的社交网，可获得资源、信息和人力。

- **操作建议**：
  - 扩大规模，但要建立过滤和管理结构。
  - 只关注对自己有价值的人（如行业革新者、经理董事），不关注体育明星、电影明星、政治家。
  - 以 **“人际关系行动计划”** 为指导，把核心目标（50-150人）和更大范围的朋友圈切换交流。
  - 利用社交平台的标签或名单功能分类：
    - “1”类：立即建立交往，线上线下单独交流。
    - “2”“3”类：边缘关系，通过推特、博客、邮件互动。
    - 定期浏览“3”类朋友的信息，获取最佳想法。

### 6.2 强调多样性——激发集体智慧

- **研究结论**（密歇根大学斯科特·E.佩奇）：
  - 多元背景的群体比单一高智商群体表现出更高的集体智慧。
  - 多样性比单个人的脑力更重要。

- **核心观点**：
  - “你之所见决定了你怎样改变。”（威廉·泰勒）
  - 目光聚焦之处决定了你的见识。

- **具体建议**：
  - 审查信息流中的朋友是否在背景、职业、地理位置、年龄、种族上有多样性。
  - 从边缘社交圈中选取重要朋友。
  - 在推特和Facebook上分类：艺术家、企业家、技术专家、杂技演员等。
  - 每周固定时间查看不同类别名单，扩充内容。
  - 研究其他领域的领军人物，借鉴他们的思想解决自己的问题（“借鉴并改变”）。

### 6.3 线下活动是最好的网络过滤装置

- **斯考伯的做法**：
  - 通过现实世界的会议和行业聚会筛选网络社交圈。
  - 直接找不认识的人交谈，了解他们的工作、激情、生活角色。
  - 面对面交流能获得丰富信息（长相、举止、真实感）。

- **最佳实践**：
  - 最出色的线上社交是虚拟与现实来回切换。
  - 线下活动建立初步联系，线上互动维持联络。
  - 与密友每三周聚会一次，保持节奏。

### 6.4 拥抱小圈子里的名人

- **案例：娜塔莉·莫利亚·尼诺**
  - 在纽约女企业家中很有名气，但大众不认识。
  - 创建TED巴纳德学院女性组织，与格洛丽亚·费尔特、芭芭拉·科克兰等大人物合作。
  - 她的工作：把无秩序的事情组织起来，在不同网络社群之外创建共同体。

- **价值**：
  - 做小圈子里的名人，强化与目标人物的关系，同时保持与边缘的联系。
  - 通过人际关系行动计划，介绍圈子里的朋友相互认识，让所有人受益。
  - 找到连接各圈子的方法：邮箱列表、领英群组、Facebook主页、晚宴、研讨会等。
  - 分享信息、创造价值，把联系专家作为自己的工作。

- **重要提醒**：
  - 不要把所有时间花在吸引大名人（如理查德·布兰森）上，概率几乎为零。
  - 领英创始人里德·霍夫曼每天收50封投资请求邮件，从未投资任何一个。
  - **正确做法**：研究明天的大人物，寻找伟大的想法、少见的智慧、新鲜的面孔。
  - 建立联盟，5-10年后你们都能更好地帮助对方。

## 7. 驯服领英

- **作者的错误做法**：

## 详细内容整理

### 1. 引言：内容是在线建立信任的关键载体

- **核心观点**：无论技术多强大、信息多丰富，没有信任，梦想就会受挫。
- 内容（文章、博客、状态更新等）是网上建立信任的载体。
- 人们通过你分享的内容了解你是谁、想要什么、能提供什么。
- 善于管理信息的人能创造最大、最有价值的社交圈。

### 2. 信任的计算方法

> **公式：** 包容度 + 脆弱性 + 责任心 + 公正度 = 信任

- **常见误解**：以为要展示完美、聪明、高效。
- **真实案例**：作者录视频时过度化妆打扮反而不自然；后来在飞机办公室轻松录制，效果更好。
- **结论**：你越真实，人们越相信你传递的信息。边缘保持粗糙，不要打磨光滑。

### 3. 探索最本质的诚实

- **人物**：詹姆斯·阿尔图切尔（金融领域作者）
  - 不完美、失败经历丰富、说话直率。
  - 通过写博客分享失败、恐惧、忧虑，浏览量增长20000%。
  - 热门文章标题示例：《希望我的孩子是瘾君子》《亿万富翁的三个故事》《瑜伽让我丢尽了脸》等。
- **核心观点**：
  - 人们大多在竞争中不诚实，你一旦做到诚实，就会脱颖而出。
  - 诚实使你散发人性之美，吸引他人认真倾听。

### 4. 包容度（Generosity）

#### 4.1 开始谈话前已加入其中
- 案例：加里·维纳查克（Gary Vaynerchuk）
  - 先搜索并加入关于葡萄酒的对话，分享热情、提供帮助，再推广自己的视频。
  - 谦逊、活泼、真实的个性吸引粉丝。

#### 4.2 用有分量的语言说话
- 争夺注意力：必须清晰、高效地介绍自己和关联点。
- 案例：《碳水化合物适度的饮食》无人问津 → 重命名为《迈阿密饮食法》成为文化现象。
- 从读者角度思考，回应他们的需求。

#### 4.3 标题都是高音部分
- 标题要满足“实用”或“有趣”。
- 案例：丹尼尔·平克（Daniel Pink）《全新销售》指出，好标题要么说明用处，要么引发好奇。
- 警告：内容必须符合标题，不要诱骗。

#### 4.4 信息要有支撑
- 避免无聊的“我现在在干什么”信息（如遛狗、吃烤奶酪）。
- 改为分享“你注意了什么东西”：
  - 例：“我在看威尔·法瑞尔的新电影，真有意思，必看！” + 提供链接。
- Facebook提示语从“现在在干什么？”改为“你在想什么？”——更好但仍不够。
- 作者推荐：“你注意了什么东西？”

#### 4.5 条件允许就一起创造
- 案例：米尼翁·福加尔蒂（Mignon Fogarty）——“语法女孩儿”播客
  - 发现同事常犯语法错误 → 每周录三段音频 → 粉丝提问 → 成为iTunes最受欢迎教育资源。
- 邀请别人参与创作（如Kickstarter众筹）会让成果更吸引人。
- 注意：坚持不懈，不辜负他人的努力和热情。

### 5. 脆弱性（Vulnerability）

#### 5.1 试试穿戴异性的服装（混淆私人/职业信息）
- 不要严格区分Facebook（朋友）和领英（业务）。
- 家人和朋友也有兴趣了解你的工作（只要有趣）。

#### 5.2 把轶闻和效用混合起来
- 案例：佩内洛普·特伦克（Penelope Trunk）
  - 先写农场猪、家庭教育等个人轶事，最后给出高价值内容（如如何提升生产力、找更好工作）。

#### 5.3 把失败暴露在公众面前
- 坦诚谈论错误，让人知道你无隐瞒。
- 巨大成功来自一次次失败和不断尝试，而非一帆风顺。

#### 5.4 照相做鬼脸
- 照片比文字更能传达你是谁。
- 适度虚荣可以，但不要让自己毫无特点。
- 领英头像建议：清爽、传神、充满活力。
- 案例：一位作家经纪人用和儿子滑雪的照片做领英头像，赢得信任。
- 案例：克雷格弹吉他吹口琴的头像，成为公司最受欢迎员工。

### 6. 责任心（Accountability）

#### 6.1 做自己的主编
- 制订编辑计划表：发表什么内容、何时发表。
- 创造自己的“特色”（他人做不了的事）。
- 案例：诺亚·斯凯林（Noah Scalín）——“一天一个颅骨”计划，持续一年，最终出版两本书。

#### 6.2 让自己与时俱进
- 虚拟世界适合探索、实验。
- 按理想工作的要求写领英简介，而不是当前工作。
- 作者自己的教训：请营销专家设计邮件活动，但内容与自己的风格相差太远 → 收到负面反馈，感到难受。
- **关键**：实验不能偏离真实的自己太远；生活改变时，要主动更新网络记录。

#### 6.3 对身份信息的责任（几点考虑）
- **新业务、新渠道不代表新价值**：清楚自己是谁、关心什么，真实展现。
- **没有借口**：对任何带有你名字的东西负100%责任。
- **明确处理社交关系的规则，坚持到底**：引用赛斯·高汀（Seth Godin）——承诺就是承诺，不能随意更改。
- **遇到麻烦的时候，直言不讳**：承担责任，寻求帮助，一步步纠正。

### 7. 坦率（Candor）

#### 7.1 说别人不说的话
- 案例：编剧乔希·奥尔森（Josh Olson）
  - 在《乡村之声》发表犀利文章《你手稿有#S%我是不会读的》，骂骂咧咧但真实。
  - 结果：感谢信涌来，见到更多重要人物。
- 引用阿尔图切尔：“千分之一的诚实人，你会与众不同，获得成功、财富、幸福、健康。”
- 每疏远一个人，会吸引另外十个人。

#### 7.2 创造虚拟的即时情境
- 案例：视频会议中，一位女士的猫跳上桌子，她简单介绍后继续讨论，让所有人重新投入。
- 建议：利用视频会议、网聊练习即兴发挥。
- 作者做法：谷歌环聊时在公司办公区中央会议室谈话，允许员工参与。

#### 7.3 摸索，失败，调整，重复
- 最成功的企业家通过实验、试错、巧合前进。
- 阿尔图切尔：散播想法，观察回馈，据此成长。
- 不要只宣扬伟大的里程碑，而是记录点滴变化（遇到问题、小成功、可采纳的建议）。
- 分享从探索到失败的每一个环节，接受前辈指导。

## 9. 如何建立个人品牌

### 9.1 建构自己的个人品牌信息

- 品牌 = 别人听到你名字时脑中的印象。
- 信息来自你的**内涵、独特价值、自我发展**。
- 步骤：
  - 思考：你希望别人想到什么？
  - 写出**90天、1年、3年目标**。
  - 写出你希望别人用来形容你的词。
  - 问最信任的朋友：他们用哪些词形容你（好与坏）？
  - 列出你展现的技能与长处。

> 示例（作者本人）：基思·法拉奇是一个创新型且注重底线的营销官和CEO，开创多个“第一次”，热情像一束光。

- 作者年轻时戴**蝴蝶领结**让人记住，后来随品牌升级而改变。
- 建议：建立**个人网站**（如wix.com、about.me免费工具），是最便宜有效的推广方式。

> 小小改变就能引起大大的不同。

### 9.3 推广你的个人品牌

# 推广你的个人品牌 – 知识整理大纲

## 一、为什么要推广个人品牌
- 知名度是扩大影响力与社交圈的关键
- 让别人知道你的价值，才能实现个人与公司的最大化收益
- 案例：唐纳德·特朗普 – 自我推广带来的品牌价值与商业机会
- 事实：懂得走出小隔间的人，更容易找到工作、升职、扩大人脉

## 二、推广的核心前提
- 如果你不推广自己，没人能帮你推广
- 成功不仅取决于工作成果，也取决于他人是否知晓这些成果
- 可利用的媒介非常丰富

---

## 策略一：打破“过滤气泡”

### 1. 什么是过滤气泡
- 社交与搜索平台用算法过滤信息，只展示用户感兴趣的内容
- 要抓住注意力，需创建可共享、能二次传播的内容

### 2. 案例：Upworthy.com
- 创始人：依莱·帕里泽（前moveon.org）
- 特点：左倾政治新闻也能病毒式传播
- 成效：点击量媲美《人物》《德拉吉报道》

### 3. 三大法则

#### （1）寻找视觉冲击
- 照片、信息图、动态gif、幻灯片更易传播
- 长文本博客适合建立信任，但不适合吸引新访客

#### （2）关注即分享
- 激发情感的内容更易传播
- 积极情绪（敬畏、欢乐、欣喜）传播力强
- 愤怒等强烈情绪也能引发关注
- 避免发布不成熟的帖子

#### （3）管理，而非创造
- 挖掘已有热度的内容，重新包装
- 使用吸引人的标题和易于共享的模板
- 借鉴加里·维纳查克的方法：观察正在进行的对话，融入内容

---

## 策略二：控制媒体

### 1. 媒体曝光是策划的结果
- 大公司有公关部门，小公司与个人需自己塑造形象
- 记者依赖他人提供新闻线索，具有“从众心理”

### 2. 一篇报道带来的连锁反应
- 创造知名度 → 其他记者关注 → 更多报道 → 更高知名度

### 3. 案例：YaYa公司 vs “大男孩软件”
- 背景：两家公司竞争同一新兴市场
- 挑战：对手资金雄厚，YaYa资源有限
- 成功方式：创造“热点”（Hot Spot）

### 4. 什么是热点（Hot Spot）
- 决定个人、公司、产品命运的强大现象
- 由草根、口碑、媒体渴望推动
- 案例：Napster – 即使倒闭，品牌热度也能卖出5000万美元

### 5. 如何创造热点

#### （1）催化时刻
- 类似比赛中突然逆转的关键球
- 需要内幕消息、疯狂泄密等引发讨论

#### （2）利用媒体力量
- 案例：杰西·温图拉竞选州长 – 借助媒体曝光，花钱少但效果好

#### （3）寻找“影响力人物”
- 早期使用者、名人、专家
- 案例：KPE机构 – 通过白皮书建立行业领导地位

### 6. 公关活动必须真实
- 从小事做起：地方报纸、行业期刊、白皮书
- 案例：YaYa公司通过KPE白皮书成为“广告游戏”行业领导者

---

## 三、与记者打交道的实用技巧

### 1. 建立真实关系
- 定期联系，提供内幕信息
- 主动接受采访，永远不说“无可奉告”
- 案例：与《财富》记者汤姆·斯图尔特的长期互动

### 2. 不要试图“操作”媒体
- 不要纠缠记者
- 不要试图纠正记者的文章角度

### 3. 警惕负面报道
- 案例：与《华尔街日报》记者哈尔·兰卡斯特的教训
- 记者提出问题的方式决定了你的报道角度
- 如果记者在写“失业人员”，无论你说什么，都会进入那篇文章

---

## 四、品牌公关行动策略（行动计划）

### 1. 你是自己最佳的公关代表
- 公关公司只能起杠杆作用
- 没有人比你更能讲好自己的故事

### 2. 了解媒体界
- 媒体是生意，追求收视率与发行量
- 提前研究记者与媒体的定位、受众

### 3. 研究角度
- 世上没有新故事，只有新角度
- 案例：宠物商店 → 零售创业潮、转行原因、赠送小狗给孤儿

### 4. 从小媒体开始
- 地方报纸、大学报纸、行业数字新闻稿
- 不要一开始就指望《纽约时报》

### 5. 让记者感到幸福
- 接电话要及时
- 帮助记者建立联系，提供资源

### 6. 掌握“精句”艺术
- 10秒内说清楚你是谁、做什么
- 现代总统片段平均7秒，你只会更短
- 准备好三条最有趣的要点

### 7. 不要讨人厌
- 被拒绝时，询问需要什么条件才能发表
- 有进取心，但适可而止

### 8. 将所有东西记录下来
- 小心被断章取义
- 不是所有报道都是正面的

### 9. 宣传消息内容，而不是传递消息的人
- 注意名声与恶名的区别
- 所有宣传必须服务于目标，而非自我膨胀

### 10. 把记者当作关系网中的一员
- 让记者喜欢你
- 无论报道是否公正，都表示感谢
- 媒体圈子不大，保持良好关系

### 11. 自抬身价
- 将报道与名人、政治家、知名商人联系起来
- 即使不真正认识，也可以引导记者去追

### 12. 做好市场营销
- 报道出来后，积极传播
- 社交媒体、校友杂志、课堂讲义、邮件签名等
- 可以幽默地自我推销（如：法拉奇再次不谦虚的尝试）

## 五、提升个人形象的其他方法（不限）
- 兼职承担自由项目，接触新人
- 在公司承担能展示新技能的额外项目
- 在公司上课、培训、参加小组讨论
- 利用朋友圈、同事、客户圈子传播你的价值

> **你不仅要做得好，还要让别人知道你做得好。推广你自己，是你最重要的职业能力之一。**

### 4.2 关键要点（作者多年经验总结）

## 5. 实战案例：与名人交往的经验

### 5.1 案例一：理查德·希夫（《白宫风云》演员）

**场景**：白宫记者晚宴后的《名利场》聚会

**做法**：
- 装作不认识对方
- 简单自我介绍
- 当对方迟疑时，自然地问“你是……？”
- 结果：对方打开心扉，最终交换邮件并保持联系

**启示**：不表现出过度崇拜，把名人当普通人，反而更容易建立联系

---

### 5.2 案例二：霍华德·迪恩（佛蒙特州州长）

**过程**：
1. 第一次：观看迪恩演讲
2. 第二次：再次观看基本相同的内容
3. 接触其竞选助手，表达兴趣
4. 第三次：提供**真诚的演讲改进建议**（强调几点、淡化几点、缩短长度）
5. 结果：迪恩在演讲中采纳了建议
6. 之后：在筹资晚宴上被迪恩亲自认可——“他在早期给了我很多建议，十分受用”

**启示**：
- 敢于冒险，真诚地提出建设性意见
- 即使对方地位高，你也能发挥真正的作用
- 帮助权威实现目标，他们会感激你

## 6. 接近权威的具体途径

### 6.1 组织与协会

| **核心策略** | 加入并成为活动中的**核心人物**，权威人士会主动找你 |

### 6.2 政治募捐者

- **特点**：政治把资金、热情和权力连接起来
- **做法**：
  - 每年做3~7次募捐
  - 支持地方和全国性政党
  - 参加当地活动，成为议题的直率拥护者
- **回报**：今天帮助的无名之人，明天可能是重要政治角色

### 6.3 会议

- **优势**：作为发言人，讲独特内容时，你暂时变成了名人
- **做法**：
  - 找到与你有共同爱好的知名人士
  - 参加这些人可能出席的会议
- **作者案例**：对领导力、灵性和人权充满激情，参与多个组织并在会议上发言，遇到无数知名人士

### 6.4 非营利性机构董事会

- **步骤**：
  1. 找到四五个对你十分重要的事情
  2. 在当地获得支持
  3. 目标是成为董事会一员，坐在知名人士旁边
- **注意**：确保你真正关心并想从事这个事业

### 6.5 运动（尤其是高尔夫）

**为什么高尔夫是首选？**
- 美国商业精英最喜欢的运动
- 球场是**公平竞争的环境**，名望不起作用
- 体育运动让人**放松警惕**，回到单纯孩童时代
- **远离办公室**，更易建立友情

**高尔夫的规则注意**：
- 一些俱乐部禁止明显暗示未来生意往来
- 另一些俱乐部允许光明正大地做
- 需要**自己发现规则**

**高尔夫的价值**：
- 总裁能通过打球判断对方：
  - 是否谨慎
  - 是否遵守规则
  - 能否处理压力
  - 能否相处愉快

**参与方式**：
- 行业协会定期举办高尔夫户外活动和比赛
- 慈善机构、会议等组织也会举办
- 即使不是会员也能参加

**作者个人选择**：
- 不打高尔夫（时间成本太大，4小时太难）
- 替代运动：巴里训练营、壁球、跑步
- **核心原则**：加入联盟、俱乐部或活动，一定会遇到充满激情的新朋友

### 6.6 网络与社交媒体

**现状**：
- 许多企业家和政治家开始使用社交媒体与粉丝建立即时联系

**优缺点分析**：
| 优点 | 局限性 |
|------|--------|

| 近距离感受魅力 | 真正引起注意需要别人介绍 |
| 鼓舞自己实现目标 | — |
| 学习到东西 | — |

**实用建议**：
- 加入名人在网上的**谈话**
- 浏览动态作为**侦查工作**——收集线索，了解他们需要什么
- 有可能：对方回复你的机智评论、收到直邮、在电梯里“偶然”相遇并交换名片

## 1. 核心观点：为何要自己建立组织

> **金句**：倘若你在这个山头无用武之地，没有理由不自己建一个。

### 1.1 传统俱乐部的局限
- 许多有价值的俱乐部/会议（如达沃斯论坛）门槛极高，普通人难以加入
- 年轻时人脉不广、资历不足，常被拒之门外
- 但人们渴望与志趣相投的人相聚，以利于做生意、建立人脉

### 1.2 自己建组织的优势
- 即使没有受邀资格，也可以主动创造机会
- 自己建立的组织能带来：
  - 志同道合的伙伴
  - 长期持续的人际关系（而非项目一结束就断绝）
  - 个人影响力的建立

## 4. 建立组织的关键步骤与方法

### 4.1 明确你的独特销售主张（USP）

> 你需要想清楚什么是你的 **USP**（Unique Selling Proposition）

**个人筹码可以包括**：
- 专业技术
- 人际关系
- 兴趣爱好
- 对特定事业的热情
- 个人经验

### 4.2 找到共同利益基础

所有俱乐部都基于共同利益而建，共同点可以来自：
- 相似的工作
- 相似的人生观
- 相似的兴趣爱好
- 亲密的邻里关系
- 同属一个种族、宗教或时代

> **核心**：人们需要一个结伴而出的理由

### 4.3 招募会员策略
1. 先邀请朋友加入
2. 朋友邀请他们的朋友
3. 时间久了，会出现更有趣、更具魅力的新人

### 4.4 建立可信度
- 邀请有影响力的人物加入董事会或作为支持者
- 高层人物的参与会为新组织带来足够的可信度

### 4.5 后续运营
- 公司创立、运营、融资
- 着眼于同样重要的细节工作

## 5. 线上与线下组织的结合

### 5.1 线上组织的适用场景
- 不住在城市或小镇
- 身边没有相同职业兴趣的人

### 5.2 线上组织搭建方法
| 平台/工具 | 用途 |
|-----------|------|
| 领英（LinkedIn） | 建立专业群组 |
| Facebook | 建立社群 |
| 谷歌环聊（Google Hangouts） | 每月发起线上聚会 |

### 5.3 组织发展进阶路径
1. 对成员进行民意调查，了解组织对彼此的帮助程度
2. 内部共享信息、互相推荐客户、应对挑战中互相学习
3. 组织壮大后，选择方便的城市举行**年度线下聚会**
   - 预订酒店楼层
   - 预约演讲者
   - 正式聚会

### 5.4 商业化模式参考
- 成功的互联网站（如iVillage）起初都是聚集一群有共同主张的人
- 基于成员归属感，建立可盈利的公司
- 例：飞行常客奖励计划、杂货店忠实客户打折

## 6. 著名社交案例：本杰明·富兰克林

### 6.1 早年困境
- 1723年，17岁的富兰克林搬到费城，人生地不熟
- 怀揣创业激情，但无财富、无成就

### 6.2 关键转折：创立“共读会”
- **组织形式**：星期五俱乐部
- **成员**：声望或成就有限的年轻人、普通商人
- **章程**：
  - 每个成员必须依序提出一个或几个问题供讨论
  - 问题可涉及道德、政治或自然哲学（物理）
  - 每三个月撰写一篇文章，在会上当众宣读

### 6.3 富兰克林的社交智慧

> “我很快认识到，当你想出了一个有用的项目……你最好不要宣布自己就是这个项目的发起者。我尽可能地隐藏自己，声称这个项目是朋友的计划。”

**谦逊与团队协作**：
- 隐藏自己的发起人身份，减少阻力
- 项目进展更顺利

### 6.4 通过“共读会”网络完成的公共项目

| 年份 | 项目 |
|------|------|
| 1731年 | 费城图书馆公司（北美第一个收费流动图书馆） |
| 1735年 | 改革费城巡夜制度 |
| 1736年 | 成立费城第一家消防公司 |
| 1749年 | 创立费城第一所大学（后来的宾夕法尼亚大学） |
| 1751年 | 成立费城第一家医院 |
| 1751年 | 创立第一家火灾保险公司 |
| 1747年 | 组建宾夕法尼亚州第一个志愿民兵组织 |
| 1756年 | 发起铺路、安装路灯和清扫街道项目 |

### 6.5 最终成就
- 成为18世纪传媒大亨（《宾夕法尼亚州公报》）
- 参与起草和签署《独立宣言》和《美国宪法》
- 1790年逝世，约2万人出席葬礼

> **核心启示**：一个志同道合、奋发向上的团体对彼此成功带来的影响不可估量，团队协作可以完成个人不可能完成的事。

## 1. 指导的核心价值与定义

> **核心观点**：指导是发挥个人最大潜力的有效策略，尤其在节奏快、流动性强的动态环境中。

- **适用范围**：音乐家、职业运动员、演说家、商业领域
- **组织环境**：跨职能小组、扁平化管理、快速应对变化
- **企业价值**：
  - 帮助员工建立有助于事业的人际关系
  - 减少员工流失
  - 强化对外关系
  - 提高公司收益

## 2. 指导的现代转型：从单向到双向

| 传统观念 | 现代观念 |
|---------|---------|
| 资深高管对上进新人的单向培训 | 不断给予与接受的双向过程 |
| 以职称或资历高低为标准 | 以特定技能为纽带 |
| 固定师徒关系 | 师父与徒弟角色不断互换 |

### 英特尔公司案例（1997年，新墨西哥州芯片制造厂）

- **创新做法**：
  - 通过内部网站和电子邮件打破部门障碍
  - 建立基于所需技能的指导者与被指导者伙伴关系
  - 互相学习对方的宝贵技能
- **成果**：
  - 快速将最佳实践模式在全球分公司传播
  - 培养业内最优秀员工

> **关键转变**：个人既是知识的传播者，又是知识的接受者。
