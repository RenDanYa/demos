# 个人笔记
- **达摩克利斯（权利成功）是脆弱的：宝剑落下不过是时间问题**
	- 当你拥有的东西更多以致失败成本更高时：**黑天鹅**事件便会与你不期而遇——成功的关成本（脆弱性）
	- 复杂/精密机制 → 从**复杂性**中受益/更脆弱 → **反脆弱性**（凤凰：九头蛇）
- **初级反脆弱性**
	- 剥夺系统的压力或重要的压力源并非好事，我们将变得脆弱
		- **米特拉达梯式解毒法**：不断摄入小剂量某种物质，随着时间推移，对大剂量同类物质产生免疫力（疫苗接种，药物过敏测试）
		- **毒物兴奋反应**：小剂量的毒药能够刺激酵母发酵，大剂量造成伤害
	- 强韧化的道路始于一点点伤害，会触发一些过度反应
	- **毒物兴奋效应才是常态**：重新确立自然剂量，缺乏毒物效应对我们有害
- **领域独立就是领域依赖**
	- 思维领域依赖性，无法脱离熟悉的环境来理解事物
	- 当同一种思想在不同背景中出现时，无法辨别，被表象东西蒙蔽
	- 无法跨领域理解事物的情况是人类固有的缺陷
### 随处可见的过度补偿和过度反应
- **创伤后应激障碍/成长：随机性→正/负面**
- 对挫折的**过度反应**所释放出来的多余能量，成就了创新
- 现代人试图依赖舒适、安全和可预测性来进行创新，不能接受“必要性确实是发明之母”的概念，创新和成熟源自最初的创新必要性，但却超越了满足这种必要性的状态（带来意外副作用）
- **压力源、反毒物效应与挑战缺乏**会导致**应激反应不足**，降低**最佳表现水平**
- 过度反应机制或类似**的机制**，使我们在有一点点背景噪音的地方更好地集中精力。有助于让观众切换到积极的思维模式
- 冗余→反脆弱性反应
	- **层层冗余**是**自然系统**集中管理风险的显著特征。（人天生不喜欢冗余）
	- 一个过度反应系统采用**超额模式**，建立额外能力、力量，预期更坏结果，对危险概率反应
	- **冗余**是机会主义，能带来一定益处，更像是**投资而非保险**
	- 冗余并非低效，往往常常高效
- 混乱、爱、信息、其他意料之外压力受益者的反脆弱性
	- 政治运动和叛乱，具有高度**反脆弱**特性。愚蠢行为就是**用暴力控制**，而不是操控，以退为进
	- 文艺作品中为**反脆弱**的**激情**所困扰的角色（难以泯灭的爱、恨）
	- **禁书/被批评、广泛评论书籍**，如果**挨过诋毁**，将大大受益（选择性偏见）
	- 忽视信息是**反脆弱**强。信息有能力**控制**那些试图控制它的人
	- 想法，越试图控制你的想法，你的想法就能控制你
- 给我们带来最大利益的并不是那些**试图帮助我们**的人，而是**曾努力伤害我们但最终未能如愿**的人
### 复杂体非复杂体
- 合成材料出现**内在自我强化反应**，人造物品只有**提高反脆弱性**才能用作生物组织
	- 复杂系统：各部分间的相互关联度非常重要；冗余的概念非常模糊，无法界定/确认，因果模糊性
- **压力源、传递信息 → 复杂系统 → 反脆弱性**。（压力就是知识，知识也可以是压力）
	- **频率**非常重要。在急性刺激下会比慢性刺激下表现得更出色。急性刺激后经历较长恢复期，使得**压力源**成为信息的传导渠道；慢性刺激不给人以恢复期
- 均衡
	- **非有机体**：静止状态
	- **有机体**：静止即为死亡。均衡离不开**波动性、随机性、信息的交换、压力**
- **压力太大 /恢复时间短** → **肌肉松弛、骨骼脆弱、智力退化** → **功能失调**（功能与环境随机性的结构错配）
	- **老化** = **衰老**
		-  自然：死亡
		- 现代：人为衰老
- **消除波动性**
	- 直击灵魂东西，了无生息
	- 降低多元性、差异性
	- 人 → 洗衣机：按照用户手册做出反应，目的是获得舒适、便利、效率
	- 游客与冒险家关系：观光化与生活关系，**吸进最后一点不确定性，还给人以获益错觉**
	- 电子日历
	- “目标驱动”型态度 深深伤害了内在自我
	- 现代生活充满本可以避免的慢性应激损伤

| - **机械体、非复杂系统** | - **有机体、复杂系统** |
|-----------------|----------------|
| - 需持续修复和维护      | - 自我修复         |
| - 厌恶随机性         | - 喜欢随机性（小幅变化）  |
| - 无法修复          | - 灾后需要恢复       |
| - 相互依赖性低或没有     | - 相互依赖性高       |
| - 压力导致材料疲劳      | - 缺乏压力导致萎缩     |
| - 磨损导致老化（损耗）    | - 闲置导致老化       |
| - 冲击下反应不足       | - 反应过度         |
| - 时间会带来老化       | - 老化和衰老        |


### 杀死我的东西却让其他人更强壮
- 某些人的反脆弱性需要牺牲他人的利益来换取
- 系统中，为了其他单元式整体利益，往往有必要**牺牲某些单元**
- 存在多个层级和层次（系统→有机体→细胞），反脆弱性变得复杂（子单元相互竞争）
- 创新企业脆弱，提高整个经济的反脆弱性
	- 系统内部某些部门必须是脆弱的，系统才具备反脆弱性
	- 有机体本身是脆弱的。遗传给新生命的基因编码却具有反脆弱性
- 反脆弱性
	- 初级概念
		- 从波动、意外、伤害中获益有限、剂量超过，保护、有益作用逆转（毒物兴奋效应）
		- 从外部现象，权力感
	- **进阶概念**
		- 伤害兴奋效应：从外部而非内部看，在伤害作用下愈加强大
		- 内部单元不因应激反应而增强，相反它会死亡，但完成了利益输送
- 大自然和个体生命间存在一定紧张关系
	- 个体：相对脆弱
	- 自然：压力、随机、不确定、混乱→反脆弱→进化
- 随机性
	- 自然运行经济→不会有常设政府、预测部门掌控未来
	- 历史→不必让某个帝国永远统治全球
	- 有机体
		- 完美预测未未/过度反应→永生不死
		- 一次生命/事物方向而非预测未来→代际延续修正
- **突变**随机性 + **环境**随机性 → **进化** → 服务于整个自然，自然在一定范围内反脆弱，但这一范围临界点非常高
- **毒物兴奋效应**：有机体能直接伤害中受益。（有机体即群体）
- 从其他人的损失中受益，从外部看，似乎是**毒物兴奋效应**，从内部看，却有赢家、有输家（群体即有机体）
- 内部的生存压力，在有机体遭遇外部压力时，有助于其整体改善
- 错误(信息源) ——> 并没有那么随机，有效解决方案 ——> 努力更像支出而非错误 ——> 反脆弱
- **墨守成规，减少改变(弊>利)** ——> 明确预测方法 ——> 脆弱
- 改变 ——> 不关心结果的多种可能性，利>弊 ——> 反脆弱性
- 往往从别人的错误中复盘 —— 遗憾的是，受益人不是犯错误的人，从他人的错误中学习，非系统性错误
- 一些企业也看重自身的错误(保险)
- 要让经济具有反脆弱性，并经历所谓进化。每个独立企业都必须是脆弱的：并暴露于风险中，将脆弱性从集体转移至不适合生存的公司；幸存的公司比最初的更强壮，为了改善系统，有人付出生命代价
- 人类以自我为中心，为了人类生存，不得不牺牲其他物种，不顾生态脆弱性
- 集体和个体间利益权衡：不打破个体利益，整个经济体无法生存；过度保护，为了个体利益，制约进化力量毫无必要
### 自治与集权
- **黑天鹅事件**：完全切断一个人的收入来源事件
	- └─ 人为消除随机性：清晰、稳定，但经受不住大的冲击，雇员风险是隐形的
	- └─ 自然随机性：承受每天波动性，经得起打击
- **集权制/联邦国家**，定性差异：人数增加改变了各方面关系的性质
- 大的系统
	- 「别人」是一个抽象概念：公事、公办（数字，电子表格……）思维
	- 不具体东西嗤之以鼻

- 缺乏波动性导致破坏性力量在平静表面下积聚，风险增长，当危机来临，损失惨重
- 波动性就是信息
- 随机性有助于系统
- 现代化

	- 以**理性为标志**的一个时代的精神
	- 核心要素：
	    - 天真干预，纠偏产生成本
	    - 代理人从神（上帝）→ 无条件相信科学预测，热衷于将未来浓缩于数字运算中
- 叙述性谬误：为冒险行动做出合理辩解
- 当有叙述的需要时，得命名，在行动中不需要，缺少“黄色”这个词，思想家就会遇到麻烦，实干家不会
### 天真的干预
- **加剧医源性损伤**：医院热、干预、帮助→伤害：
- 委托代理问题：一方的个人利益（代理人）与使用服务的另一方（委托人）的利益脱节。
- a
	- 不要造成伤害：**否认反脆弱性**（要做些什么才能让万物运转）→伤害
	- 赞美拖延
		- 有效的策略不是实现自己的目标，而是接受一个事实；他们的目标就是转移对方目标，让事件自行发展，让积极分子在制定不可逆政策前改变想法，发挥事物反脆弱特质，避免犯错，帮助我们筛选信息。
		- 有人将其与意志力薄弱，即缺乏自制力表现联系起来，但拖延不是因为不理性。
	- **限量供应信息**
- 理论→脆弱，现象→强韧，拥有理论是一件非常危险事情（不考虑理论误差产生影响）。
	- **欺骗分子，强调为他人做了什么远比强调帮助他人避免了什么更能让个人感恩戴德**
- a
	- 物理→误差越来越小，分散→平均斯坦
	- 社科→多有分歧，权力集中→极端斯坦
		- →在消除经济繁荣与衰退的周期，是经济脆弱性根源；负责某事（**干预（不真）**/无为而治（拥有英雄气概））
		- 社科和经济学家脑中没有医源性损伤知识。
- 更高层面医源性损伤：医疗健康、生态、政治、经济、商业、都市、预测、文学、教育、科技、媒体
- 医源性损伤对立面：伤害→帮助，资本主义（个人追求私利→集体有利）
- 大量信息，人从容镇静→ **神经过敏**（真实信息做出反应→对噪声做出反应）
- 无法区分**噪声和信号**→过度干扰
- 压力源就是信息，信息过多意味着压力将超过其反脆弱性的临界点
	- 医生需要证明他们无愧于你付的钱，无为不足以体现职业道德
	- 商业决策中，数据采集部门支持“及时”信息，将**噪声当作信号**
	- 媒体事后解读、提供理论、解释 → 认识世界错觉，学会在缺乏有意义的信息素材时保持沉默
- 情绪上对噪声反应过度，→**只看重大数据**，限量供应信息
- 难以向人们解释，数据多
	- 医源性损伤，人们仍抱有“科学”就意味着更多数据错觉
	- 领域依赖性，忘记对照现实世界查看心中对世界认知
	- 生活在**脆弱世界**，却认为它越来越容易理解
- 将催化剂（情报失误）与起因（对复杂系统，糟糕政策有误解）混为一谈，
    - 脆弱性→失败（√）（摇摇欲坠沙柱→桥梁倒塌）
    - 预测失误→失败（×）（预测哪辆卡车→大桥压垮）
### 预测
- 社会、经济、文化生活处于**黑天鹅**领域，物理世界离这个**领域**较远
    - 黑天鹅：无法预测，后果严重（数学局限/不可测量，预测的将永远不可测量，预测）→少即是多
    - 物理世界：可预测，后果不严重 
- 我们的复杂性不断让我们超越自己、超前发展，因此创造出事物越来越超出我们理解与掌控
### 非预测性的世界观
- 好奇心/图书馆**反脆弱性**：越满足感觉越强烈，博学会成为智慧
- 多余的财富，无异于沉重的负担
- 将重点放在行动上，避开言辞，依赖外部认有损健康
- 你无法预测未来，但你可以预测到：那些依赖预测行事的人，往往承担更多风险
- 胖子托尼具有反脆弱性：识别脆弱性，在脆弱事物的崩溃上下注
- **非对称性**：失去的远多于你得到的，变得脆弱（财富、成功的非对称性，让我们成为身外之物奴隶）
- 驯化情绪而非消除情绪
	- 消化你拥有的东西在你心中的地位
	- 将情绪关注转移到对产生情绪的核心根源的关注上
	- 对善进行投资，不会被剥夺；如果一无所失，获得的只有利益；反脆弱的
- **驯化而非消除不确定性**（增加有利因素，减少不利因素）
	- 减少不利因素
		- 降低脆弱性（强制要求，伤害不可逆）
		- 路径依赖：重要的是选择路线与事件发生的顺序，而不仅仅是目标
			- → 具有脆弱性经济增长，不能脱离衰退风险
			- 商人集中于产生利润，而风险和生存之后才考虑问题
	- 杠杆
		- 根据风险厌恶+极端风险偏好，消除毁灭性风险，减少极端
		- 不利因素的侵害，而不是改善中间因素（具有迷惑性，受巨大测量误差影响）
		- 体验压力，得到保护（策略：忽略小风险 + 保护他们免受严重伤害）
			- 人们非常反感小的损失，对于极端风险毫无反应
		- 职业分阶段：先做非常安全工作→投机性强工作
		- 反脆弱性是积极主动+保守偏执→有利的**非对称性**
### 可选择性
- **智力有时会让你低估反脆弱性，忽略可选择性的力量**
- 以低廉价格买入，即当非对称对我们有利时，我们不需要完全了解某一事物，可选择性带来的优势就是当你正确时，你会获得更大的收益，这使你不必每次都正确
- **期权是反脆弱性的一种代表**
- 除非你面对选择权和各种选择，否则你将永远不了解自己及自己的真正喜好
- 最高境界：同时实现财务独立和拥有知识生产力
- 可选择性
	- **有利因素 > 不利因素，成本小**
	- **非对称性（有限的损失，无限的收益），不确定性越大，选择权越有价值**（从低租金中受益，不受高租金影响）
	- 喜欢分散的事物、结果、有利因素，不关心平均结果（不存在负分情况（关注负面结果），具备可选择性）（奢侈品，房地产···）
	- 成功科学家数量累积于尾部效应、即极端情况；社会增长来自尾部人数增加、少数冒险者
- **泰勒斯主义**：关注回报，即行动结果，非对称性，非线性效应

### 时间与脆弱性
- 反脆弱性意味着旧的事物胜过新的事物，时间能够看到目的，但我们的眼睛和逻辑思维能力却无法捕捉它们
- 非对称性：反脆弱事物受益于波动、混乱，脆弱事物会因此受到伤害。（**时间与混乱一样**）
- 现代病：新事物狂热症（**干预主义**）
- 正确想象是从未来中排除或剔除不属于未来的东西：即否定法
- 脆弱事物终将破碎，会被其他东西替代，但其他东西是可预测的，低估“黑天鹅”事件的人终将淘汰
- 易于遭遇“黑天鹅”事件的事物终将被历史吞噬，时间提升事件的发生概率，长期预测比短期可靠
- 但换种思路，一般预测（反脆弱）的准确度随时间而降低（**非线性**）
- 我们倾向于将未来过度技术化，过去的历史，能比现在更好地预测未来

### 医疗凸性不透明
- 寻求能够带来非常大的健康收益，或收益明显超过潜在危害的医疗技术
- “你有证据吗？”谬误：存在证明无害证据与缺乏，证据证明有害混为一谈
- 在非线性情况下，简单的“有害或有益”陈述都会失灵，关键在于剂量
- 医疗护理成本：拒绝承认反脆弱性，力求过安逸生活，却损害了身体健康
- 医源性损伤（原则）
	- 经验主义：不需要找到受到伤害的证据，才能宣布某种药物是危险的（收益小，成本大，但滞后，隐形）
	- 非线性反应：不应该让基本健康的人去冒险；处于危险中的人应该冒更大风险（有条件生存概率结构）
- 凸性效应在选择权、创新、医疗等事物上，以相同方式运作：在一定剂量范围内，事物具有反脆弱性
- 大自然不透明逻辑：原子弹、创造生命工具、金融衍生品→错误的可扩大性、最不可预测随机性
- 法律上二分法：
	- 无罪推定（证明有罪前，我们是无罪的）
	- 有罪推定（证明无罪前，我们是有罪的）
- 大自然行为背后有严密逻辑，直到你证明事物并非如此
- 人类科学行为背后有缺陷，直到你证明事物并非如此

- 生物学的无罪辩护：现象学（此理论强有力）
- 解释总是在变化（因果不透明和原因发散性），人们慢慢养成了一种**无明确理论支持就不能形成观点习惯**；但经验却能保持不变。
- 理论只要在大脑回路中找到依据，似乎就更加**“科学”**，有说服力。即便它只是随机性地与心理神经相关
- 从医者看
	- 理性主义者（基于预设理论，需全面理解事物的存在原因）
	- 怀疑经验者（拒绝理论）
	- **方法论者**（不理会理论，寻找更加务实方法，成为经验主义者）√
- 适合企业的运作模式应该是**理论变化前保持强韧性模式**
- 当一个人将正面结果归功于自己技术；将失败归咎于运气时，**归属问题便产生了**。
- 由于对噪声反应过度，私人医生最终导致病人一命呜呼
- 使用较为复杂理性主义（数学）的唯一条件：相信自己并不拥有完整信息，而且行为也要符合。——**更成熟，接受自己并非无所不知的事实**

### 长寿，但不要太长。
- 人类预期寿命增加：卫生、青霉素、犯罪率↓。**挽救生命**手术出现
- **患致命疾病从医药、治疗中受益，此时医源性损伤非常低、凸性效应**
- 减法
	- 只有可能，用人类反脆弱性来替代医生
	- **幸福学：教我们分辨什么是不幸。“追求幸福”并不等于“避免不幸”**
	- 金钱也有自己**医源性损伤**
	- 随机营养，匮乏形成压力源（食物成分、效率）（功能的多样性应对环境、结构的多样性）
- **消除一些没有经历过进化历程事物有助于降低黑天鹅这种极端事件发生概率**
- 富裕、复杂而非贫困，单纯 造成**医源性损伤**（干预偏见，做些什么，肯定法），是知识不完全事物，而非完全无知带来恶果
- 基因是具有反脆弱性性的，它本身就是信息
	- **基因载体是脆弱的**，“我”在地球上不是为了求生苟活
	- 骗子可以识别出来：会给你正向意见，而且只有正向建议
	- 人们通过负向方法寻找成功，这也是进化过程选择
	- 对错误事情了解远胜于正确的事情，负向知识在错误面前比正向知识更强韧，证伪比证实更容易
	- 找到证据
	- 少即是多：简单预测，推断法要复杂方法好很多（极端领域）

### 切身利害：反脆弱性和牺牲他人的可选择性
- **代理问题**：非对称性，部分人受益而其他人受害，伦理与法律割裂。
- 如果你勇于承担风险，有尊严面对自己命运，你做什么都不会贬低自己做的事情
- 如果你不承担风险，做什么都不会使自己伟大
- 最佳风险管理规则：**建筑师自己控制风险，设定反诱因**，防止一个人在履行职责过程中伤害别人
- 启发法
	- 千万不要登上没有飞行员的飞机→奖励和惩罚间非对称性→决策者在“游戏”中下注
	- 确保飞机上还有一个副驾驶：**建立冗余、安全边际、避免优化**
- 不承担风险就不要发表意见。如果有自己的看法，就必须这样做，否则，就不会真正持有意见
- 知识世界导致知识和行动分离，导致社会脆弱性。（伪知识必然会转向空谈）
- 事后预测（脆弱推手）者反脆弱性，非对称性
	- **采用过滤式选择法挑选观点，并将失误预测丢入垃圾桶**
	- 获得免责选择权：**惩罚豁免权，我们却必须为此付出代价**
- 但行动是对称的，不允许过滤式选择，错误代价是昂贵的，消除免费选择权
- **斯蒂格利茨综合症=脆弱推手+事后过滤式选择**
	- 将卓越的分析能力、无视脆弱性、选择性记忆，缺乏切身利益集于一身
- 风险需要用行动而不是空谈去解决
- **预言是对信念的承诺**
	- 对信念保证，需转化为行动
- **真实荣誉感才能促进商业发展**
- 商业世界问题在于，只能通过加法而非减法来逆转。公司没有自然伦理，只服从于资产负债表（经理人代理），只有荣誉感才能促进商业发展

### 给职业戴上伦理光环
- 道德（信念）→ **职业** ✓
- 职业 → 道德（信念） × （**诡辩，事后捍卫**）
- **跑步机效应**：为了停留在同一个地方，你需要赚越来越多的钱。
- **贪婪具有反脆弱性**：贪婪受害者不然（**富有但不独立**）。
- 从事公共服务的人不应随后从商业竞争中赚取比薪水最高的公务员收入更多的钱
	- 人经过一个阶段的认知灌输后，**便迅速成为一种职业奴隶**：任何观点以利己为目的，对集体来说不再可靠。
	- 个人无法划清公共事务界限：**代理问题**
- 你应更重视提供不利于自己的证据的证人和意见
