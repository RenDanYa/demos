# 个人笔记

### 阅读的活动与艺术
- 主动的阅读（条件）
	- **阅读的主动性**
	- **阅读中每一种活动都需要技巧**
	    - **真正的阅读**  
		    - 活动  
			- 技巧  
			- 读物（有价值）  
			- 对应水平的读者  
			- 阅读 → 深入  
			- 模糊 → 理解
- 阅读的目标
	- 为获得资讯（书 **←→** 大脑，**完全理解**）  
		- 资讯：读得懂；记住更多的资讯  
	- 为求得理解而读  （书 **⟷** 大脑，**不完全理解**）  
		- 借助于（并非真正阅读）
			- 书
			- 人
		- 理解：必须要读；理解更多的事情（**理解程度不同**）  
		    - 把不相等的理解力克服到一定程度（差距）  
		    - 理解程度不同
- 阅读就是学习
	- **指导型学习**：被指导的艺术  
		- 吸收咨询知识
		- 记忆力  
	- **自我发现型学习**：阅读自我或世界的学习  
		- 被启发、吸收资讯
		- 理解力
	 - 增加知识
		- 阅读也是一种发现
		- 途径
			- 思考  
			- 观察  
			- 记忆  
			- 想象  
			- 分析


### 阅读的第2个层次：检视阅读
- 有系统的略读或粗读
	- **先看书名**  
	- **研究目录页**  
	- **索引**  
	- **出版者介绍**  
	- 书→ 头脑
		- 判断
		    - **① 是否值得**  
		    - **② 时间有限**  
		- 决定
			- 从目录中挑几个看起来和主题相关的篇章（开头、结尾摘要）  
			- 把书打开，东翻西翻，念一两段，有时连续读几页，不要太多  
			- 寻找主要论点，留意主题脉动
- 粗浅的阅读
	- **难读，不要停下来查阅、思索，从头到尾读一遍**  
	- 在读第二遍的时候，会对那个地方的了解更多一些
### 如何做一个自我要求的读者
- 主动阅读基础（**核心：提出问题并且回答**）
	- 本书的主旨（主题--分解--从属关键议题）
	- 怎么说的：论述过程
	- 有道理吗？是全部有道理还是部分？（了解之后做出判断）
	- 本书与我的关系（获得资讯、启发）
- 做笔记
	- 原因
		- 保持清醒
		- 阅读前思考：思考要用语言表达出来
		- 我所思与作者所说结合，记忆方便
	- 方法
		- 画底线
		- 在空白处做符号或夹书签
		- 关键论点句子圈出来
		- 在空白处编号或记下其他页码，让散落东西集中起来
	- 类型
		- 结构笔记（检视阅读：第一遍）
		- 概念笔记（分析阅读：>一遍）
		- 辩证笔记
### 分析阅读第一阶段
- 给书分类 （书名，类型）← 检视阅读（笔记）  
	- 小说类 
	- 论说类：传递知识  
		- 类别
			- 历史：过去故事陈述——个人评论、观念、意见、
			- 哲学：一般人的生活经验  
			- 科学：任何时间、地点、定律、通则（超乎日常生活）  
			- 数学
		- 性质
			- **实用性**：知道并且行动（应该、应当、好/坏、结果、意义）  
			- **理论性**：知道怎么回事（是陈述）
- **骨架**：整体（干什么）使用单一的句式或最多几句话、叙述整本书内容  
- **血肉**：拆解将书中重要篇章列举出来，说明按何种顺序组成整体结构  
	- ![[Pasted image 20251018103532.png]]
- **阅读** （发现骨架）⇨ **写作**  （以制造骨架开始，想办法隐藏骨架）
- 作者要问的问题
	- **理论性**：  
	    - ① 某件事存在吗？  
	    - ② 是什么样的事？  
	    - ③ 发生的原因是什么？  
	    - ④ 目的是什么？  
	    - ⑤ 造成的影响是什么？  
	    - ⑥ 特性及特征？  
	    - ⑦ 这件事是如何进行的？  
	- **实用性**：  
	    - ① 有哪些结果可以达成？  
	    - ② 要采取什么手段才能获得结果？  
	    - ③ 要达到目的应该采取哪些行动？  
	    - ④ 以什么顺序？  
	    - ⑤ 这些条件下，什么事是对的，会更好？  

### 分析阅读第二阶段
- 找出重要单字,透过他们与作者达成共识  
	- ![[Pasted image 20251018085003.png]]
- **关键字，专用语,及特殊情况**  
	- 字义
		- 单一的意义
		- 两三种意义
			- 判断1还是多 
			- 多之间的关系  
			- 某个地方出现:使用哪一种意义  
	- ![[Pasted image 20251018085154.png]]
- 找岀关键句,找岀主旨句  
	- 一本书,关键字只有少数几句  
	- 标明重点,提醒读者注意的地方  
	- 对难以理解的部分,需要努力来诠释
- 主旨
	- 在一本书最重要的论述中——不是前提就是结论 
	- 复杂的句子通常要说明的通常不只一个主旨  
	- 能不能转述成你自己的话  
	- 能不能举出一个自己所经历过的在主旨所形容的经验  
	- 能不能就特殊情况说明其中通用于一般的道理

### 分析阅读 第三阶段  （真实的吗？意义）
- 主动的阅读，必须能评论，提出批评，才算真正完成这件事。  
- 最能学习的读者，也就是最能批评的读者。
- 智慧礼节的一般规则
	- 在评论前，要肯定地说我了解了。（分析阅读一二阶段）** 暂缓评论 
	- 不认同作者的观点时，理性地表达；不要无理的辩驳或争论  
	- 当读者与书中观点的不合时，尊重知识与个人观点的不同
	- 在作任何评断前，找出理论基础（读者为自己的不同意见找到理论基础）
- 不争执的 ：当读者与书中的观点不合时  
- 成因 → 达成共识
	- 排除误解
		- 人类复杂天性
		- 理性  
		- 情绪与偏见  
		- 语言不完善的媒介 
	- 增加知识 
		- 知识的不相当
- 作者的立场（知识）←→ 强化基础（个人观点）
- **批评观点的特别标准**  
	- 证明作者的知识不足  
	- 证明作者的知识错误  
	- 证明作者的不合逻辑  
	- 证明作者的分析与理由是不完整的
- 找出论述 
	- 如果可以，找出书中**重要论述段落**。但是，如果没有，你要从这一段或那一段中挑选出来，然后整理出先后顺序的主旨，及其组成的论述
	1. 所有论述包含**声明**：理由、结论
	2.  区别论述的不同之处
		1. 归纳：特殊 → 通则 
		2. 演绎：通则 → 更通 
	3. **假设** / 自明之理 → 证实 （论述开端）
- 找出作者在书中想要解决的**主要问题**。

- 两种实用性的书 （任何实用性的书都不能解决该书所关心的实际问题，只有行动可以）
	- 理论性的原则会归纳出出色的行事规则
		- 学实例
		- 规则背后的原理
	- 只说明规则（烹饪、驾驶）  
- 了解
	1. 本书在谈论什么
	2. 主旨论述  
	3. **内容真实性、意义**  
	4. **本书与我何干（对实用说，需要采取行动）** 
		- 规则
			1. **通用**
			2. 特殊（蛋糕）
- 要点
	- 对你感兴趣的事件或时期，阅读一种以上的历史书  
	- 不要只关心过去发生什么事，要懂得在任何时空之中，尤其是现在，人行动的原因  
- 阅读历史
	- 提出的问题
		- 主题
		- 用什么方法读故事？  
		- 批评
			- 不够逼真（科学与故事结合，具备可操作性，与我何干）  
			- 误用了资料
