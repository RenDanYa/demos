# 个人笔记
### 相对论的真相
- 人们很少做**不加对比**的选择。内心并没有一个**内部价值计量器**，只有在具体情境里，才能比较出优劣，估算价值
- 需要有个诱饵在那里，它是秘密的原动力，能帮助我们决定
- ![[Pasted image 20251016074559.png|300]]
- 我们总是靠周围的事物确定彼此关系。
- 相对化容易理解，但侧面却总是给我们造成失误：不但喜欢拿事物与事物做比较，还喜欢把容易比较的事物集中做比较——避免把不容易比较的事物做比较
- 薪酬一旦公开，与其他人比较，到头来主管们的工资火箭般地往上蹿。
- 人心不足蛇吞象，打破相对论怪圈
### 供求关系的谬误
- **任意的一致**：首次的价格大都是‘任意’的，一旦价格在大脑中得到确立，形成不仅对某一产品，还包括对其他有关产品出价意愿
- 锚具有持续效应，对当前价格是这样，对将来价格也是如此
- **羊群效应**：基于其他人的行为来判断某事物好坏
- **自我羊群效应**：我们基于自己先前行为而推想事物的好坏
- 把锚从邓肯甜甜圈店转移到星巴克：不是从价格，而是商品上，营造一种与众不同的体验，敞开思想接受新锚
- **一桩两可体验（买→卖，工作→玩耍）任意转换愉快，痛苦体验，第一印象一旦形成，锚已设定（并非偏好）**
- 一旦做出首次的决定，以后的决定就会以一种似乎合乎逻辑、前后一致的方式紧随而来
- 我们有能力改进自己的非理性行为
	- 做一切事情都应进行自我训练，置疑自己一再重复的行为
	- **特别关注我们所做的首次决定**
	- 清点生活中的印记和锚：对老的选择重新考虑，向新的决定机会迈进。
- **锚和幼鹅 → 消费者偏好 （×）→ 固化锚定（√）**
- 供需决定市场价格，提出质疑
	- 消费者的购买意愿被操控，不能很好地把握自己的偏好及不同商品体验的价
	- 供给，需求各自独立，但锚（零售、广告价格…，供方变量）的控制作用显示互相依存，需求并不是完全独立于供给
	- 供求关系不是基于偏好而是基于记忆
		- 对过去价格记忆 
		- 想与过去保持一致的欲望
- **自由市场贸易：清楚双方所交换物品的价值**
- **宏观调控：选择总是最初锚定影响，而非偏好。不能使个人利益最大化，政府干预非理性行为** （价格↑，需求↓，适应新的价格，锚做了调整，需求恢复，长期看，价格对需求影响小）
### 零成本的成本
- 零：能唤起强烈的情绪——成为一个非理性兴奋来源：
- 免费最大问题在于：引诱你在和另一件商品之间挣扎，并引导做出不明智决定。
- 正常交易有有利的一面和不利的一面：免费让我们忘记了不利的一面，造成一种情绪冲动。
- 厌恶损失，免费商品不会有显而易见的损失，普通商品有风险。
- 零成本吸引力不仅限于现金交易，物物交换、现金交易，免费像地心引力，难以抗拒。
- 想让顾客盈门：拿出东西免费送；想增加销售，拿出一部分商品免费。
### 社会规范成本（白干活高兴，赚钱反而不高兴）
- 生活在两个不同的世界下
	- 社会观念：包藏在社会本性和共同需要，友好/界限不明/非即时的报酬
	- 市场规范：界限清晰/利益比较/及时偿付，按劳取酬
- 在各自的轨道中运行，顺利向前；发生碰撞、麻烦。
- 当涉及金钱，市场规范就起作用，社会规范随之而去，涉及到自立帮助，个人主义（不要问价格几何）
- 把市场规范引入社会交换，就违反社会规范并且伤害了社会关系，难以恢复。
- **礼物规范**：界限高度/利益比较/及时偿付，按劳取酬州
- 充分自由的轨道中运行，怕冲向何方；发生碰撞、麻烦。
- 当反制金钱：市场规范起作用，社会规范随之而去，涉及到自己去帮助（个人主义，不要用价格衡量）
- 把市场规范引入社会交换，就违反社会规范并且伤害了社会关系，难以恢复。
- 礼品能把我们保持在社会交换环境里，脱离市场规范。
- 鱼与熊掌不可兼得
	- 照章办事：明示能提供什么，期望什么回报，市场交换，忠诚度就弱。
		- 难以摆脱追求利润，削减成本困扰
	- 礼物有其自身价值，营造亲善氛围
	- 创新比机器重要：工作、休闲界限模糊，有热情、勤奋，并且关心公司，激励员工，保持忠诚（降低福利，从社会关系→市场关系）
- 考试分数与工资业绩直接挂钩：把教育从市场→社会，金钱的作用是有限的，且经常是最昂贵的激励方式。
- 社会规范不仅成本低，而且更有效，与其把注意力集中到分数、竞争、工资上，倒不如灌输一种目的感，使命感和自豪感
- 过去几十年市场规范怎样接管了我们的生活——强调高工资、高收入高消费，让社会规范回归可能并不是什么坏事
### 非理性与自我控制
- 在非兴奋状态中，他们不知道自己在兴奋状态下会怎样，预防、保护、节制及道德感都消失了
- 他们以为了解自己，了解自己的偏好、能力，但他们对自己反应估计不足
- 了解冷静状态也需了解激情状态
- 了解不同，怎样对我们的生活有益，又如何把我们引入歧途
- 拖沓、非理性拖延是同一性质问题，我们发誓存钱、健身，那是在清醒状态
- 因受到短暂冲动而偏离长远目标，这就是拖沓
	- 医生们有拖沓习惯
	- **严格限制**他们的自由是治疗拖沓有效手段
	- 让他们自己设定期限，也可以解决问题（并不是每个人都能了解自己拖沓倾向，对问题有完全的了解，自己设定期限，未必能使他们的能力得到最佳发挥）
- 自我控制有困难（即时满足与延迟满足），又都有潜在自我控制机制
	- 外部声音发出命令
	- 给人以预先参与机会，选择他们喜欢的行动路径
### 所有权的高昂代价
- 所有权的三大非理性：
	1. 对已经拥有的东西迷恋不能自拔
	2. 注意力集中到失去什么，而非得到什么
	3. 假定别人看待交易的角度和我们一样
- 宜家效应：在某种事物上投入的劳动越多，对它的感情越深
- 虚拟所有权：在实际拥有所有权之前就对某物产生了拥有的感觉。（拍卖会时间拖得久、花的钱越多；广告业**试用**促销）
- 不仅局限于物质的东西。对人的观点、看法同样适用：我们对它难以割舍，想到失去就惶惶不可终日。最后转变为一种僵化且顽固的意识形态。
- 在生活节节向上时，想当然认为随时可以倒退回去，实际上不可能。
- **“非拥有心态”**看待每一桩交易，给自己和感兴趣的物品拉开适当距离
### 多种选择使我们迷失目标
- 保留余地的同时我们也放弃了别的东西，忘记了在真正重要的事情上下功夫
- 在明确的目标指引下，我们都应努力追求最大限度满足（不会损失情况下）
- 在门与门之间奔忙
	- 非理性冲动，使我们追逐毫无价值的选择——几乎消逝或对我们不再有价值机会，要让所有门都开着，保持所有选择是愚人戴尔游戏
	- 认识到某些事物已经成了消失的门，需要我们当机立断去珍惜。
- 在集中关注两种选择间细微异同的时候，偏偏没有考虑到迟疑不决的后果。
### **预期的效应**
- 蒙上顾客的眼睛进行品尝测试、配方未必有多大用处；但是事先给我们这种信息，用以改变我们的预期，却能有效影响我们的味道判断
- 不要低估展示的力量。（预期的影响）
- 市场营销做的就是这方面工作。——向消费者提供信息，提高他们对产品的预期和真实快感。
- 预期还可以形成成见，影响我们对世界其他方面体验
- **蒙眼**状态有可能帮助我们更好地认识事情的本质。
- 完全消除固有观念和现有认识是不可能的。但起码我们可以承认人人存在偏见。有了这种共识，我们就可能接受，冲突的解决一般需要中立的第三方制定规则，需要一个未被预期影响的第三方。
### 价格的魔力
- 预期
	- 人们对视觉、味觉、其他感官现象的认知与解析
	- 改变他们主观甚至客观体验从而施加影响
- 形成预期
	- 信念（对药品、手术、有关人员、信任）→安慰疗法
		-  **暗示的力量**
	- 条件反射
- 熟悉未必产生轻视，但他确实产生预期——价格偏见
- 对医药、软饮料、大众化妆品和汽车来说，预期价值可能成为真正价值
### 为什么我们不诚实
- 不诚实行为
	- 基于成本收益计算，决定是否动手抢银行
	- 自认诚实之人——正人君子（普遍）
- 有了适当机会，很多平日里诚实的人也会作弊
- 一旦在诱惑下作弊，程度并不像人们通常认为的那样，与害怕被发现有直接关联
- 不诚实
	- **成本收益** → 没有对诚实进行帮助、监督，就只剩成本收益的理性分析 → 不诚实
	- **美德内化** → 只有在考虑重大越轨行为时才被触发
- 鼓励人们诚实的并不是十诫的条文本身，而是出于对道德准则深思→ 纠正不诚实
- 受到诱惑时，得到有关道德提醒，我们可能保持诚实，认识到弱点，可以防患于未然，避免发生
### 为什么现金能使我们更诚实
- 在非金钱事物上作弊，而在金钱上阻止作弊；离现金一步之遥的作弊最容易发生。
- 擅长把细微的不诚实想法和做法合理化，很难清楚地确定非金钱事物对作弊影响
- 象征货币给人们道德约束**松绑**：为自己行为做出合理化解释能力突飞猛进
- 人人都是脆弱的，所谓的好人——讲道德并且自认为有操守的人们所不想逾越的不诚实界线
- 动机带来的刺激可能愚弄我们，不管我们是不是好人，是不是奉行道德准则
- 人们选择食品和饮料时好像有2个目标
	- 给自己带来最大享受
	- 在朋友心中炫耀自己某些正面人格特征
		- 标新立异
		- 归属感
