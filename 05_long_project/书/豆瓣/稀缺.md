# 个人笔记
### 专注的得
- 稀缺
	- “拥有”少于“需要”的感觉
	- 当我们经历任何一种形式的稀缺时，都会对稀缺的事物全神贯注；思想会转向未得到满足的需要
- 稀缺俘获大脑（注意力）
	- 拥有太少感到不悦
	- 影响我们的所见和所做的速度（对注意力的俘获）
	- 影响对周遭世界的认识（改变思维方式，强行侵入我们的思维）
	- 进一步延续并加剧稀缺
- 经济学：实物稀缺
- 稀缺
	- 实质上的约束
	- **心态，带宽**（认知能力和执行控制力，降低带宽容量，缺乏洞察力和前瞻性，减弱执行力）
- 得之积累的努力浓缩成即使产出的成果，不能凭空创造技艺，给了一股推动力整合
- 迫使他人做选择，抽象的事物变得具体起来
- **专注有利**
	- 会议一开始没有重点，内容也抽象。中途修正后，关键问题成了焦点话题，取得实质性进展
	- 截止日期提高工作成效
	- 礼券吸引人们注意力
	- 销售人员完成业绩
	- 毕业生享受美好时光
- 不同背景、经历 → 截止日期 → 不同行为（**稀缺本身就是原因所在**）
	- 帮助我们做成一些凭借自身力量很难做到的事
- **佯装稀缺是很难做到的**
### 管窥的失
- 管窥
	- 专注于某一事物就意味着我们会忽略其他事物，也叫隧道视野
	- 会放大其他事物成本，缩小其他事物收益
	- 目标抑制 → 稀缺 → 专注红利（管窥之见）
	- 对一项事物的专注会抑制竞争意识，当生某个人的气时，抑制作用会令你忽略他的优点；专注于反感，抑制记忆中积极一面
- 管窥→同时处理多份任务；只注重管子里的视野（客观评价取决于管窥引发的不利后果）
- **营销**/**保险**，在管窥视野里是**成本**，视野之外是**收益**
- 稀缺会不断将我们吸入管子的视野之中，形成宽带负担
- 干扰我们思考
	- 外部噪声
	- 稀缺、内部干扰，内源性思考
- 自上而下：是指对所关注的事物进行有意识选择
- 自下而上：注意力会被很难为我们所控制的刺激俘获（非自愿行为：行为的发生不受意识控制）
- 自上而下的注意力无法阻止自下而上的侵入
- 稀缺会减少带宽容量——不是减少某人与生俱来的宽带容量，而是减少其当下**能用上**的容量
- 前摄干扰：当我们心里想着其他事物时，就会将其置于思想中心，而产生忽略行为
- 带宽
	- 认知能力
		- 解决问题、获得信息、进行逻辑处理等能力背后的心理机制
		- 核心：流体智力（在进行抽象思维和推理时，无须特定学习或体验下解决问题能力）
	- 执行控制力
		- 作用在我们管理自身认知能力的过程中，计划、关注、发起并抑制行为、控制冲动
		- 核心：自我控制
### 带宽负担
- **自我控制**
	- 对未来评价
	- 意志力（控制注意力和想法）
	- 执行控制力
- 稀缺会形成带宽负担，不仅会降低流体智力，而且会降低自我控制力
- 贫穷会成为心智的负担，有效能力比富人要弱一些，大脑已被稀缺俘获了
- 什么都不缺的人，只有一部分会满腹心事；而所有经历着稀缺的人，每一位都心事重重
- 压力是长期的，稀缺是即时的。将某些现象视作压力和忧虑结果，就会忽略深层原因，限制对稀缺心态理解
- 稀缺
	- 分心
	- 爱忘事 → 误解 
	- 大脑因稀缺而疲惫不堪→缺乏冲动控制能力
- 稀缺心态一旦出现，应对心智资源更加紧缺问题
### 装箱余闲（权衡式思维）
- **余闲不仅意味释放，更是一种奢侈的心理态度**
- 如果感觉充裕，所做的选择、平均决策，**“粒度”** 就会变小一些
- 时间行李箱、金钱行李箱、热量行李箱
	- **大件行李箱**：随心所欲
	- **小件行李箱**：谨慎、满当
- **稀缺 → 权衡式思维**（没有被满足的需要俘获了我们的大脑）
- 余闲可以将我们从**权衡的苦差事**中解脱出来
	- **资源丰富时进行资源管理的特定方式**
	- 不是刻意保留的空间，而是因为装箱时间充裕而产生的“副产品”
- 边际效用递减：拥有越多，对你而言，每一种额外增加的物品的价值就越低
- 不去思考是否有用，而是思考为什么不呢，脱离了权衡困境，低效和浪费产生
- 余闲给予我们无须做选择的自由
- 稀缺让我们通过**坚持最初计划**来避免选择负担，实际上就是选择了不去做选择
- 余闲会将错误吞噬，将错误引发的后果最小化；没有余闲，就没有那么容易逃避犯错的后果了
	-  给我们周转的空间和失败的空间
- **失败成本高，可能性大** → 谨慎（×）（减少失败，仅凭努力不够，偏见是大脑运行结果，而且并不总能对后果响应）
- 财富可将诱惑转为可以买到的奢侈品
- **稀缺** 
	- 提高失误成本
	- 创造更多的机会犯下错误，做出不明智选择
- **心理**
- **数学**：大脑处于计算难度更大环境
### 行为经济学告诉我们道理
- 在小物件上连几毛几分都会计较，而在大物件上却挥金如土（**认知具有高度相对性**）
- 相对性认知是大脑处理信息时的固有特征，**经验与技能能让我们摆脱限制，对认知形式改变**
- 富人是金钱价值方面专家，更接近于理性的经济学理想，更接近经济人。
	- **评估1美元时，会用大脑中内化的衡量标准情形（不受背景环境影响）**
- 充裕资源使得我们不那么容易了解金钱价值，容易陷入他人的操控之中。
	- **利用背景评估，缺乏清晰价值观，富人视角，富人避免**
	- **不划分心理账户，而富人划分，用不公平眼光区别对待**
- 身陷贫穷会让人比那些经济学家更懂经济学
	- 经济学家：**收入颇丰，存在资源、不习惯权衡日常生活**
- 经济学本意遵从稀缺的内在逻辑，经济学预测，对于真正拥有稀缺心态的人来说，更加准确。
### 借用与短视
- 借用与稀缺同时存在
- 明天的稀缺注定无法像今天的稀缺一样俘获你的注意力
	- → **信用能有效解决当下的燃眉之急，后果往往被人忽视**
	- → **管窥之见（对信用偏见）**
- 当今天的资源：无论是时间、金钱、能在当下为你带来比未来更大的利益时，借用行得通
- 当有了管窥之见，借用就会超越得失均衡限度，我们的信用，从长期来看不合理
- **现时偏见：将未来利益作为代价，过高估计即刻利益**
- 穷人之所以会产生借用行为，是因为贫穷本身造成的；所以不必在短视、财务知识缺乏等因素上寻找解释
- 将重要却不紧急的事务搁置，与借用无异，这种倾向不仅有关时间，也有关于金钱
	- → **忽视未来**
	- → **无力规划（解决眼下问题）**
- 绝大多数选择是被动的、容易的 → 生活在一片狼藉之中
### 稀缺陷阱
- 稀缺陷阱
	- 某人的行为有助于稀缺形成的一种情况，继承稀缺元素
	- 并不限于物质稀缺，根源在于对资源的错误利用，从而导致实际短缺
- **东西→将可预期事件当作突发其来事件进行处理根源**
	- **持续从一项重要任务转换到另一项重要任务中，只能局部、暂时性地解决问题**
	- 时间紧迫、任务繁忙
	- 宽带用来为抛向空中消失未坠落的球而焦虑
- **逃离稀缺陷阱→制定计划**
	- **持续、永恒警醒，任何时候都能抵御诱惑能力**
		- 重要不紧急，忽略
		- 对未来进行计划需要带宽，而稀缺则是带宽的沉重负担
		- 形成真正计划，难度大，杂耍和稀缺陷阱拼凑式的凌乱补丁图，找出最佳解决方案，难
		- 执行难度大
		- 意外事件
- 意志力不会随着经历困难的次数而提升，频繁使用后变得疲惫，逃离稀缺陷阱就会变得很难
- **余闲的缺乏（问题的根源）→落入稀缺陷阱**
	- 不能吸收不确定元素，但由其影响自己的生活
	- 阶段性稀缺会引发一些行为，最终将我们拖回稀缺陷阱中，建立针对突发事件缓冲机制
		- **风险隔离→ 风险隔离**
		- 免于跌落到陷阱中
- 余闲→低效→足够充源资源→逃离
	- ![[Pasted image 20251016103730.png|300]]
	- 稀缺锁定注意力，成为一种障碍
	- （节食者、孤独者面对稀缺时，关注于每一个小细节）
- **稀缺造就一种心态，令稀缺长存**
	- 环境造就结果、并非个人特质，通过一些措施加以改变
	- 改变心态，逃离稀缺陷阱的唯一希望
	- 制定计划
	- 余闲
### 穷人为什穷
- 各种类型的稀缺都有着相同元素：管窥、借用、余闲、带宽负担
- 在不同环境下，以不同方式表现出来：金钱稀缺，借用是主要特征；孤独、借用没有什么意义
- 拥有自由裁度能力的人来说，稀缺陷阱会威胁到他们，但程度有限。人们无法逃开贫穷去休假
- 无能 → 贫穷 → 无能
- 轻松工作一天后，就会表现出“中产阶级”的样子；辛苦工作一天后，就会落入“贫困人民”的队伍
- **良好教养水平需要足够的带宽，复杂的决策能力和自我牺牲精神。**
- 若想成为一位合格的家长，需要许多方面能力，最重要的大脑的自由。(奢侈品)
- 许多行为都依赖带宽，带宽过重，意味着处理新信息能力减弱
- 穷人又缺钱又缺带宽，穷人的无能主要原因在于贫困本身
- 带宽负担可以将许多现象包罗在内，跨越时间、行为、地理
- 贫困
	- 物质 
	- 心理
- 缺乏带宽 → 缺乏技能（学习社会技巧还是养成良好消费习惯）
### 如何让穷人摆脱贫穷
- 从**稀缺**角度出发，可以预见到失误的频发和不可避免，而失误与否，与人们是否有积极性无关。
	- **包容失误，可以确保穷人在勇于承担责任时，拥有提升空间**
- 规定福利救济年限，产生稀缺，对资源利用进行管理，却存在缺陷。
- 若想让限制手段影响人们的行为，须令其进入“管子”视野之内。其中一种方法是，定期发送剩余月份提醒
- 通过唤起人们关注，将遥远问题纳入“管子”视野之中。另一种方法是，**改变限制结构**（制造出范围更小，更频繁限制）
- 为了让激励机制发挥作用，人们必须在“管子”视野中看到它的存在。而大多数激励机制都有落在“管子”视野之外风险，这就等于是无形而无效的。可见，我们需要对激励机制**精心设计**
- 节省带宽才是好方法。
- 日托服务
	- ├─ 节省时间
	- ├─ 节省金钱
	- └─ 节省无限量的大脑自由带宽
### 如何解决组织中的时间稀缺
- 计划中
- 计划外（余闲），面对稀缺威胁时，余闲是必须品，而我们却总是不能将其计入计划之内
- 更有效率，就会丢掉余闲（人、组织）
- 之所以没有能力构建起余闲，是因为专注于当下工作，预测不到未来可能发生的所有事情
- 融资机构
	- 稀缺 → 精益化，节约化，经理人时刻保持警惕；降低成本；提升认知能力，进行艰苦谈判，检查细节，以确定每笔支出是否需要）
	- 太过精简：用按揭未来的手段弥补当下青黄不接，造成破产
		- **专注力在稀缺状态下是自然而然的，在充裕的时间对困难
	- 经理人在花钱时更加睿智
- 环境投入充裕的关注度，弥补当下资源不足，造成脱节
- 银行经理忙于利润，实现最大销售目标；而首席风险官的唯一目标就是：**监管风险**。（分工）
	- 助理
	- 应对急诊手术
- 时间管理智慧：
	- 增加工作时长：让员工卖力干活是**管窥心态**表现
	- **有效带宽最大化**，而非工作小时最大化
- 完成项目
	- 时间
	- 宽带
	- **双管齐下，组织中的个人犯下错误，组织却有可能将问题放大，多米诺骨牌效应
- 餐饮业也被稀缺问题所累
### 如何应对日常生活中的稀缺
- 直接干预，防止事态进一步恶化，防止稀缺陷阱出现
- 对某人所处环境进行小小改变，缓解稀缺带来的不良后果**
	- **稀缺心理是原生的，若想从内在改变，难度很大**
- 对自身所处环境进行**稀缺防御**。可以引入类似**减速带或助理这样工具**，利用我们对事物本身的洞察力，来构建出更为理想效果，关键在于你所采用手段的内在逻辑。在于是否真正理解稀缺能帮助我们用不同方式思考，以及帮助我们管理长久以来得不到解决的问题
- **将重要事情拉入意识视野中**
	- **提醒（助理和私教）** 
	- 冲动
		- 储蓄卡片
			-  挖掘出人们潜藏于心中目标：为孩子存钱、买房……
			- 提供一种为目标而行动的简易方法：购买卡片
- 当忽略问题存在时，改变行为结果往往比直接对抗更有效（设置默认选项，让疏忽等同默许）重复性高，可预测事件
- 选择
	- **一次性（产生管窥心态后，一次性做正确事情，比重复做正确的事情容易得多）**
	- **时刻保持警醒，持续不断重复选择**。
- 只要有可能，**将警醒型行为转变为一次性的行动**
- 相反，把存在问题时的一次性行为转为需要时刻保持警醒的行为也十分必要（选择反复变化，某些一次性选择可能存在误导）
- 稀缺虽然会使你选择权宜之计，但同样提高你因权宜之计受到伤害的可能性
- **找准关联行动时机：利用人们所期望事物（涨工资）与所希望事物（增加储蓄）之间联系，先发制人，做好过度工作，将事务提前放在视野中。**
- 你以为保持灵活度是在帮自己，却让自己产生拖延和分心，带宽负担↑
- **带宽变化时机的重要性，在于能让人们将计划与更为理想的宽带变化时机联系起来**
	- 而非仅根据时间是否充裕
- 为宽带不足的人提供便利，当我们的宽带承受负担时，最简单的障碍也会创造出巨大的损失
- 带宽是一种核心资源，在为人父母、健身都会用到，带宽会影响我们的思维，影响我们选择
