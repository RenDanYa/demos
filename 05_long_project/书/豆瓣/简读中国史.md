# 个人笔记
### 历史脉络
1. 未曾中断
	- 地理隔绝
	- 黄河中下游农业发展
2. **早熟**

	- 周秦之变（封建→帝制）
	- 卷入全球化

### 中国文明起源
- 夏王朝起源
	- **天下之中**（文明是在交流、碰撞中产生的）
	- 青铜、牛羊、小麦从**中东传入中国**，提高生产力，马，战车（**石器时代进入青铜时代**）
		- 人口增加
		- 组织力提高 → **国家**
- 中国→ 农业定居文明，稳定，加强血缘关系
	- 特点
		- **地理封闭**
		- 黄河土地肥沃
	- 血缘关系
		- 祖先崇拜（**夏商周的政治结构）
		- 血缘家族（ 自给自足、小农经济）
		- 家长制（文明是连续的）
- 希腊→ 海上贸易、商业、冒险，打破血缘纽带
	- 特点
		- **开放性、海岸线长**
		- 土地贫瘠
	- 打破血缘纽带
		- 神庙兴起
		- 奴隶社会（商品经济）
		- 突破式的
- 夏（部落）→ 商（发明分封制；重视神、占卜）→ 周（达到顶峰，通过**礼仪宗法制度**保证；信天命，意识形态改变；重现世，而外国重来生）→血缘纽带的扩大（氏族、部落、邦族 → 国家，血缘地位和权力地位是一体的）

### 周秦之变
- 西周
	- **礼治治国**，**贵族精神**（中世纪骑士精神），保持尊严和风度，到秦从贵族社会→平民社会
- **东周**
	- **春秋（争霸）**
		- 多极的政治格局和激烈的社会环境
		- 经济发展、国家知识（人才）交流频繁
		- 分封制不再维系
	- **战国（变法）**
		- **铁器出现，井田制瓦解**
		- 郡县制在楚国有雏形
		- 法家摧毁以礼乐为代表的贵族→强化集权，扩大军队
			- 用制度而非道德来治国；**理性主义、经验主义、现实主义**，政治与国家结合产物，功利主义
			- 围绕君主利益：通过严刑峻法（法，术，势）
			- 秦：蛮夷化程度深，法家改革成功
- 中国（统一）
	- 分久必合
	- 崇尚一元
	- 地理封闭、资源有限、不成功成仁
- 希腊（分裂）**
	- 崇尚多元**
	- 海岸线上、失败逃往海上、无法形成征服与屈服专制
	- 城邦制、小国寡民、可以实行民主
- 周
	- 分封制
	- 小共同体（熟人社会）
	- 横向：各宗主国国很少往来、不连贯
	- 纵向：我附庸的附庸不是我的附庸
	- 社会价值：父高于君
- 秦
	- 郡县制
	- 大共同体：陌生人社会
	- 重权直达社会底层
	- 贵族消失：官僚集团权利完全来自皇帝；血缘分家，整个社会原子化
	- 社会价值：君高于父

### 秦汉与罗马
- 周
    - 受天命制约
    - 受诸侯制约
    - 受权利和责任制约
- 秦
    - **郡县**
    - 三权分立（丞相、太尉、御史大夫，保护皇权，西方是制约最高权力）
    - **焚书坑儒**
    - **修驰道、筑长城**
- 汉
	- **外儒内法**（罢黜百家，独尊儒术，三纲五常）
	- 找秦灭原因：汲取过度→黄老之术，无为而治；思想文化建设
- 秦汉帝国
	- 同：
		- 青铜→铁；文明边缘→创造新文明
	- 异：
		- **政治制度**：一家一姓；选举
		- **社会结构**：郡县制，全能政府；自治
		- **文化思想**：焚书坑儒，独尊儒术；意识形态不统一（东罗马希腊、西罗马罗马文化）
		- **民族**：同化内汉；征服者与被征服者
- 罗马
	- 游牧民族入侵（马）
		- 五胡乱华→隋唐（合）→高度划一；语言文字（表意文字）固化；地理（黄河、长江）
		- 哥特人→西罗马灭亡（分）
			- 自治，没有形式中国意义的统一；表音文字，不稳定
			- 高山、河流分散

### 周期性崩溃
- 封建制
	- 稳定的统治者，世袭领地
	- 出现内乱，受害的是局部；但动荡和冲突，持续而连绵
	- 世袭（印度种姓制度、日本武士制度、欧洲贵族制度）
	- 西欧人口曲线平滑（局部，不引发全局崩溃）
	- 建立在契约关系上：有法庭；有社会保障（权责不对等）；有冲突可以向国王请愿，国王领主相互制衡
		- 大政府：税收重，福利好
		- 小政府：税收轻，福利差
		- 中国：税收重，福利差
- 郡县制
	- 地方长官利益在于升迁，暴虐百姓
	- 天下大乱，无一处幸免；承平时间持续，久分必合，创造众多的物质、精神成就
	- 公平选拔人才，流动、竞争（科举制）
	- **频繁农民起义→频繁改朝换代（独有），王朝寿命短**
		- **秦末农民起义→汉（200）**
		- **隋末农民起义→唐（200）**
		- **元末农民起义→明/清（>200）**
		- 在农民起义基础之上的王朝，彻底打破旧的社会结构→长寿

### 循环中的变化
- 秦朝建立了大一统郡县制度→终结贵族统治，民众原子化→荐举制（弹性大，粗糙标准+官员免税特权）→士族门阀→皇权衰弱，地方势力扩大 → **唐朝的封驳制度和世界制度：君臣共治** → 宋完善科举制度→君权稳定性提高，中央集权真正完善（宋以前贵族——半贵族社会，之后为平民 ）
	- 「杯酒释兵权（宋太祖）→崇文」
	- 地方权利—分四
	- 科举制解决地方势力分裂
	-  谏官制度，科举制让士大夫“以天下为己任”的意识强烈，朝省集议制度（集权决策）
- → 政治、物质文明达到高峰，商品经济、社会流动性，平等化（取消奴隶制）的最高峰
- → 元朝奴隶制恢复，君丞地位悬殊→明：废除宰相和相权→清：大兴文字狱
- 汉→宋：曲折但上升，宋达到顶峰。元→清：退化到全面衰落
### 欧洲与中国交错
- 中世纪早期欧洲
	- 西罗马灭亡带来无数的混乱、死亡和灭亡
	- **黑暗制度**：宗教裁判所、火刑（天主教组织结构本身带有专制倾向）
- 技术进步（早期奴隶制不利于技术进步：庄园自给自足，与农奴接触）
	- **重犁的使用**
	- 牛→马做牵引（畜力） 
	- 水力、风力代替人力
	- 耕作效率↑ → 经济发展、人口增加
	- 商业复兴、城市兴起、文化复兴
- 自治城市
    - 条件
	    - 贵族有特权而缺钱
		- 商人有钱而**被束缚**→ 花钱从贵族手中购买城市中享有治权
		- 武装斗争
    - 由城市会议管理城市，在议会决议下，城市可以立法、组织军队、发行货币、决定如何收税
    - 自由：封建主领地和庄园里，有各种规定束缚，城市里没有
    - 平等：市民在法律地位上平等
    - 诞生资本主义：反特权、反等级制，在平等、自由的条件下，经济要素才能流动起来
- 中国晚清产生资本主义萌芽（×） → 复杂社会现象：政治、经济、文化、整体性
    - 强大的中央集权
    - 对产权保护的不力
    - 过高的交易成本（税收、腐败）
    - 法律不完善
- **皇权不尊重私人产权**（（权力万能）抑商）
	- 迁徙富豪
	- **采购时：强买强卖**
	- **遇财政困难，公开掠夺个人财产**
	- 不可能产生完善的产权保护制度→ 经商和权力结合起来**
- **欧洲出现自治城市**（金钱万能）
	- **希腊——罗马文明**
	- **封建自治法**：（没有大一统郡县制，城市自治不同于封建自治）
	- **不存在对商人制度性打压**
		- **日本**：武士享有权力
		- **印度**：种姓制度
		- **中国**：学者占据政府
- **商业是推动社会发展的重要动力**
- 阻碍民营经济发展
	- 官营工商业（国家垄断经济）
		-  **增加财政收入**
		- **民资迅速↑，令政府不安，更好地控制社会**
	- 文化上的贱商
- **新王朝建立**→ **减税+放松管制**- → **经济繁荣**→ **抑商**→ **发展受到制约，财政困难** → **增税**→ **农民起义**
- **西方**
	- **经济原因**：自发形成，无规划
	- **自由**
- **中国**
	- **政治结构**：规划城市
	- **控制**（里坊制，宵禁）
	- **民营经济内部也存在等级**（主人与雇工，师儒学徒）
- 西方自治城市 
	-  **中央集权民族国家出现**（商人提供国王金钱，人才对抗贵族，由封建制→民族国家）
		- 西班牙的极端专制
		- 法国的集权专制**
		- 英国的君主立宪
		- 荷兰的松散联邦
	- 民族国家出现后，对海外事业进行资助，全球化开始，日不落帝国，海上帝国
	- 有限责任公司：荷兰，英国东印度公司

### 打开国门后的世界
- 日本：转型最顺利
	- **政治/权力结构：** 诸侯林立（郡县），至尊未必至强，至强未必至尊
	- **社会结构：** 世袭等级别（科举）  
	- **武工制度：** 文人治国
		- 读书不够，注重实际
- 中国朝贡体系  
	- **统治：** 赔本赚吆喝  
	- **政治：** 协调
	- **面子外交 → 错误决定
- 跨海而来的英国人和中国历史上传统蛮夷不同，**对接管政权不感兴趣**，**原意帮助进行内政改革，发展经济**  
- **殖民主义**  
	- 破坏旧的社会  
	- 建设新的西方式社会

	- 财富有限，你多我少
	- 征服有黄金白银地方
	- 本国封建体制渗入殖民地
- **荷兰、英国公司式殖民：**
	- 互通有无，通过贸易，创造更大财富
	- 蛮荒之地也开发、原材料市场
	- 自治（美洲）：乡镇→县→州→美国
	- 英国殖民地 → 发达国家；西班牙殖民地 → 发展中国家（政治遗产）
