- ### 因果胜于统计
- 统计信息被比率：当有与事件相关的信息时，忽视比率，叶贝斯定理失败

	- → 思维定式：提高判断准确性
	- → 在敏感情境中，社会规则与思维定式发生起冲突，不想根据团体的相关统计对个人下结论
- 系统1处理的事件中各项因素是有因果关系的，但在统计推理中这样的关系很薄弱
- 心理老师希望你学到：别人的存在会削弱我最初的责任感，责任感的淡化
	- A组不知道实验结果 → 预测结果反映了他们对人性的看法及情境理解
	- B组知道实验结果 ，了解基础比率 → 相信会为陌生人提供帮助
- 如果没有对社会环境影响力形成一个新的认识，就没有从实验中学到任何有价值东西 → 改变看法很难，改变对自身阴暗的看法更难 → 令人惊讶的个案影响甚大，是教授心理学的有效手段
- 通过计算选定样本大小，**而非听从自己的判断（应该）** 大数法则
- 普遍性偏见：**小数定律**（对事物的信任多于质疑）（如果一则消息没有马上被视为谎言，不管其可靠性如何，都会对联想系统产生影响**）

	- 有规律事件：并不只是偶然发生的，还是机械的因果联系或是人的意志的结果

- ### 少即是多
- 合取谬误：人们认为两个事件的**联合出现**比只出现其中一件事的可能性要大
- **典型性**属于一连串可能同时发生且联系紧密的基本评估，**真具典型性的结果**与**特征性描述**结合在一起 ，就会生成**最有条理的信息**，**最有条理的信息**可能性却不一定是最大的
- 人们对**更详细丰富的描述**做出的**概率判断更高**
- 预言家总会给其顾客设下陷阱：对情节加以详述会使其**更可信**，却**更不可能成为现实**
- 概率就像**经济价值**，是一种**总体变量**。单一评估会产生**少即是多**的模式，系统1会**取平均值而非累加值**  
	- 变量总体性对概率的影响小于对金钱的影响  
- 金钱
	- 单一评估：直觉反应
	- 综合评估：**逻辑原则**
	- 琳达
		- 单一评估：直觉
		- 综合评估：直觉也会推翻逻辑
### 回归平均值
- **奖励和惩罚毫无关系，是回归平均值现象。**
- 统计学角度
	- 对人友好而受到惩罚
	- **举止无礼而得到嘉奖。**
- 生活（违背统计学角度）
	- 当别人取悦我们，我们也会对他好。
	- 当别人对我们不友好，我们也会产生厌恶之情。
- 第一天和第二天不存在因果关系，只是数学问题，运气起了很大作用。
- 相关系数：两个值共有因素的相对比重 → **只要两个数值相关度不高，就会出现回归平均值的情况**
- 当发现有回归效应时，因果关系解释就会被激活，但回归平均值可以用来解释现象，却无法找出其中原因
### 驯化直觉预测
- **预测
	- 直觉：系统一，反复训练的技能和经验（直觉性预测）
	- 分析（修正）
- 受试者没有对未来的预测和当前的预测区别开来，预测时存在系统偏见，应该回归到平均值上来
- 无偏见预测
	1. 先做出判断平均值（基准）
	2. 根据对证据的印象算出与之匹配的平均值（直觉性预测）带有偏见
	3. 对证据和平均值的关联性做出估计（离开基准线靠向直觉）
	4. ②=①±①X关联度（以直觉为基础，削弱了直觉的影响，回归到平均值上）
- **直觉性预测**——真理。如果期待做出有效预测，那结果不会偏离平均值太多  
- 特征：信息只有在非常有效时，才允许人们对罕见或极端的事件做出预测
- 有时会歪曲判断来掩盖自己的无能无力，如果选择接受极端的预测来蒙蔽自己，会意识到对自己的纵容
### 认知错觉
- 眼见为实 → 光环效应 （**夸大评估一致性）→ 思维连贯性 → 必然性错觉
- 想看一个解释是否**外通**，可以做一个**最终测试**，看能使事件被预测到
- 宕机是不可知的：知道、预感、直觉**给人一种暗示（先见之明没有）：世界比事实更可知
- 后见之明
	- 人的大脑没有足够的能力重构过去的知识结构或信念。一旦接触新的世界观，丧失过去观点的回忆能力
	- 导致**你低估自己受往事影响的程度**
	- **恶劣影响**：导致观察者不是根据判断过程的合理性来评价判断的好坏，以结果为判断标准
	- 结果偏见：激进冒险→成功（运气）夸大
- 系统一的意义构建让我们认为世界比现实中更可预知、更富逻辑性
- 营销，成功类书籍，满足了系统一因果关系的追求，忽视了运气的决定性力量和回归的不可避免性
### 有效性错觉
- 对一段原则的认识，对我们个案的信心没有什么影响。**基础比率不会影响他们的判断**
- 自信是一种感觉，体现认知放松的一致性
- **认知错觉：行为改变**
- 技能错觉和有效性错觉：想在改变，行动未变，由专业文化来支撑，对技能产生无限放大感觉
- **预测错误不可避免**
- 不应该相信自信是准确性的指示器

### 直觉与公式
- 在有效性低的环境（不确定性和不可预测性），专家预测的准确性要低于简单运算的结果
- 专家预测不准确→ 要提升预测准确度，结果应公式给出
	- 总想跳出思维框架，将不同特征进行复杂结合
	- 对复杂信息的最终判断很难达成一致
		- **对极端条件的依赖**，环境中不被注意的刺激物在本质上影响我们的思想和行为
		- 由于对我们的思想中物东西没有一个清晰的认识，环境变化时，会做出不一祥判断
- 对大多数人来说，错误的原因最重要：由运算法导致的错误更疼痛，上升到道德层面
- 面试之所以失败，是因为它允许面试者按个人喜好做决定

- 技能习得条件→ 培养直觉
	- **可预测、有足够规律可循的环境**
	- 长期训练学习规律的机会
- 在环境缺乏可靠的规律时，不要相信直觉
- 能否培养直觉性专业技能，取决于反馈速度和质量，以及是否有足够时间练习

### 伯努利错误
- 简单的风险问题 → 选择
- 更复杂的决策问题 → **静态**
- 观察者思维中的主观量和物质世界中的客观量呈**对数关系**（心理物理学）
- 对财富变化的心理反应（效用）是**已积累财富**的反比 → 效用是关于财富的**对数函数**
- 人们的选择基于**心理价值**而非金钱价值
- 财富的边际价值递减**可以解释风险规避**问题
- 圣彼得堡悖论：风险期望无限大时，人们只愿花**几达**克来赌一把
- 伯努利的错误
	- 以静态参照点（当前财富状态）决定是否快乐
	- 而非财富变化决定快乐程度，可解释**风险规避**，却**不能解释对风险收益的偏好**
### 前景理论
- 效用是伴随财富的**变化**出现的，而非财富状态
- 在效用理论中，**盈与亏的区别被认为至关重要**
- 效用理论（财富状态→效用）
- 前景理论
	- 参照点：现状，期待结果
	- **损失厌恶**
	- 维度和财富变化中降低敏感度原则
- 损失厌恶系数：要想平衡100元的损失，需要得到的最少收益是多少（1.5-2.5）
	- ![[Pasted image 20251017232125.png]]
- 当得失都可能出现时，损失厌恶会**极力规避风险选择**
- 在肯定有损失和可能有损失中，降低敏感度会引发**冒险之举**
- 前景理论无法应对令人失望之事（建立了一个新的参照点，相对期望而言，什么也得不到不是损失）
- **禀赋效应**：个人一旦拥有某个物品，他对该物品价值的评价就会比未拥有之前大大提高。
- **无差异曲线**：曲线上每点位置的效用相同，你对它们的位置并不在意
	- ![[Pasted image 20251017232342.png]]
- **忽略参照点**：即所有时间点效用完全由当前情况决定，与过往不相关
- A,B位置稳定
	- 人们选择偏好稳定（无差异曲线）
	- 规避损失（前景理论）
- **无差异曲线没有预见到**
	- 人们的偏好并非一成不变，会随着参照点发生变化
	- 改变的不利之处比有利之处更突出；有利之处包括倾向于现状的偏见
- **前景理论**（参照点）可以解释禀赋效应：损失厌恶（是否拥有）系统1自主评估
- **交易商品**（禀赋效应消失：得到的快乐和放弃的痛苦间的不对称没有关联）商人
- 消费享受的商品
- 贫困的人处于前景理论中第三象限；总是处于损失中；身上没有禀赋效应
- 贫困的人意味着生活水平低于自己的参照点；参照在外界而非过去；得到的较容易是减少了的损失，而非所得
- 花钱意味着损失，在不同损失间做选择（穷人）
- 有得有失，交易（商人）

- 损失厌恶与大脑双系统结合研究：相较于积极与靠近而言，**消极**与规避更能占据主导地位
- 杏仁核（风险中心）对威胁性东西有强烈反应，快速通道，威胁优于机遇
- 威胁
	- 威胁字眼：战争、犯罪、呕吐
	- 不必的意见：“我认为安乐死……”
	- 负面情绪（损失厌恶）
- **长期健康饮食不仅依赖于寻求幸福，更在于避免负面情况出现**
- 损失厌恶系数
	- 参照点：现状；将来的目标。（没有实现即为损失，超过即为所得）
	- **规避损失的动机**
		- 人们对没能实现目标的损失厌恶比想要超过目标的愿望更强烈
	- 获得利益的动机

	- 理论：经济学逻辑：下雨天收入多，晴天多休息
	- 现实：损失厌恶背后逻辑：下雨天较快达成目标，收工早；晴天就慢达成目标，工作时间长
		- （短期目标会努力实现而非超越）

### 体验自我与记忆自我
- 快乐测试值，曲线下的区域，对各个时刻的体验做了加权（体验自我）
- **回顾性评级**：相较于其他时刻，会对两个单一时刻（高峰、末端）进行加权（记忆自我）
	-  **峰终定律**
	- 过程忽视：过程的持续对疼痛的评估没有影响
- 目标
	- 减少峰值和结束时的疼痛感，逐渐减轻比极具减轻更可取（减少痛苦记忆）
	- 迅速完成这个过程（减少实际体验到的痛苦）
- 在思考生命时，唯一能采取的观点来自**记忆自我**。
- **混乱的体验** + 对此体验的**记忆** → 认知错觉
- （经验自我无法表达自己的感受）+（记忆自我有时是错误的）→ 决策（×）→ 偏向反映出自身的爱好
- **经验自我** ←→ **记忆自我**
- **体验效用** ←→ **决策效用**（互相冲突）
- 系统1用平均值、标准和原型表示集合，而非统计

- 总体幸福感取决于**少数极易想到观点**；而并非取决于生活中所有领域的仔细衡量
- 目标对他们所做的事及感觉是如此重要，以致只关注**体验自我**是站不住脚的
- 幸福=记忆自我+经验自我
- 系统1会常会作为生活的一部分替代成整个生活，人们所关注生活的任意方面会在整体评估中被放大（**聚焦错觉**）
- 想要适应一个新的环境，无论是好是坏，很大程度上取决于慢慢地不去想这个环境
- 生活的长期状况（婚姻、健康）在部分时间会对人们的状态产生影响，即当人们关注这些状况的时候。
- 人们在**聚焦错觉**中犯的错误，包括关注选定的时刻、忽略其他片段发生的事、忽略对注意力的减退及对新状态适应**。
- **体验自我中注意力是关键**
- 经验自我的幸福感会随着收入↑而提升，但超过某阈值后就不再提升。
- **故事所关注的是有意义的事件和值得珍藏的时刻**，她的一生由一个典型的时间段代表的，与总体时间无关。
- （平均）典型可替代总体
- 大多数人对他们经验自我遭遇的痛苦都是漠然的，**旅行中只关注记忆自我，更少关注经验自我**。（不去回忆或穿越丛林，这些体验在当时大都是痛苦的，只有提醒自己这些身体的痛苦与快乐都是值得的，才能让自己坚持下去）
