# 大纲
# 思路梳理
- 拖延是作为一种身体的保护手段，下意识的反应来逃避不舒服的感受，这些不舒服的感受涉及
	- 恐惧
	- 希望，梦想，拖延为什么要逃避希望/梦想呢？
	- 记忆，那记忆应该是有好有坏，拖延帮忙逃避坏的部分吧
	- 怀疑
	- 压力
	- 这几个情绪中恐惧应该是核心，拖延涉及到对成功的恐惧，对失败的恐惧，对亲近与疏远的恐惧，对受控制的恐惧
		- 首先是对失败的恐惧
			- 首先是对失败这件事情的定义太过宽泛，上升高度，本来是一件事情的失败，直接变成人的失败，对人进行批判，让人产生挫败，所以就会为了避免失败，而不去做这件事情，形成拖延
			- 恐惧失败的假设
				- 自我价值=能力（智力上的能力）=表现
					- 表现直接决定个人价值和能力，表现不好否定自己的个人能力和价值
					- 拖延打断能力（个人价值）和表现之间的等号
						1. 首先避免出现**尽全力**仍表现不好，知道自己的能力上限，对自我进行批判，这样不需要面对自己个人价值的现实
						2. 这样当**拖延但表现不好**时，可以将表现不佳归因于未尽全力而非能力不足，拖过拖延打破链条之间的等式，但其实这个等式本身就是有问题的
						3. 当**拖延但表现还不错**时，可以维持自我形象，即等式成立，表现好→能力强→自我价值高
					- 这样可以让自己进可攻，退可守；当拖延表现差时，等式被打破，表现不和个人价值和能力挂勾；当拖延表现好时，等式成立，表现好=个人价值高=能力强
				- 对努力后失败的恐惧>拖延的痛苦
					- 这样避免清晰认识到个人能力水平，拖延成为缓冲垫
				- 将表现不佳的后果进行上升
					- 表现不佳=做人失败/不受欢迎，是深层恐惧的投射
			- 完美主义与拖延的关联
				- 首先拖延是对失败的恐惧，既然是失败，那与之相对的是成功，那怎么定义这个成功呢？失败的定义可能畸形，成功的定义也可能畸形
				- 完美主义者有两种，一种是适应性，一种是适应不良型
					- 适应性
						- 高标准+接受现实差距，推动进步
					- 适应不良型
						- 首先是制定不切实际的目标
							- 设定目标是为了让自己取得进步达到目标，不是让自己陷入沮丧和挫折
						- 其次是目标制定偏差
							- 即要么做到所有事，要么感到不够好，未完成=0价值，中途易因失望而放弃
						- 当完美主义成为一个问题的时候，拖延就很可能成为一个问题
				- 完美主义者的5大核心信念
					- 平庸招致轻蔑，事情必须做到极致
					- 优秀不用努力，应该轻松成功，而非付出巨大努力
					- 一切都得自己来，视求助为软弱
					- 总有一个正确答案，未找到完美方案前拒绝行动
					- 对“全知全能”的执念
						- 隐藏信念：认为自己应该“看透未来，掌控结果”，而非受制于现实
						- 现实冲突：拒绝接受人类的局限性（非全知全能），用拖延逃避可能出错的风险
				- 两种心态对拖延的影响（德威克理论）
					- 固定心态：逃避困难→避免失败，失败=个人价值否定→能力无法精进，导致拖延，关注个人价值，维持自我形象，所以会避开一切可能造成失败的挑战和困难，而拖延就是这个手段
					- 成长心态：拥抱困难→失败=学习契机→提升个人能力，对抗拖延，关注进步而非证明个人能力
				- 完美主义者的思维特征
					- 绝对化思考：对事情结果有极端化判断
						- 完美主义的一大问题就是设定不切实际的目标，需要设定现实目标，允许不完美的进步，避免“要么全部有要么全部没有”的极端思维 
					- 灾难化预期：夸大微小错误产生的后果
						- 具体表现
							- 模糊恐惧：不够完美=生活悲惨的莫名幻想
							- 连锁式灾难推理：从单一事件缺陷推导出极端负面结果
							- 心理瘫痪：过度担忧导致行动力丧失→拖延
						- 应对方向
							- 明确化恐惧：将模糊的灾祸具体描述出来
							- 现实检验：审视灾难化预期与实际可能性的差距
								- 退后一步，重新审视预判，对所谓的“命运”的“必然性”另眼相看
					- 固定心态驱动：将不完美=彻底失败
						- 将表现=能力=个人价值的链条假设打破，区分事情和个人价值，不要划等号
						- 从固定心态转向成长心态
							- 困难，失败，能力
		- 其次是对成功的恐惧
			- 定义（多元）
				- 社会标准：地位，收入，影响力
				- 关系维度：家庭幸福，友爱关系
				- 个人感受
					- 满足感
					- 及时追随目标，过程努力本身就是成功
			- 文化压力对拖延的影响
				- 这些压力有双面性，即对一部分人有利，但对另一部分人构成限制
				- 普遍作用机制
					- 文化规范
						- 抵抗型拖延（主动性拖延）
							- 拖延→拒绝“忙碌至上”，主动放弃“拥有一切”
						- 渴望型拖延（被动型拖延）
							- 渴望成功→拖延→无法实现目标
					- 性别
						- 女性的成功焦虑（”野心“标签）
						- 男性的情感压抑
			- 无法追求成功的原因
				- 内心冲突
					- 对文化压力的抵抗
					- 对文化压力的适应性障碍
				- 害怕成功所带来的不利一面
				- 焦虑通常极为微妙，不能直接被感知到
			- 拖延→避免成功到来
				- 拖延的负面影响
					- 无法及时追随目标→自我失望→感到失败
					- 成功后难以享受喜悦，被冲刺过程抵消
					- 自我否定：认为成功是赶出来的，非真正能力
				- 关键问题
					- 不在于是否对成功具有摇摆不定的心理
					- 而在于成功所引起的冲突是否强烈到足以阻碍通往成功的道路
			- 逃避成功的心理原因分析
				- 成功需要付出太多
					- 竞争恐惧：拖延→降低成功机会→降低别人的期待
					- 承诺恐惧：担心承诺后卡在“成功扶梯”上无法下来→拒绝迈出第一步，拖延→原地踏步
					- 工作狂焦虑：拖延→避免失去对生活的控制权
				- 成功是危险的
					- 害怕伤害他人：成功=侵犯他人，忽略他人承受力
					- 文化家庭隔阂：与原生家庭价值观脱节
					- 恐惧遭受攻击：职场：弱肉强食→不提升职位→避免权利争斗，认为自己无力还击
				- 成功是禁区
					- 不配得感（成功=“罪过”的自我惩罚）
						- 真实过错内疚
						- 想象过错内疚
					- 自我否定
						- 自我观念：“我的”和“非我的”，成功属于非我的，避免任何证明自我价值的机会
					- 完美恐惧悖论
						- 矛盾心理
							- 渴望完美→优越感
							- 因完美遭嫉妒→保留致命缺陷→“不配拥有成功”的自卑感→能力无法与外界期待匹配
						- 对特别的依赖
							- 拖延成为一种“选择缺陷”策略：并非做不到，只是不想做，维持若全力以赴就能成功的幻想
							- 普通=被爱，追求卓越=被孤立，陷入“成功与爱只能二选一”的思维陷阱
			- 应对方向
				1. 重新检验“成功必遭孤立”的核心假定：恐惧≠事实
				2. 接受成功伴随的自然变故与未知风险
				3. 理解改变是渐进过程，允许前进两步后退一步的波动
				4. 区分真实风险与想象风险，关注成功带来的积极改变
		- 接下来是对受控制的恐惧
			- 拖延是部分人获取掌控感的策略，表现为对外部规则和他人要求的抗拒，可能是有意识的权利斗争，也可能是无意识的条件反射
			- 触发场景与自我觉察
				- 规则冲突：在必须顺从规则时拖延
				- 角色压力：在试用期，留校查看期是否拖延
				- 人际影响
					- 是否因拖延导致他人日程调整或工作负担加重
					- 他人是否对拖延表现出烦躁，愤怒或放弃施压？
			- 典型类型
				- 规则对抗型：抵抗外在限制对“自我感”挤压
				- 权力平衡型：存在权利等级关系
				- 边界防御性：抵制侵犯感，维护私人空间
				- 冒险刺激型：通过边缘状态获取快感，享受最后一刻完成的刺激，将生活置于崩溃边缘
				- 报复宣泄型：感知到伤害背叛，用拖延让对方付出代价
				- 现实逃避型：拒绝接受无法控制/限制他人，无人拯救的现实
		- 最后是对亲近与疏远的恐惧
			- 拖延是维护心理舒适区，调节人际亲疏关系的策略，帮助个体维持心理安全感与舒适感
			- 心理舒适区的人际边界需求
				- 关键问题
					- 涉入他人生活的程度
					- 人脉维护的广度
					- 关系投入的深度
					- 独处与共处的时间分配
				- 离开舒适区会引发不适，需要通过策略回归平衡
			- 对疏远的恐惧
				- 心理动因：自我完整感的缺失
				- 行为模式
					- 需要帮助型：无法独立完成任务，需外部框架指引
					- 第二顺位依赖型：回避主导角色，偏好副职或从属，恐惧承担独立决策后的孤独感
					- 紧急呼救型：主动制造危机情境，期待他人救援
					- 关系滞留型：不结束不再有益的关系，熟悉的痛苦替代未知的孤独
				- 深层心理补偿机制
					- 重温旧情：通过冲突重现家庭互动模式
					- 忠实伙伴：将拖延作为心理寄托，用未完成事项填充孤独感
				- 替代方案
					- 养宠物，建立社交关系，书写日记等
			- 对亲近的恐惧
				- 行为模式
					- 依赖“雷达系统”扫描环境，警惕被侵犯
					- 他人逼进时采取撤退行动：拖延，回避
					- 对人际距离要求难以明确表达，易因边界被踩而心怀怨恨
				- 深层心理动因
					- 恐惧被消耗
					- 害怕荣誉被窃取→拖延→避免被侵占
					- 创伤经验与防御：亲密=咨询烦恼，拖延保护自己不受伤害
					- 对“内在阴暗面”的焦虑：担心亲密关系会暴露不为人知的负面特质，他人无法接纳真实的自我，彻底回避关系
					- 对完美关系的矛盾渴望：期待完美伴侣，认定不存在，拖延→拒绝开始
				- 应对：平衡独立与依赖
					- 关系的本质需求
						- 平衡独立与依赖
						- 分歧解决是拓展自我认知的途径，而非威胁
						- 主动学习表达边界，而非被动逃避
					- 拖延的代价
						- 虽能维持心理舒适边界，但阻碍个人成长与深度连接
- 拖延的相关研究结论
	- 无关智力
	- 性别差异：男人比女人在拖延问题上略胜一筹
	- 年龄影响：年纪越大，拖延越少
		- 不想浪费有限时间
