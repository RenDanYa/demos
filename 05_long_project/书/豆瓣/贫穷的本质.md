# 个人笔记
- **总体原则**
	- 对于几乎无钱可投的人来说，一旦收入增长速度慢于需求，就会掉入**贫困陷阱**
	- 如果穷人快速增收潜能很大，并且随着富裕程度而提高，**“贫困陷阱”** 也就不复存在了
- **曲线描述**
	- **S形曲线**：存在 **“贫困陷阱”**
	- **反向L形曲线**：不存在 **“贫困陷阱”**（施舍并不能永久提高收入，只能让他们前进得更快、不改变前进方向）
	- ![[Pasted image 20251015171258.png|300]]
### 贫困陷阱
- 隐含假设：如果穷人有机会可以吃得好一点，就能做一些有意义工作，走出贫困陷阱地带
- 实际情况
	- 没有绝对匮乏，没有人挨饿
	- 多余钱购买其他必需品
	- 即便花在食物上的钱，也并没有全部用来增加能量或微量元素，而是获取味道更好的能量
- 卡路里生产能力关系并不是S形曲线，而是反向L形曲线 
	-  一旦人们能够吃饱饭，收入就不会产生大幅度跃升；与不那么贫穷的人相比，摄入更多的卡路里对非常贫穷的人来说更有好处
- 为什么穷人吃得少
	-  当穷人告诉自己应改变饮食结构时，持怀疑态度，钟爱自己吃的食物
	-  并不了解微量元素重要性，在大多数情况下，食用既不会发生小变化，也看不到大变化。
	-  比饮食更重要的东西 ① 让自己生活少一点乏味 ② 对于美好生活基本需求
	-  怀疑那些想象中的机遇，怀疑其生活产生根本改变的可能性。（想法：任何改变都需要花很长时间 → 关注当前）
- 基于营养说的`贫穷陷阱`的基本机制似乎并不适用于成人，但并不意味着穷人在营养方面没有问题
	-  相反，对儿童及孕妇的直接营养投资，会产生巨大社会回报
### 健康陷阱
-  **原因**
	- 人们对**蚊帐**（消毒剂）价格很敏感，对**收入**却不敏感。要想移到S形的右半部分，即通过收入改善健康状况。收入增长远高于价格。低需求量表明：健康问题依旧难以得到根本性的解决：跳出“贫穷陷阱”的“梯子”是存在的，但并非总放在了正确的地方，人们不知道或根本不想踏上“梯子”
	- 常把钱花在昂贵的治疗上，而非廉价的预防
	- 治疗找私人医生，而非政府免费提供的医生和护士 → 系统运转不起来
		-  政府保健服务者高缺席率 ② 动力的缺乏
	- 人们根据价格判断质量：廉价→没有价值（廉价的预防）[实验显示补贴→高使用率，影响不大]
	- 脆弱的信念，病急乱投医，与真正的坚定信念相比，基于方便与舒适的信念更加灵活（并不了解信念，不坚定）
	- 必要的希望。
	- 好处只能在未来某个时刻实现，费用却需要今天来支付。“时间矛盾”会阻碍我们将目标转化为行动，让我们无限期地延迟决定，我们天真自信地以为，自己有能力在未来做正确的事 → 助推，免费提供预防性服务或是奖励那些利用这种服务的家庭，使其成为“默认选择”
	- 对生活在富裕国家的人来说，周围充满了无形的推力。几乎用不着有限的自控及决断力，而穷人则需要不断运用这种能力
### 教育陷阱
-  供应学校教育，让孩子上学是非常重要的一步，是学习的起点，然而并没有规定孩子们必须在学校里学到东西，只说他们应完成的基本课程。
- 需求达人：
	-  除了有明确需求，否则没必要提供教育，这一结论涵盖了几十年教育政策缺点
	-  教育不过是另一种形式的投资：投资于教育，目的是得到更高的收入。增加未来收入
	- 从家长角度看
		- 1.教育不仅是一种投资，还是赠送给孩子礼物
		- 2.家长有权支配孩子——决定是否上学，工作等
	-  教育的经济回报固然重要，但其他因素影响忽视 → 当孩子达到一定年龄，必须送他们上学。
- 父母收入对是否进行教育投资很关键，让市场操纵一切（需求方观点），并不能让每个孩子根据自己的能力接受教育。除非家长们能完全清除收入差距，否则供应方干预必要的
	- ← 供应策略：公立学校、义务教育，有成效但教育质量并不乐观
	- ← 需求策略：私立学校，比公立孩子学到更多，但质量并非完美无缺。市场没有发挥其应有的作用
	- ← 关键问题：对教育成果的具体期望会歪曲家长们的需求；期望包括名公立/私立学校价值孩子们的成就，及随之而来的巨大浪费
### 期望的诅咒
-  家长
	-  期望的致富途径是一份政府工作，其次是某种办公室工作
	-  将教育看成一种彩票，而不是一种安全的投资  
-  并非学生没有雄心壮志，而是当家长+老师对孩子们取得好成绩期望不高，相信S形曲线，使人们选择放弃，不相信孩子能够跨过顶峰，进入S形曲线陡峭部分 → 一条实际不存在的S的曲线
	-  1. 除非家长们愿意平等的对待每一个孩子，否则他们只能孤注一掷，将所有希望寄托在他们认为最有前途的孩子身上，确保这个孩子接受足够教育，而非教育分摊
	-  2. 教育每个阶段都是有价值的（经济，非经济利益）
		-  家长行为产生一个本来不存在的`贫穷陷阱`
		-  只要阶层不被突显，低/高阶层孩子一样好，一旦依阶层得到提醒，表现变差
- 学校失败
	-  为某种艰难的公共考试培养优秀的孩子。因为那是通往更大成功的踏脚石，大多数学生跟不上
	-  官方教学方法，及过于注重教学大纲带来了很多限制，造成了障碍。
	-  体系有失公平，浪费资源、人才浪费。
-  人才能的挖掘不存在捷径，花费大量时间，研究教育体系做的事 
	- ①给予人们足够的机会展示自己长处。
	- ②给予每个人一套健全技能
	- ③发掘人才
- 降低家长们的期望（设定一些更为直接的目标，避免专注于难以定性的多年以后的成果）+注重核心能力的开发+使用技术辅助或在必要时利用教师代替的教学方式。
- 学校的服务对象是其现有的学生，而不是他们可能想要的学生。

### 人口陷阱
-  人口政策观点
	- 国家拥有的资源基本上是固定的，人口增多使得国家变得更穷
	-  降低生育率可使每个人资源增多 ← 马尔萨斯理论
- 仅靠理性不可能解决问题
	- 生育率高 → 贫穷 （×）
	- 贫穷 → 生育率高（√ 恶性循环）
	- 高生育率 → 人均资本投入 ↓ → 贫穷（跨代传递贫穷）证据不足（可能孩子没受委屈，目前吃亏）
- 生育控制权 
	- 避孕渠道或许可向人们提供更方便的节育方式，但似乎并不能降低生育率
	-  扩大避孕渠道好处在于，帮助青少年推迟怀孕期
	- 很大程度，穷人对于自身生育、性欲、节欲方面的选择都极为谨慎，如果明知怀孕对自己来说代价很大，却仍这样选择，说明他们是主动的
	- 从最明显层面看，他们能控制 —— 生育决定是选择平衡；同时，他们无法立即控制的某些因素也影响了他们的做出选择：女性或许会受到生更多孩子压力；这种压力来自于她们的丈夫、婆婆、或社会准则（经济利益体现，不错经济投资）
- 对很多父母来说，孩子是他们的经济未来：一种保障政策，一款存储产品，也算是种彩票保障；生育率↓，储蓄率↑ → 养儿防老观点，但还在少并不意味着他们越健康或受教育程度高 → 给孩子投资只能得到较低回报会减少投资
- 家庭特点不在于成员间契合度有多大，恰相反，他们遵守社会承认的简单规则，要社会规则强化，政策地位并不是完全取代家庭，避免政策滥用，发挥家庭功能

### 贫穷的风险
-  穷人生活充满风险，且同样一场灾难，对他们造成更大的伤害 ①财富减少 ② S形世界一场灾难总无法承担后果
-  陷入‘贫困陷阱’，这一过程伴随心理过程强化 
	- 1.丧失希望，降低自控力
	-  2.担忧 → 压力 → 抑郁，难以集中注意力，降低效率。
		-  皮质醇水平和决策力大小
-  应对
	-  1.增加工作量
	-  3.业务多元化，缺乏效率，难以成为任何一个领域专家
	-  4.保守经营，降低风险
	-  5.互相帮助，力所能及范围内，出于道德责任感 （程度高）
		-  有些风险未得到规避，没有提供更多帮助
			-  1.避免道徳风险（被帮助人产生惰性心理，提出不合理要求）
			-  2.相互帮助许诺并未真正实现
			-  3.有些问题在道德责任感外，需要更高社会契合度
- 保险
	- 非正式互助网络的保障有代价，有昂贵的风险及局限性。
	- 正式保险
		-  1.很难对保险公司进行有效管理，也很难对被保险人进行监督
		-  2.道德风险
		-  3.欺诈行为
- 穷人如何买保险
	-  政府破坏市场，灾难降临时，伸出援手
	-  时间不一致，当支付保费，未来收益
	- 可信度缺乏
### 贷款
- 正规渠道，利率高（不愿与穷人接触）
	- （违约率高（评估难度大+担保））
	- 避免违约：收集借款人信息，费用不以贷款多少衡量，即使贷款额小，也必须收集信息，监督费用由利息抵消，利率更高
	- 乘数效应：利率↑ → 借款人不愿还贷款 → 更细致审查，成本↑ → 利率↑（穷人将积蓄存在银行，而非贷给放债人）
- 收集借款人信息产生费用，竞争也未使利率下跌
- 小额信贷
	1. 紧密监督客户
	2. **动用关系网络，施压** → **降低贷款管理费用，降低利率**
	3. 利率谨慎，追踪还款情况
	- 局限
		- 企业家精神常意味着冒险，小额信贷不容忍任何失败
		- 阻碍人们选择赚钱慢的项目
		- 分担责任，每周还款，不灵活
		- 对生活产生影响，但没迹象表明发生了彻底转变
- 小额信贷坚持零违约率，对违约率放松限制会产生破坏性后果
- 成功源于将还款设定为一种固定社会契约
	- 所有人都还款
	- 小额信贷不是一种发现企业家有效方式，包括进一步创建更大企业，扩大规模的企业家
### 在钱心理学
-  时间矛盾：在花钱同时，计划在将来省钱，希望“明天的自己”比“今天的自己”更有耐心
-  缺乏自求控制，最可靠方式是加以强硬的规定（妨碍别人也防自己）
-  有些方法可以解决自我控制问题，但要利用这些方法，需要掌握自我克制
-  意识到问题并不意味着这些问题就能顺利得到解决，只是预见到会在哪方面遭遇挫折
-  自我克制难 
	-  →减少将来受到诱惑可能，当下消费，存不下钱
	-  ← 自我控制像一块肌肉，用多会感到疲惫
-  存款行为关键取决于人们对未来预期

	-  2.在大多数情况下，这些小生意赚不了多少钱
