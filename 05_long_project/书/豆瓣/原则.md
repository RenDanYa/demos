# 个人笔记

### 理解人与人大不同
- 任何人的单一体验本质上都是存在扭曲的
1. 明白你与其他人的思维方式能带来的力量 
	- 很多心理差异都是**生理性的** 
		- 并非有意采取低效率做法，而是由大脑运行模式决定的
	- 我们拥有各种天生特性：既可能帮助也可能伤害自己，取决于如何运用
		- 如果你不了解人（包括你自己）的特性就对他们抱有期待，肯定会遇到麻烦
		- 大脑“思维机器”运行模式
			1. 人的复杂程度超出我们的想象
			2. 新生儿的大脑里包含数亿年进化的成果
			3. 人类自视为最高级物种：过度强调自身优势重要性
			4. 所有脊椎动物脑“进化自下而上”：爬行动物是最古老的，顶层区域是最新的
2. 有意义的工作和有意义的人际关系不仅是我们做出的美好选择，而且是我们天生的生理需求
	- 人的逻辑特征是**社会智商**，而非情商
	- 组织里，个人利益与集体利益哪种力量胜出，取决于文化，文化又取决于人
	- 人脑的进化史是从本能、关注自身发展为更抽象、更关注普遍事物
3. 理解大脑里主要斗争，及如何控制斗争，以实现“你”的愿望 
	- ① **意识与潜意识**在不断斗争 
		- 意识：逻辑
		- 潜意识：指示有合理的进化意义上的理由 
			- 潜意识与意识连接，加以训练，形成通信流
	- ①**要知道最常发生的斗争是情绪和思考斗争。**
		- **杏仁核** ：“绑架”来得快去得快，放任自己做出本能反应，很可能会反应过度。
		- **前额皮层**：拖延抵制**杏仁核**发出的信号
	- ②**调和你的情绪和思考**
		- 情绪：先爆发后平息 
		- 思考：变为稳定持久，分析什么导致“杏仁核被绑架”
	- ③**培养你的习惯**
		- 信息
		- 形式/常规
		- 奖励
	- ④**坚持友善地训练“较低层次的你”，以养成好的习惯**
		- 并非让“较高层次自我”和“较低层次自我”斗争，争夺控制权，而是训练潜意识的，情绪性的自我。
	- ⑤**理解左脑思维和右脑思维的差别**
	- ⑥**理解大脑可以改变的程度**
		- 从身体锻炼到学习冥想，都能给人带来物/生理性变化，但并非有无限灵活性
4. 认识自己和他人特性
	1. ①内向/外向 
	2. ②直觉/感觉
	3. 思考/感觉 
	4. ④计划/发觉
	5. ⑦**创造者/推进者/改进者/激励者/变通者** 
	6. 负责设定任务与关注目标
	7. 职场人格量表
	8. ⑧**塑造者是能从构想一路走到构想实现的人**（塑造者=理想家+务实思考者+坚毅者）
		- **最需要勇气不是驱使你战胜别人的勇气，而是不管别人对你有何期望，始终坚持做真实的自我的角色**
5. 无论要实现什么目标，让合适的人各司其职以支持你的目标，是成功关键
### 学习如何有效决策
1. **认识到影响决策效果的最大威胁是有害情绪**
    - 决策流程
		- ①先分解 
		    - ↓↓↓了解决策基础知识，包括相关事实（**提升**）
		    - 对事实背后因果机制的**宏观**理解
				1. 综合分析 
				2. 直接结果/后续结果
		- ②后决定（忌先决策，后挑选与决策相符的证据）
2. **综合分析眼前形式**
    - 你能做的最重要的决定之一是决定问谁
    - 不要听什么信什么，区分观点/事实
    - 所有东西都是放在眼前看更大
    - 不要夸大新东西的好处
    - 不要过度分析细节（**点**），看到大局
3. **综合分析变化中的形式**
    - ①始终记住改善事物的**速度**和**水平**及两者关系
		- ![[Pasted image 20251015123448.png]]
    - ②不必过于精确（避免陷于例外而忽略常规）
    - ③谨记“80/20法则”，明白关键的20%是什么（同样：80%努力→20%价值，不陷于细节）
    - ④不要做完美主义者（边际收益有限）
4. **高效地综合考虑各个层次**
    - 每天在不同层次间转换，一个是价值观，另一个是你为实现价值观做的事
	- ①用**基线以上**和**基线以下**来确定谈话位于哪一层
		- 基线以上：**关注要点**：有序走向结论
		- 基线以下：**关注分点**：补充细节
		- 分析混乱、迷茫→**基线以下**细节
	- ②决策要在合理层次做出，但也应在各层次间保持一致
		- 任何问题都存在很多层次
		- 针对问题，需分层研究哪个层次
		- **有意识在不同层次间转换，而非把问题视为无关联、区别的事**
		- ![[Pasted image 20251015124441.png|300]]
	- **决策**
		- 以**证据和逻辑为基础**
		- 以**潜意识和情绪为基础**
5. 综合分析现实，理解如何行动的最佳工具是**逻辑、理性和常识**
6. 根据预期价值计算做决策
    - **会赢的决策通常具有正向预期价值的决策（奖励×概率＞惩罚×概率）**
### 做有意义工作，发展有意义人际关系
有意义人际关系 → 文化（信任支撑）
1. 忠于共同使命，而**非对某人三心二意之人**
2. 对相互交往要清楚
	- 确定原则，等价交换含义，是否达成共识
		- ① 确保每个人理解公平合理与慷慨大方的区别
		- ② **员工多体谅他人，少向别人索取**
		- ③ **清楚界线在哪里，站在公平的另一端**（别人）
		- ④ 酬劳与工作相适应
3. 机构规模过大会对建立有意义人际关系构成威胁
4. 记住很多人只是假装为你工作，实际在建立一己之利
5. 珍视诚实，能力强，表里如一的员工
### 打造允许试错的文化
- 犯错 → 痛苦 → 有问题（如果回顾往事，没有为自己的愚蠢震惊，没有吸取足够多教训）
- 隐瞒错误 → 形成不诚实、削弱学习成长能力的文化
1. 意识到错误是事物演变中的自然选择部分
	- ① 失败 → 好事（向极限挑战）
	- ② 不要懊悔，要珍爱它（吸取免于未来犯类似错误）
2. **不要纠结于一时成败，放眼于达成目标**
    - 不要纠结于“埋怨”还是“赞美”，专注于“准确”还是“不准确”（除了作为未来教训，不要纠结）
3. **观察错误模式，判断是否因缺点引起**（**缺点会以错误的形式体现出来**）
    - 分析错误 → 面临挑战 → 缺点
4. **在经历痛苦后要反思**
    - ① **痛苦+反思=进步**（面临痛苦“战或逃”，冷静下来反思）
    - ② 没有人能客观看待自己，每个人都有责任给予别人诚实反馈，让其承担责任，以开放心态解决分歧
    - ③ 教导并强化吃一堑长一智
5. **知道哪些错误可以接受，哪些必须避免，不要让员工犯不可接受错误**
### 求取共识并坚持
- 问题原因：① 简单的误解 ② 根本上存在分歧，回避冲突也回避了解决冲突的机会
1. **冲突→良好的人际关系**（冲突→原则是否一致）
    - 要不吝时间与精力，最佳投资
2. **知道怎样求取共识和掌握分歧→达成决策**（**了解最终决策权归属何方**）
    - ① 把可能的分歧摆上台面 
	- ② 区分苍白抱怨和有助于改进工作的诉求 
	- ③ 记住故事的另一面
3. **保持开放心态，也要坚定果断**
    - ① 区别心态开放/封闭的人 
	- ② 远离心态封闭的人 
	- ③ 提防那些羞于承认自己并非无所不知的人
    - ④ 工作负责人以开放的心态对待问题和他人意见 
    - ⑤ 寻求共识双向负责 
    - ⑥ 实质重于形式（沟通形式、批评方式）
    - ⑦ **提建议、提问题与批评不一样，别混淆**
	    - 提建议：并未下结论说有错，只是想确保对方已考虑所有风险
	    - 提问题：确认某人没有忽略什么事
4. 主持会议，把握好对话
    - ①明确会议主持人和会议的服务对象（**为实现某个目标为目的**）
    - ②表达清晰 
    - ③根据目标/**优先次序**确定采取什么样的沟通方式（**拒绝群、个体思维**）
    - ④主持讨论时要果断、开明 
    - ⑤在不同层面的讨论中穿梭对照
    - ⑥避免跑题 
    - ⑦维持对话的逻辑性（**与情绪化讨论比，逻辑清晰，交换意见，不会轻易被终止**）
    - ⑧不要因集体决策而丧失个人责任 ⑨运用两分钟法则避免持续被别人打断
    - ⑩当心讲话不易置疑的“快嘴王”（**别人来不及评估、批判**）
    - ⑪善始善终 
    - ⑫总结：结束讨论
    - ⑬行动清单 
    - ⑭分配任务
    - ⑮截止日期
    - ⑯运用沟通手段
5. 伟大合作如同演奏 1+1=3；3-5人效率＞20人 
	- 边际效益递减
    - 互动效率更低
6. 珍惜志同道合者
7. 无法调和分歧——是否值得维持
### 做决策从观点可信度出发
1. 采用创意择优，确保每个人观点长处
2. 关注可信度高，与你观点不一致的人，理解其推理过程（**分析人们可信度**）
3. 考虑扮演老师、学生、同事中哪个角色（**学生理解老师比老师理解学生更重要**）
4. 了解人们提出意见的过程和逻辑
	1. 无论向谁提问，对方一般会提供**答案**，仔细考虑向谁提问
	- ② 提防以“我认为…”开头的发言
	- ④ 系统梳理员工工作记录，评估其可信度
5. 处理分歧需高效
	- ① 适时终结辩论，推动就下一步措施形成共识
	- ② **可信度可作为工具，不能取代责任人决策**
	- ③ 没时间全面检视每个人观点，**需筛选最有可信度观点**
	- ④ 若由你负责决策，可信度加权后得出结论，**需和自己想法做比较**
6. 每个人有权和义务设法了解重要的事
    ① 沟通是为了获得最佳回应，**故应与最相关的人沟通**
    ② 以教育或促成共识为目标，**不仅是为了获得最佳答案，故应让更多的人参与**
    ③ **认识到没有必要凡事做出判断（不了解）**
7. 关注决策机制是否充分，而非是否如你所愿
### 知道如何超越分歧
1. 达成相互协议时不能忽视原则
2. 不要把公平辩论的权利与决策权相隔离，**要有大局意识**
3. 不要对重大分歧不闻不问
	- ① 则被琐事惊扰
	- ② 不要被分歧束缚
		- 投票
		- 上级制定
