### 合同核心条款

- [ ] 合同中定好拍摄总张数、入册张数、底片是否全送、加片费用
- [ ] 定金慎重支付，不一次性交完全款
- [ ] 看客片确定实际效果
- [ ] 确认是否提供免费化妆及化妆品，写清价格和数量
- [ ] 合同中写明照片是冲印还是喷墨打印
- [ ] 合同中注明摄像师、化妆师是否分等级及对应费用
- [ ] 口头承诺的赠品写进合同，多选精修等实惠服务

### 付款方式

- [ ] 确认详细付款方式、付款时间及尾款时间
- [ ] 确认现金还是刷卡，刷卡是否有附加手续费

### 化妆与服装

- [ ] 确认化完妆不喜欢是否可以改主题或妆容
- [ ] 确认改妆是否免费，是否可以随时跟化妆师沟通
- [ ] 确认婚纱礼服是否任意穿及具体套数
- [ ] 确认外拍服装清洁费是否额外收取

### 后期与服务

- [ ] 确认后期制作是否收费，精修是否修到满意为止
- [ ] 确认增加后期成品制作如何收费
- [ ] 确认相册排版设计是否提前给确认
- [ ] 确认初版不满意是否免费修改
- [ ] 确认外景是否一对一服务
- [ ] 确认外景交通、吃住行、门票、停车费等由谁支出
- [ ] 确认恶劣天气无法拍摄时的处理及补拍时间
