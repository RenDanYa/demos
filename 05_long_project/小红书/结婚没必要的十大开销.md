### 婚礼开销规划

- [ ] 结婚前梳理可节省的十大开销清单
- [ ] 优先把钱花在婚后日常能用到的地方
- [ ] 婚纱照不额外加精修和相框相册，套餐精修足够
- [ ] 床品准备 2-4 套，注重品质
- [ ] 婚纱礼服先了解酒店免费提供的款式，再决定是否租赁或定制
- [ ] 婚鞋选择一双可日常搭配的白色鞋款

### 婚礼服务与用品

- [ ] 先找婚庆，根据婚庆推荐选择酒店或场地
- [ ] 考虑婚庆+四大金刚+酒席打包服务
- [ ] 纸质请柬控制数量，主要用于领导同事
- [ ] 确认手捧花和胸花是否由婚庆提供
- [ ] 接亲套房对比婚庆赠送的大床房与升级套房性价比
- [ ] 婚纱礼服优先考虑租赁，不必花大价钱定制

### 珠宝首饰

- [ ] 三金选择日常可佩戴的经典款
- [ ] 钻戒不保值，可考虑性价比方案
- [ ] 对戒选择日常可长期佩戴的款式

### 其他开销

- [ ] 车队不必花大价钱，头车安排好即可
