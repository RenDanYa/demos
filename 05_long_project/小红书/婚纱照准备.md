### 拍摄前一个月准备

- [ ] 拍摄前一个月开始运动减肥
- [ ] 拍摄前做好防晒，避免油腻辛辣食物，防止长痘
- [ ] 拍摄前 3 天染发（黑茶色偏棕），保持头发颜色统一
- [ ] 拍摄前在网上做攻略，确定自己喜欢的风格
- [ ] 拍摄前多练习微笑，保存好看的 POSE 学习
- [ ] 拍摄前多与摄影师沟通
- [ ] 提前准备好需要自带的道具
- [ ] 拍摄前做好身体和面膜护理补水

### 拍摄前一到两天准备

- [ ] 前一晚保证充足睡眠，睡前不饮酒
- [ ] 前一晚敷面膜、刮腋毛/胡须
- [ ] 不带贵重物品，不提前化妆
- [ ] 准备乳贴或 NUBRA
- [ ] 做好美甲
- [ ] 提前熨烫婚纱礼服和衬衫
- [ ] 试穿婚鞋，准备创可贴、高跟鞋贴、润滑剂
- [ ] 洗头不用护发素和发膜
- [ ] 男士提前修鼻毛、刮胡子
- [ ] 避免带多余家属，以免延长拍摄时间
- [ ] 准备好拍摄当天的早餐

### 婚纱店选择

- [ ] 选婚纱店时先确定自己喜欢的风格再选店
- [ ] 到店后不要因为便宜而将就，多去后面看看婚纱
- [ ] 确认化妆品品牌，避免过敏
- [ ] 要求看客照，确认皮肤修图效果
- [ ] 对比婚纱店时重点看后期服务

### 男生物品准备

- [ ] 准备黑白船袜各一双，彩色袜子也带上
- [ ] 准备黑色皮鞋和小白鞋
- [ ] 准备备用白色衬衫
- [ ] 准备领带、领结等配饰
- [ ] 夏天准备防汗喷雾和清凉喷雾
- [ ] 准备口气清新喷雾
- [ ] 准备浅色内衣裤
- [ ] 准备隐形增高鞋垫
- [ ] 准备腰带
- [ ] 准备方便穿脱的开襟衣服
- [ ] 提前一周理发

### 其他拍摄建议

- [ ] 前一晚早睡少喝水，可喝美式咖啡去水肿
- [ ] 携带零食（士力架/蛋白棒）补充体力
- [ ] 如有小宠物可带上拍摄
