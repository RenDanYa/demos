### 拍摄前一个月准备

- [ ] 拍摄前一个月开始运动减肥
- [ ] 拍摄前做好防晒，避免油腻辛辣食物，防止长痘
- [ ] 拍摄前 3 天染发（黑茶色偏棕），保持头发颜色统一
- [ ] 拍摄前在网上做攻略，确定自己喜欢的风格
- [ ] 拍摄前多练习微笑，保存好看的 POSE 学习
- [ ] 拍摄前多与摄影师沟通
- [ ] 提前准备好需要自带的道具
- [ ] 拍摄前做好身体和面膜护理补水

### 婚纱店选择

- [ ] 选婚纱店时先确定自己喜欢的风格再选店
- [ ] 到店后不要因为便宜而将就，多去后面看看婚纱
- [ ] 确认化妆品品牌，避免过敏
- [ ] 要求看客照，确认皮肤修图效果
- [ ] 对比婚纱店时重点看后期服务

### 拍摄前其他准备事项

- [ ] 拍摄前一晚早睡少喝水，避免水肿，早晨可喝杯美式咖啡促进排水
- [ ] 携带零食（士力架或蛋白棒）防止拍摄时间过长血糖低
- [ ] 有小宠物的话带上一起拍摄

### 拍摄前一到两天准备

- [ ] 拍摄前一晚保证充足睡眠，睡前不喝酒
- [ ] 拍摄前敷面膜、刮腋毛/胡须
- [ ] 不携带贵重物品，不提前化妆
- [ ] 准备乳贴或 NUBRA
- [ ] 拍摄前做好美甲
- [ ] 提前熨烫婚纱礼服、衬衫等防止褶皱
- [ ] 试穿婚鞋并准备创可贴、高跟鞋贴及润滑剂防磨脚
- [ ] 拍摄前洗头不使用护发素和发膜
- [ ] 男士拍摄前修鼻毛、刮胡子
- [ ] 避免带多余家属
- [ ] 准备好拍摄当天的早餐

### 男生拍摄物品准备

- [ ] 准备黑白船袜各一双，有彩色袜子也带上
- [ ] 准备黑色皮鞋和小白鞋各一双
- [ ] 准备白色备用衬衫
- [ ] 准备领带、领结等配饰
- [ ] 夏天准备防汗喷雾和清凉喷雾
- [ ] 准备口气清新喷雾
- [ ] 准备浅色内衣裤
- [ ] 准备隐形增高鞋垫
- [ ] 准备腰带
- [ ] 准备开襟等方便穿脱的衣服
- [ ] 拍摄前一周理发
