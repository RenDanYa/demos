### 服装造型确认

- [ ] 确认内外景可选择几个
- [ ] 确认婚纱、饰品能否任意挑选
- [ ] 确认一套服装能拍摄几个造型
- [ ] 确认化妆品、饰品、道具是否额外付费
- [ ] 确认摄影师、造型师等级及能否指定

### 拍摄服务确认

- [ ] 确认是否提供一对一服务
- [ ] 确认化妆师是否全程跟妆
- [ ] 确认有无拍摄花絮短片赠送
- [ ] 确认午餐、出行费用由谁承担
- [ ] 确认不满意能否当场重拍
- [ ] 确认拍摄夜景是否额外收费

### 底片精修确认

- [ ] 确认底片共多少张，是否全部赠送
- [ ] 确认底片提供格式为JPG还是RAW
- [ ] 确认能否先修几张预审，不满意能否修改至满意
- [ ] 确认精修数量是否满足入册需求

### 产品确认

- [ ] 确认产品是否包邮
- [ ] 确认是否赠送结婚登记照（有无精修和打印）
- [ ] 确认套餐含几本相册、几个相框，材质和尺寸如何

### 付款确认

- [ ] 确认尾款支付时间及定金金额
- [ ] 确认定金是否可退，是"定"还是"订"

### 时间安排确认

- [ ] 确认遇到恶劣天气是否可以改期
- [ ] 确认因天气原因改期是否需要支付费用
- [ ] 确认因个人原因改期需提前多久通知才无需支付费用
- [ ] 确认拍完后多久可以选片及精修照片多久能拿到

### 注意事项

- [ ] 确定好需要自费的项目，避免额外消费
- [ ] 将赠送产品和沟通细节写入合同
- [ ] 不要急着交定金，不要相信销售的"价格保留到今天"话术，满意可先交几百块并确保可退

### 砍价技巧

- [ ] 坚决不向商家透露预算底线
- [ ] 预算6000元先从7000元套餐入手谈，争取6000元拿下
- [ ] 让谈判强势的一方出面砍价，通常可砍500-1000元
- [ ] 价格降不了时争取多要精修，把用不上的产品换成精修
