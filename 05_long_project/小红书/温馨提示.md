### 拍摄顺序安排

- [ ] 四套服装按秀禾、主纱、礼服、轻纱顺序安排
- [ ] 主纱不放在首尾拍摄
- [ ] 第一套拍摄中式秀禾（优先室内，复古红或黄色纯色背景，准备道具）
- [ ] 第二套拍摄主纱（室内可选白灰色/简约光影背景；外景可选草坪/森系）
- [ ] 第三、四套拍摄礼服或轻纱，提前确定内景或外景

### 拍摄前沟通

- [ ] 挑选摄影工作室后当天确认是否建群服务
- [ ] 在群里问清选婚纱时间（拍摄当天还是提前，提前几天）

### 服装挑选

- [ ] 提前整理并告知礼服师自己想要的婚纱风格
- [ ] 遇到喜欢的服装向礼服师索要样片确认效果
- [ ] 慎选泡泡袖主纱，如喜欢优先选一字肩泡泡袖款式
- [ ] 避免选择主题/款式过于相似的服装，保证婚纱照风格多样

### 拍摄效果确认

- [ ] 每拍完一套服装现场查看原片效果，不满意及时要求重拍
- [ ] 拍摄中提醒摄影师注意男生的姿势与面部拍摄角度
- [ ] 修片确认时检查整体环境、服饰细节，不只关注脸部

### 选片

- [ ] 选片时直接删除不喜欢、相似、光线差、角度差、姿势重复的照片

### 时间规划

- [ ] 预留4套婚纱拍摄全天时间（化妆2h+拍摄4h+换装2h+午餐1h）
- [ ] 控制服装套数在4套左右，避免因过多导致疲惫影响成片
