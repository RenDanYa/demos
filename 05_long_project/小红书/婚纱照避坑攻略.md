### 预算与决策

- [ ] 提前规划婚纱照预算，并预留1000元左右的价格浮动空间
- [ ] 多看客片而非样片，作为选店参考
- [ ] 不要当场交定金，回家思考后再决定是否下单
- [ ] 根据自己喜欢的风格和预算选择工作室或影楼（工作室的风格偏自然、创意多、更有个
性，更尊重顾客想法，影楼拍摄风格单一，流水线拍摄，基地大，不用四处跑外面去拍，最后选择哪一种类型根据自己喜欢的风格和预算来决定）

### 价格与费用

- [ ] 尝试砍价，砍不动时争取换相框相册、精修、加服装、礼品、孕妇卡等
- [ ] 确认除套餐外是否还有其他消费（化妆品、道具、出行、景点门票、午餐等）

### 付款与退款

- [ ] 确认定金金额及尾款支付时间
- [ ] 确认因个人原因不拍了定金是否可以退

### 套餐产品

- [ ] 确认套餐具体包含哪些内容
- [ ] 确认相册、相框等产品的材质和尺寸
- [ ] 确认成品是否包邮
- [ ] 确认是否赠送结婚登记照并包含精修打印

### 底片与精修

- [ ] 确认拍摄底片数量及是否全送
- [ ] 确认是否有强制选片及加精修单价
- [ ] 确认精修张数及是否修到满意为止
- [ ] 确认底片是否提供 raw 和 jpg 双格式

### 时间与改期

- [ ] 确认恶劣天气是否可以改期
- [ ] 确认因自己原因改期是否需付费
- [ ] 确认拍完后多久选片、精修完多久拿成品

### 拍摄服务

- [ ] 确认是否一对一服务
- [ ] 确认摄影师/化妆师等级及是否可以指定
- [ ] 确认是否可以指定修图师
- [ ] 确认拍摄当天不满意是否可以重拍
- [ ] 确认选片不满意是否可以重拍

### 服装造型

- [ ] 确认是否可以试纱及是否收费
- [ ] 确认几服几造，男女服装数量是否一致
- [ ] 确认一套衣服是否可多背景拍摄
- [ ] 确认是否可以自带服装和要求造型
- [ ] 确认场地婚纱是否分区，实地查看是否有喜欢的款式
