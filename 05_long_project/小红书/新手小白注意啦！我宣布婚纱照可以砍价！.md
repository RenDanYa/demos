### 付款与合同确认

- [ ] 确认尾款支付时间和定金金额
- [ ] 确认定金是否可退，明确是"定"还是"订"
- [ ] 看客片并确认拍摄风格、角度、姿势是否符合喜好
- [ ] 将赠送的产品和沟通细节写进合同
- [ ] 确认需要自费的项目和额外消费
- [ ] 不着急付定金，警惕"价格只保留今天"等销售话术

### 砍价技巧

- [ ] 不透露自己的预算给对方
- [ ] 看高价套餐争取按预算价拿下（如6000预算看7000套餐）
- [ ] 让谈判强势的一方出面砍价，通常可砍500-1000元
- [ ] 砍不动价格时，争取多要精修张数
- [ ] 将用不上的产品（如摆台）换成精修(精修拿到手才是最值钱的)

### 底片与精修确认

- [ ] 确认底片数量及是否全送
- [ ] 确认底片格式是JPG还是RAW（RAW可后续自己找网上精修）
- [ ] 确认精修是否可预审，不满意是否可修改到满意
- [ ] 确认精修数量是否够入册，避免被动加精修

### 拍摄时间与改期确认

- [ ] 确认遇到恶劣天气是否可以改期
- [ ] 确认改期是否需要支付改期费
- [ ] 确认因个人原因改期需提前多久通知可免费用
- [ ] 确认拍完后多久可以选片、多久可以拿到精修

### 服装造型确认

- [ ] 确认内外景可以选择几个
- [ ] 确认参观的婚纱、饰品是否都可以随便选
- [ ] 确认一套服装可以拍几个造型
- [ ] 确认化妆品、饰品、道具是否需要额外收费
- [ ] 确认摄影师、造型师等级及是否可以指定

### 拍摄服务确认

- [ ] 确认是否提供一对一服务
- [ ] 确认化妆师是否全程跟妆
- [ ] 确认是否有拍摄花絮短片赠送
- [ ] 确认午餐、出行费用谁承担
- [ ] 确认不满意是否可以当场重拍
- [ ] 确认拍摄夜景是否需要加钱

### 产品交付确认

- [ ] 确认套餐包含几本相册、几个相框，并明确材质和尺寸
- [ ] 确认产品是否包邮到家
- [ ] 确认是否赠送结婚登记照，照片是否给精修打印
