### 女生准备

- [ ] 提前一周做头发（染发、护理），发色建议巧克力棕、蜜茶棕、黑茶
- [ ] 提前一两天做简约法式美甲，选纯色
- [ ] 拍照前刮腋毛
- [ ] 准备硅胶胸垫、无痕内裤、拖鞋、厚底鞋
- [ ] 佩戴日抛棕色系美瞳，带润眼液
- [ ] 前一晚做好面膜和保湿
- [ ] 提前一两个月健身塑型和体态管理
- [ ] 准备白色蕾丝中筒袜、湿巾、一次性吸管、高跟鞋磨脚贴

### 男生准备

- [ ] 提前一周剪发，不要过短
- [ ] 做好面部清洁和补水保湿
- [ ] 刮胡子、修鼻毛
- [ ] 准备隐形眼镜，避免眼镜反光
- [ ] 准备商务风手表
- [ ] 准备黑白船袜和中高腰袜
- [ ] 准备皮带
- [ ] 修剪指甲

### 季节物品

- [ ] 夏天准备防晒、遮阳伞、防蚊液、小风扇
- [ ] 冬天准备暖宝宝、保温杯、长款羽绒服、打底裤
