### 女生准备

- [ ] 提前一周做头发（染发、护理），发色建议巧克力棕、蜜茶棕、黑茶
- [ ] 提前一两天做简约法式美甲，选纯色
- [ ] 拍照前刮腋毛
- [ ] 准备硅胶胸垫、无痕内裤、拖鞋、厚底鞋
- [ ] 佩戴日抛棕色系美瞳，带润眼液
- [ ] 前一晚做好面膜和保湿
- [ ] 提前一两个月健身塑型和体态管理
- [ ] 准备白色蕾丝中筒袜、湿巾、一次性吸管、高跟鞋磨脚贴
- [ ] 提前一周做好面部清洁管理（清洁+补水）
- [ ] 前一晚洗头发，不用护发素
- [ ] 敏感肌自带护肤品化妆品

### 男生准备

- [ ] 提前一周剪发，不要过短
- [ ] 做好面部清洁和补水保湿
- [ ] 刮胡子、修鼻毛
- [ ] 准备隐形眼镜，避免眼镜反光
- [ ] 准备商务风手表
- [ ] 准备黑白船袜和中高腰袜
- [ ] 准备皮带
- [ ] 修剪指甲

### 季节物品

- [ ] 夏天准备防晒、遮阳伞、防蚊液、小风扇
- [ ] 冬天准备暖宝宝、保温杯、长款羽绒服、打底裤

### 拍摄前管理

- [ ] 提前一个月进行健身塑型和体态纠正
- [ ] 前一晚尽量10点前睡，保证第二天精神状态
- [ ] 拍照前两天不要开始剧烈运动减肥
- [ ] 拍照前一个月不要拔火罐、刮痧
- [ ] 前一晚不要熬夜
- [ ] 拍照前不要种睫毛、剪头发
- [ ] 拍照前不要尝试新的面部护理项目，避免过敏

### 拍摄当天准备

- [ ] 准备方便吃的零食（面包、巧克力等）补充体力
- [ ] 准备吸油纸
- [ ] 提前查看拍摄当天天气预报
- [ ] 根据天气准备防晒/防雨用品、外套或小风扇

### 拍摄沟通与避坑

- [ ] 提前和摄影师沟通拍摄风格（唯美，简约，文艺）、道具（烟花，鲜花）等想法
- [ ] 提前学习拍照姿势和表情管理，对着镜子练习
- [ ] 拍摄当天不要迟到
- [ ] 不要临时改变拍摄风格或妆造
- [ ] 前一晚不要吃辛辣油腻食物
- [ ] 前一晚不要喝太多水
- [ ] 不要佩戴携带贵重物品
