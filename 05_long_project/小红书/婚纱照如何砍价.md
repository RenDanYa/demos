### 砍价策略

- [ ] 和男朋友分工，让谈判强势的一方负责砍价
- [ ] 不要现场立即成交，必要时离开以争取更大优惠
- [ ] 支付小额定金时要求开具收据，并注明未选可退
- [ ] 尽量避开周末到店，选择人少时更方便讲价
- [ ] 到店询问减价活动
- [ ] 到店询问相框是否可升级加大
- [ ] 到店询问增加精修数量
- [ ] 到店询问当前套餐能否免费升级到更高一级套系
- [ ] 在报价优惠基础上继续砍价（比如说，你的心里预算是7000块，看的套系是7999，跟你说可以减少500到800不等+升级老师+多加精修片(总计精修张数最好50张)，可以再往下多砍一些金额，他说500，可以多砍500(这就要看你的胆子了)。他说800，那就多砍300到500。一般是可以砍掉800到1200的。）

### 线上筛选与预算

- [ ] 先在网上筛选合适的婚纱摄影店，再决定是否到店
- [ ] 网上联系时询问套餐分类、价格、活动及转介绍优惠
- [ ] 确定心里预算并按预算筛选套系
- [ ] 到店争取升级摄影老师
- [ ] 到店争取精修片总数达到50张
