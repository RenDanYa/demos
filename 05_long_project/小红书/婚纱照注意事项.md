### 选片注意事项

- [ ] 选片时果断删除光线、角度、动作、表情、闭眼、姿势重复不好的照片
- [ ] 选片时多留双人照，单人照各留两张，男士基本照保留
- [ ] 选片时少留背影照、剪影、半身照、远景照
- [ ] 选片时每个场景平均留几张，按精修张数分配到每套服装每个场景
- [ ] 选择相框时根据横竖版照片选择对应的相框
- [ ] 迎宾海报选全身内景照，床头挂相选外景横框照

### 男女拍摄前准备

- [ ] 女生拍摄前 3 天染发、2 天做美甲、1 天脱毛
- [ ] 男生拍摄前 2 天剪发、前一天晚上刮胡子修鼻毛
- [ ] 选片时和另一半一起去，问清隐形消费、底片张数、重拍政策、产品寄送和邮费
