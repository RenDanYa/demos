### 服装造型确认

- [ ] 确认内外景可以选几个
- [ ] 确认参观的婚纱、饰品是否都可以随便选
- [ ] 确认一套服装可以拍几个造型
- [ ] 确认化妆品、饰品、道具是否需要额外收费
- [ ] 确认摄影师、造型师等级及是否可以指定

### 拍摄服务确认

- [ ] 确认是否提供一对一服务
- [ ] 确认化妆师是否全程跟妆
- [ ] 确认是否有拍摄花絮短片赠送
- [ ] 确认午餐、出行费用谁承担
- [ ] 确认不满意是否可以当场重拍
- [ ] 确认拍摄夜景是否需要加钱

### 底片精修确认

- [ ] 确认底片多少张是否全送
- [ ] 确认底片格式是JPG还是RAW
- [ ] 确认精修是否可以修几张预审，不满意是否可以修改到满意
- [ ] 确认精修数量是否够入册

### 产品确认

- [ ] 确认套餐包含几本相册、几个相框及材质尺寸
- [ ] 确认产品是否包邮到家
- [ ] 确认是否赠送结婚登记照（是否精修、打印）

### 付款确认

- [ ] 确认尾款支付时间及定金金额
- [ ] 确认定金是否可退，是"定"还是"订"

### 时间安排确认

- [ ] 确认遇到恶劣天气是否可以改期
- [ ] 确认改期是否需要支付改期费
- [ ] 确认因个人原因改期需提前多久通知才不会支付费用
- [ ] 确认拍完后多久可以选片及精修多久可以拿到

### 注意事项

- [ ] 看客片，考察拍摄风格、角度、姿势是否喜欢
- [ ] 将赠送的产品、沟通细节写入合同
- [ ] 确认好需要自费的项目，避免额外消费
- [ ] 不要着急付定金，不要相信"价格只保留今天"话术，满意可先交几百块并确保可退

### 砍价技巧

- [ ] 不透露自己的预算给对方
- [ ] 预算3000元先看4000元套餐，谈到目标价，让强势的一方出面砍价（通常可砍500-1000元）
- [ ] 砍不动价格时多要精修，把用不到的产品换成精修
