## 一、前期准备

### 1.1 预算规划

- [ ] 确定婚纱照预算区间（建议在3000-6000元），预留1000元左右的价格浮动空间
- [ ] 确定心里预算并按预算筛选套系

### 1.2 风格与需求确认

- [ ] 拍摄前做攻略确定自己喜欢的风格→再选婚纱店
- [ ] 根据婚礼需求确定婚纱照拍摄套数→一般4套，外景1-2套即可
- [ ] 可选风格
	- [ ] 选定经典主纱风格（用于迎宾海报）
	- [ ] 选定新中式/喜嫁风格
		- [ ] 室内可选喜嫁风或汉服风
		- [ ] 室外可选新中式
	- [ ] 选定草坪外景/海边风格
	- [ ] 选定韩式简约风格
	- [ ] 选定落日夜景/夕阳风格
	- [ ] 特色风格（复古、胶片、油画、法式）按个人喜好选择
- [ ] 谨慎选择
	- [ ] 反串婚纱照
	- [ ] 工笔画风格（挑身材脸型且考验修图）
	- [ ] 情绪感风格（需颜值和表现力）
- [ ] 提前收集喜欢的拍摄风格和动作参考图发给摄影师和化妆师
- [ ] 第三、四套拍摄**礼服或轻纱**，提前确定**内景或外景**

### 1.3 店铺筛选与对比

- [ ] 先在网上筛选合适的婚纱摄影店→加客服咨询后再决定是否到店
- [ ] 对比婚纱店时重点看**后期服务**、门店**经营年限**、**工作室还是影楼**（根据自己喜欢的风格和预算选择）
- [ ] 到店实地查看婚纱**款式**、**干净程度**
- [ ] 到店后不要因为便宜而将就，多去后面看看婚纱

### 1.4 客片与样片考察

- [ ] 看客片并确认拍摄风格、角度、姿势是否符合喜好
- [ ] 确认有哪些风格并要求看真实客片
- [ ] 根据客片呈现效果选择风格
- [ ] 提前看客片、未精修原片和可选婚纱款式
- [ ] 多看客片而非样片，作为选店参考
- [ ] 重点参考客片效果，避免被样片误导
- [ ] 到店要求看客片
- [ ] 到店确认客片由店里摄影师拍摄
- [ ] 确认是否有该套餐的客照参考
- [ ] 确认是否有精修和底片对比图
- [ ] 要求看真实客片，确认摄影师、化妆师、修图师水平
- [ ] 对比多家时查看真实客片，优先参考身材/长相相似新人

## 二、谈判签约

### 2.1 砍价策略

- [ ] 不透露自己的预算和婚期给商家
- [ ] 看高价套餐按低价谈，不透露真实预算，让强势的一方出面砍价（通常可砍500-1000元）
- [ ] 按高价套餐的精修张数谈（如1万套餐68张，谈到8000时争取65张）
- [ ] 多看几家婚纱店，第二天再回访，用竞品套餐压价
- [ ] 尽量避开周末到店，选择人少时更方便讲价
- [ ] 到店前了解当前活动和优惠力度
- [ ] 到店询问减价活动
- [ ] 到店询问相框是否可升级加大
- [ ] 到店询问增加精修数量
- [ ] 到店询问当前套餐能否免费升级到更高一级套系
- [ ] 到店争取升级摄影老师
- [ ] 到店争取精修片总数达到50张
- [ ] 在报价优惠基础上继续砍价
- [ ] 按更高价位套餐标准砍相框
- [ ] 留意婚博会优惠券和朋友介绍优惠
- [ ] 能争取多送就多送赠品
- [ ] 砍不动价格时，争取多要精修张数
- [ ] 尝试砍价，砍不动时争取换相框相册、精修、加服装、礼品、孕妇卡等
- [ ] 预订前谈判争取更多底片、精修、服装和赠品
- [ ] 谈判时少做声，等销售介绍完再问问题
- [ ] 让家里谈判强势的一方出面砍价
- [ ] 不要现场立即成交，必要时离开以争取更大优惠
- [ ] 网上联系时询问套餐分类、价格、活动及转介绍优惠
- [ ] 争取多要一张登记证件照并含精修
- [ ] 稳住心态，谈不下可以走人

### 2.2 套餐内容确认

- [ ] 确认套餐内可拍摄几套服装及男女各几套
- [ ] 确认套餐内包含几个造型及男女各几个
- [ ] 确认套餐可拍摄风格及是否内外景结合
- [ ] 确认内外景可以选择几个
- [ ] 确认服装是否分价格区间及是否二次收费
- [ ] 确认套餐内衣服是否可以随意选择
- [ ] 确认手捧花是否包含在套餐内
- [ ] 确认套餐是否含夜拍、MV、鲜花费用，不含需加多少钱
- [ ] 确认套餐内有无其他收费（化妆品、道具、出行、景点门票、午餐等）
- [ ] 确认好需要自费的项目，避免额外消费
- [ ] 确认午餐、出行费用由谁承担
- [ ] 确认外景拍摄是否有额外费用（门票、停车、交通、午餐）
- [ ] 要求影楼承担外景交通并全程配车
- [ ] 确认产品是否包邮到家
- [ ] 确认赠品明细（相框、相册、摆台、登记照）
- [ ] 确认套餐包含几本相册、几个相框，并明确材质和尺寸
- [ ] 确认相框照片是否占用精修数量
- [ ] 确认精修张数、加精修单价及精修是否都入册
- [ ] 确认底片共多少张，是否全部赠送
- [ ] 确认底片格式为JPG还是RAW，优先RAW
- [ ] 确认底片交付时间和形式（网盘/邮箱/U盘）
- [ ] 确认精修是否可以修几张预审，不满意是否可以修改到满意
- [ ] 确认精修不满意的处理方式（能否重修或更换修图师）
- [ ] 确认拍完后多久可以选片及精修照片多久能拿到
- [ ] 确认拍完后能否当场看底片，不满意能否重拍
- [ ] 确认是否有强制选片
- [ ] 确认优惠活动截止时间
- [ ] 确认交了定金后的拍摄有效期
- [ ] 控制服装套数在4套左右，避免因过多导致疲惫影响成片
- [ ] 不升级套餐和相框，按基础套餐拍摄
- [ ] 用不到的套餐内容找商家替换（如摆台换精修）

### 2.3 合同条款

- [ ] 要求签订书面合同，所有商谈细节写进合同避免反悔
- [ ] 所有口头承诺必须写入合同
- [ ] 将套餐承诺（服装套数、精修张数、摄影师、重拍条款）写进合同
- [ ] 将赠品清单（相框、相册、摆台）写入合同
- [ ] 效果不满意免费重拍，写进合同
- [ ] 确认档期、化妆师、摄影师并写入合同
- [ ] 提前指定摄影师和化妆师，记住名字写进合同
- [ ] 合同中注明摄像师、化妆师是否分等级及对应费用
- [ ] 合同中写明照片是冲印还是喷墨打印
- [ ] 签订合同前仔细检查所有项目条款，核对相册相框尺寸
- [ ] 确认改期是否有费用（天气/不可抗力/个人原因/工作人员失误）
- [ ] 确认因个人原因不拍了定金是否可以退
- [ ] 拒绝隐性消费，提前确定所有价格并写入合同

### 2.4 付款与退款

- [ ] 确认定金金额、是否可退、退还政策及余款支付时间
- [ ] 确认尾款支付时间（最好拍完再付或拿到精修后付）
- [ ] 确认付款方式及是否有附加手续费
- [ ] 定金慎重支付，不一次性交完全款
- [ ] 不要急着交定金，警惕"价格只保留今天"等销售话术
- [ ] 支付小额定金时要求开具收据，并注明未选可退

## 三、拍摄准备

### 3.1 女生形象管理

- [ ] 拍摄前一周开始清淡饮食，前一晚尽量少吃
- [ ] 拍摄前一晚早睡并减少饮水，避免水肿

- [ ] 拍摄前一周坚持涂润唇膏
- [ ] 提前1-2天完成腋下及手臂脱毛
- [ ] 提前3-5天染发，推荐黑茶色或深棕色系，保持发色统一
- [ ] 提前1-2天做裸色或法式美甲，或备好裸钻穿戴甲
- [ ] 拍摄前一天洗头，不使用护发素和发膜
- [ ] 敏感肌自带护肤品和化妆品

## 四、拍摄执行

- [ ] 选择草坪/森系场景拍摄外景主纱
- [ ] 拍摄前涂抹防蚊液
- [ ] 拍摄前做好防晒
- [ ] 保持拍摄前一天饮食清淡，避免辛辣油腻食物

- [ ] 备齐拍摄当天需自备的物品
- [ ] 拍摄结束后及时卸妆并做好皮肤清洁
- [ ] 提前试穿主纱，检查舒适度与效果

- [ ] 查看摄影师实拍作品并沟通指定档位
- [ ] 要求合同中写明当天不满意可免费重拍
- [ ] 要求合同中写明重拍可换团队/场景及收费标准
- [ ] 要求一组服装可拍摄多个场景/背景
- [ ] 要求商家提供团队人数、是否 1 对 1 服务及成员构成
- [ ] 要求合同中写明团队等级及能否指定
- [ ] 要求合同中写明不满意可更换团队
- [ ] 要求商家说明本地拍与旅拍所需天数
- [ ] 要求合同中写明临时换装收费标准
- [ ] 要求影棚提供拍摄用花，否则自备
- [ ] 要求合同中写明男女服装套数及是否全场任选
- [ ] 要求商家列明可拍景点、场景及加景/换景收费标准
- [ ] 要求赠送拍摄花絮或小短片
- [ ] 查清是否存在强制消费或隐形消费
- [ ] 查清拍摄是否需要出车费
- [ ] 提前参观室内场景，核对是否全场通选
- [ ] 告知摄影师按用途拍摄：横版双人正面照、竖版全身显高照、横版大场景照、竖版甜蜜亲密照

### 4.3 现场应变

- [ ] 如遇恶劣天气，按预案改期或补拍
- [ ] 提前安排内景外景拍摄顺序与交通
- [ ] 落实外景门票、交通、停车、午餐费用的承担方
- [ ] 落实外景是否 1 对 1 服务
- [ ] 落实拍夜景/日出是否额外加钱
- [ ] 旅拍时落实是否包含住宿、接机及交通
- [ ] 慎选红色秀禾拍外景，优先浅色秀禾搭配中山装拍园林
- [ ] 配合摄影师口令眨眼与微笑
- [ ] 保持放松自信，拍摄时大胆表现

## 五、后期处理

### 5.1 选片策略

- [ ] 选片前设定预算与目标，拒绝选片师引导
- [ ] 选片时坚定不加精修
- [ ] 拒绝强制加片与诱导加片
- [ ] 选片时隔几天再决定是否加片
- [ ] 优先保留双人合照，单人照各留 1-2 张
- [ ] 少选背影、剪影、半身、远景及大全景
- [ ] 多选全身正面照与中近景照片
- [ ] 重复动作/场景只保留最好的一张
- [ ] 只选表情、光线、构图最佳的照片，不凑数
- [ ] 手或物品特写只保留最好的一张
- [ ] 保留明显个人特征，拒绝一键磨皮
- [ ] 提前告知修图师需保留的特征（如痣）
- [ ] 按相册排版预留精修张数，避免被动加片
- [ ] 查清加片单价并设定加片预算上限
- [ ] 查清赠送相册、摆台、相框数量
- [ ] 查清相册、相框材质与尺寸
- [ ] 查清是否赠送婚礼海报和电子相册
- [ ] 不升级昂贵相册，不额外加相册页数
- [ ] 按用途预留对应照片：迎宾海报/请柬/相框/床头/视频/手机壁纸

### 5.2 精修沟通

- [ ] 查看客片修图效果并提出皮肤质感要求
- [ ] 要求指定或优先安排修图师
- [ ] 要求先修几张预审，不满意可修到满意为止
- [ ] 要求初版精修不满意可免费修改
- [ ] 逐张检查精修细节、胖瘦与体态
- [ ] 检查精修的整体环境与服饰细节，不只关注脸部
- [ ] 拒绝流水线 P 图，按原五官比例微调
- [ ] 要求肤色均匀、整体提亮、气色自然
- [ ] 要求整体色调统一，避免发青发紫
- [ ] 要求色调和谐温馨，避免假白
- [ ] 要求光线正常曝光、光影过渡自然
- [ ] 要求背景颜色干净自然
- [ ] 要求去除背景杂物、穿帮及背景布褶皱
- [ ] 要求去除外景反光板、电线及背景多余人物
- [ ] 要求外景草地调色、肤色提亮
- [ ] 要求牙齿美白
- [ ] 要求修掉脏污、褶皱痕迹及裙摆周边干净
- [ ] 放大检查迎宾照及放大产品的裙摆与衣服破损
- [ ] 要求面部头部整体稍微修小，额头饱满、发际线补齐
- [ ] 要求脸型对称、五官立体轮廓
- [ ] 要求适当放大双眼、对称眼神提亮
- [ ] 要求鼻翼缩小、鼻梁适当拔高
- [ ] 要求淡化颈纹、眼角细纹，自然不抢镜
- [ ] 要求皮肤去除痘印、卡粉，干净清透且有纹理
- [ ] 要求人物整体瘦身但比例协调
- [ ] 要求身材比例适当拉长、腿部挺拔
- [ ] 要求瘦胳膊、锁骨立体、消除斜方肌和腰部赘肉
- [ ] 要求高低肩、颈部前倾、驼背自然调整

### 5.3 产品制作与交付

- [ ] 将产品严重色差可免费重制写入合同
