### 拍摄规划

- [ ] 根据婚礼需求确定婚纱照拍摄套数
- [ ] 预订前谈判争取更多底片、精修、服装和赠品
- [ ] 明确婚纱照用途（请柬、迎宾区、酒店水牌、婚房布置等）
- [ ] 确定拍摄周期，预留至少2个月，加急也需预留40天

### 谈判与沟通

- [ ] 精修优先，不要摆台/小相框/易拉宝，尽量换成精修
- [ ] 选片保持冷静，兼顾不同风格和横版竖版，能不加精修就不加
- [ ] 在既定服装数量下争取多做造型（如主纱做两套妆造）
- [ ] 争取多要一张登记证件照并含精修
- [ ] 前置沟通争取升级礼服区域
- [ ] 提前看好并指定摄影师和化妆师，记住名字写进合同
- [ ] 选定拍摄店铺后，提前了解不同档位摄影师及其作品
