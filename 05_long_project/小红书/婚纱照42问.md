### 定前必问

- [ ] 订婚纱照前确认服装是否分区、款式是否齐全
- [ ] 确认产品邮寄是自费还是包邮
- [ ] 确认邮寄产品损坏是否免费重新制作
- [ ] 确认产品出现严重色差是否免费重新制作
- [ ] 确认定金支付方式、金额及是否可退
- [ ] 确认婚纱照尾款支付时间（最好拍完付）
- [ ] 确认门店经营年限及是工作室还是影楼
- [ ] 确认套餐可拍摄风格及是否内外景结合
- [ ] 确认景点是否可以自由搭配、元素是否可全部拍摄
- [ ] 确认男女各多少套服装

### 套餐内容确认

- [ ] 确认拍摄套餐精修张数及是否全部入册
- [ ] 确认套餐是否包含微电影及微电影时长
- [ ] 确认产品相框、相册数量及是否可自己选择
- [ ] 确认是否有该套餐的客照参考

### 费用与拍摄安排

- [ ] 确认当前咨询是否有优惠活动
- [ ] 确认拍摄是否需要出车费
- [ ] 确认拍摄景点门票费是否需要承担
- [ ] 确认拍摄时间大约多久
- [ ] 确认超过拍摄时间是否需要加钱
- [ ] 确认拍摄团队餐费是否需要承担
