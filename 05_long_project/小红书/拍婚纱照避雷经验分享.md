### 主纱选择

- [ ] 选择有版型、有裙撑的 A 字版型或高腰线婚纱
- [ ] 不选裙摆脏的婚纱
- [ ] 不选超过三种材质的婚纱，首选缎面或纱面材质
- [ ] 不选长袖、带领子或泡泡袖婚纱
- [ ] 根据婚礼风格与身形选择合适的主纱
- [ ] 提前试穿主纱，确认舒适度与效果

### 套餐与预算避坑

- [ ] 不盲目选择太贵套餐，按预算合理选择
- [ ] 4套服装安排1-2套外景即可，避免过多外景
- [ ] 手捧花可选仿真花，不必额外购买鲜花
- [ ] 精修按套餐张数选择，剩余底片自己修，不加精修
- [ ] 不升级昂贵的相册产品
- [ ] 不做九宫格、十二宫格等额外排版
- [ ] 不额外加相册P页，避免选片师推销

### 风格选择

- [ ] 极简主纱选择纯色背景（纯白或渐变灰），避免重工婚纱
- [ ] 极简主纱拍摄1-3张正面带笑照片，用于迎宾海报
- [ ] 中式风格室内可选喜嫁风或汉服风，室外可选新中式
- [ ] 秀禾/龙凤褂背景简约、互动喜庆，避免繁复置景
- [ ] 森系草地/草坪婚礼风选择绿草坪外景，穿白色婚纱、轻纱或鱼尾
- [ ] 森系风格多互动抓拍，避免刻意摆拍和大太阳时段拍摄
- [ ] 特色风格（复古、胶片、油画、法式）按个人喜好选择，或选夜景/夕阳

### 避坑风格

- [ ] 慎选烟雾光影婚纱照，避免看不清人脸
- [ ] 秀禾外景慎选红色，可选浅色秀禾搭配中山装拍园林
- [ ] 红色礼服外景颜色扎眼，更适合室内纯色背景
- [ ] 情绪感风格需颜值和表现力，谨慎选择
- [ ] 工笔画风格挑身材脸型且考验修图，谨慎选择
- [ ] 谨慎选择反串婚纱照
- [ ] 避免选择只有氛围或人物过小的风景照

### 必问清单-底片

- [ ] 确认套餐内底片拍摄张数
- [ ] 确认底片是否全送
- [ ] 确认底片何时可以拷走及交付方式

### 必问清单-风格

- [ ] 确认有哪些风格可选
- [ ] 确认是否可以看客片
- [ ] 确认所有风格挑选是否免费

### 必问清单-服务

- [ ] 确认化妆师是否全程跟妆
- [ ] 确认天气原因调整档期的处理方式
- [ ] 确认拍夜景是否需要额外加钱
- [ ] 确认每组拍完后是否可以当场查看并调整

### 必问清单-团队

- [ ] 确认套餐内拍摄团队有几人
- [ ] 确认拍摄团队是否分等级
- [ ] 确认拍摄前是否会沟通风格和流程

### 必问清单-精修

- [ ] 确认精修张数
- [ ] 确认精修不满意如何处理
- [ ] 确认精修加一张的价格
- [ ] 确认精修照片是否可以重复入册

### 必问清单-服装

- [ ] 确认新郎新娘服装各几套
- [ ] 确认服装是否不分区、全场任选
- [ ] 确认每套服装是否搭配不同妆造
- [ ] 确认婚纱礼服尺码是否齐全

### 付款与注意事项

- [ ] 确认拍摄尾款支付时间
- [ ] 确认拍摄后是否需要支付服装清洁费
