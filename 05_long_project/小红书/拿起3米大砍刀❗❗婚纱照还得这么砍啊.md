### 价格谈判技巧

- [ ] 不透露自己的预算和婚期给商家
- [ ] 看高价套餐按低价谈（如1万套餐按8000谈）
- [ ] 留意婚博会优惠券和朋友介绍优惠
- [ ] 稳住心态，谈不下可以走人
- [ ] 能争取多送就多送赠品

### 服装造型与相框

- [ ] 谈服装时确保男女各4套，尽量往5套谈
- [ ] 相框材质选贵的好的，注意尺寸，主相框往大的挑
- [ ] 按更高价位套餐标准砍相框

### 底片精修

- [ ] 确认底片全送
- [ ] 按高价套餐的精修张数谈（如1万套餐68张，谈到8000时争取65张）
- [ ] 加精修后确认相册是否放得下，相应要求加页数
- [ ] 确认精修张数、加精修单价、精修是否都入册

### 费用与服务细节

- [ ] 确认外景门票、交通、餐费谁承担，要求影楼全程配车
- [ ] 确认套餐是否含夜拍、MV、鲜花费用，不含需加多少钱
- [ ] 旅拍问清楚是否含住宿、接机、交通
- [ ] 确认定金金额、是否可退及余款支付时间
- [ ] 确认产品寄送天数及邮费承担方

### 合同条款确认

- [ ] 效果不满意免费重拍，写进合同
- [ ] 将赠品清单（相框、相册、摆台）写入合同
- [ ] 确认是否赠送结婚登记照并写入合同
- [ ] 签订合同前仔细检查所有项目条款，核对相册相框尺寸

### 选片控制

- [ ] 拍完不加精修（除非摄影师技术非常好）
- [ ] 套餐精修张数能放满相册即可，无需额外加片

### 预算建议

- [ ] 控制预算在5-9K区间（非旅拍/知名摄影师）
