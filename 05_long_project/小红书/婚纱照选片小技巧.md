### 选片原则

- [ ] 多选中近景照片，少选远景
- [ ] 适当选氛围感好的照片
- [ ] 不选背影照
- [ ] 不选太多全景照片，尤其大全景
- [ ] 只选最好的照片，不凑数

### 选片细节补充

- [ ] 新郎新娘单人照各选 1-2 张即可，不宜过多
- [ ] 侧面照片少选，优化空间较少
- [ ] 删掉眼神不对焦、跑焦、手势尴尬的照片
- [ ] 姿势重复或相似的照片只保留最好的一张
- [ ] 选片前明确目的和需求，不被选片师牵着走
- [ ] 线上沟通敷衍时，到店面对面沟通修改
- [ ] 手或物品特写只选一张最好的即可
