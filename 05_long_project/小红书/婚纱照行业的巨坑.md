### 精修沟通要求-人物身形

- [ ] 要求人物整体瘦一些但比例协调
- [ ] 要求脸型对称、突出脸部立体轮廓
- [ ] 要求眼睛适当放大、大小对称、眼神提亮
- [ ] 要求鼻翼缩小、鼻子精致立体
- [ ] 要求鼻梁适当拔高
- [ ] 要求牙齿美白
- [ ] 要求高低肩调整
- [ ] 要求颈纹、眼角细纹消除
- [ ] 要求脸部细纹淡化自然不抢镜
- [ ] 要求皮肤去除痘印、卡粉，干净清透且有纹理感
- [ ] 要求保留明显个人特征，拒绝一键磨皮
- [ ] 要求头发柔顺有光泽、后脑勺饱满、发际线补充
- [ ] 要求头发修蓬松一些
- [ ] 要求身材比例适当拉长身高及腿部，身材挺拔
- [ ] 要求颈部前倾、驼背适当调整
- [ ] 要求面部头部整体稍微修小一些
- [ ] 要求额头饱满圆润、太阳穴适当填充、发际线补充
- [ ] 要求瘦胳膊、锁骨立体、消除斜方肌和腰部赘肉

### 精修沟通要求-服装处理

- [ ] 要求西装有型、婚纱礼服适当增加光线，打造立体感/质感
- [ ] 要求弄脏的地方P掉
- [ ] 要求皱褶痕迹尽量抹平
- [ ] 要求裙摆周边干净，迎宾照放大检查
- [ ] 要求衣服破损处放大检查
- [ ] 要求检查胸贴是否露出

### 精修沟通要求-背景处理

- [ ] 要求背景颜色修饰保持干净自然
- [ ] 要求色调和谐不刺眼、温馨柔和，避免假白
- [ ] 要求光线避免过曝过暗，正常曝光整体协调
- [ ] 要求光影过渡自然，避免出现不均匀现象
- [ ] 要求外景去除穿帮：反光板、电线、背景中多余的人及杂物
- [ ] 要求背景去除穿帮：破损、折痕、道具、服装线头
