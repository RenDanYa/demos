### 付款与优惠确认

- [ ] 到店前了解当前活动和优惠力度
- [ ] 确认优惠活动截止时间
- [ ] 确认交了定金后的拍摄有效期
- [ ] 确认付款方式，定金金额及收据/合同标注
- [ ] 确认定金退还政策（全退还是退多少）
- [ ] 确认尾款支付时间（最好拍完再付）
- [ ] 确认是否赠送结婚登记照及是否占用精修数量

### 到店与对比

- [ ] 到店实地查看婚纱款式、干净程度
- [ ] 多对比几家，不要表现出太喜欢

### 底片与重拍

- [ ] 确认套餐内底片数量
- [ ] 确认所有底片是否全部赠送
- [ ] 确认底片交付时间和形式（网盘/邮箱等）
- [ ] 确认底片格式（Raw 或 jpg）
- [ ] 确认拍摄不满意是否可以重拍
- [ ] 确认重拍能否更换团队或场景
- [ ] 确认重拍是否额外收费

### 精修相关

- [ ] 确认精修总数量
- [ ] 确认加一张精修的价格
- [ ] 确认精修是否会修到满意为止
- [ ] 确认精修是否可以重复入册
- [ ] 确认相框照片是否占用精修数量
- [ ] 确认不满意精修能否重修或更换修图师

### 场景与费用

- [ ] 确认内景外景是否同一天拍摄
- [ ] 确认所有景点是否都能拍、有无限制
- [ ] 确认外景地点门票和车费
- [ ] 确认主打风格及是否可随意挑选
- [ ] 要求看真实客片
- [ ] 确认拍夜景或日出是否加钱
- [ ] 确认内外景可以选几个
- [ ] 确认提前几天选衣服
- [ ] 确认鞋子是否需要自带
- [ ] 确认套餐内有无其他收费

### 团队与服务

- [ ] 确认是否一对一服务
- [ ] 确认拍摄前是否有时间与化妆师、摄影师沟通需求
- [ ] 确认化妆时间
- [ ] 确认拍摄时间
- [ ] 确认化妆师是否全程跟妆
- [ ] 确认拍摄过程中是否随拍随看
- [ ] 确认化妆师、摄影师等级及是否可指定
- [ ] 确认不满意是否可以更换团队

### 改期与选片

- [ ] 确认遇到不可抗力因素是否可以改期
- [ ] 确认改期是否有费用
- [ ] 确认拍完后什么时候选片
- [ ] 确认拍完多久出精修
- [ ] 确认多久可以收到成品

### 产品与交付

- [ ] 确认是否赠送拍摄花絮或小短片
- [ ] 确认拍摄当天是否管午饭
- [ ] 确认是否负责中午工作人员用餐
- [ ] 确认赠送相册、摆台、相框数量
- [ ] 确认产品不要可否优惠或换成精修
- [ ] 确认是否赠送婚礼海报和电子相册
- [ ] 确认相册是否包邮到家
- [ ] 旅拍确认是否含住宿、接机、交通

### 服装与造型

- [ ] 确认男女各几套衣服
- [ ] 确认服装和饰品是否有分区、可否随便选
- [ ] 确认服装是否分价格区间及是否二次收费
- [ ] 确认想要拍摄的样片服装是否确定有
- [ ] 确认提供几个妆容造型及换妆规则
