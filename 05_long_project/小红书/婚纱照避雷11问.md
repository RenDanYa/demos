### 拍摄场景与天数确认

- [ ] 订婚纱照前问清内外景分别多少个、分别是什么、能拍哪些
- [ ] 确认当天是否可以参观婚纱场景
- [ ] 确认加景、换景是否额外收费及收费标准
- [ ] 确认本地拍和旅拍分别需要多少天
- [ ] 确认一天拍摄服装不超过 5 套

### 样片与客片

- [ ] 确认哪些是样片、哪些是客片、哪些是店里摄影师拍的
- [ ] 要求多看真实客片，别被样片迷住

### 套餐与服装数量

- [ ] 了解套餐结构，判断套餐是否合适，不被附加礼包迷惑
- [ ] 确认新郎、新娘服装分别多少套，分清男女套数

### 服装选择与造型

- [ ] 确认服装是否分价格区间、可否随意挑选、是否二次收费
- [ ] 确认想要拍摄的样片服装是否确定有
- [ ] 确认提供几个妆容造型及换妆规则

### 化妆品与饰品

- [ ] 确认安瓶、指甲油、隐形吊带等是否免费提供及价格
- [ ] 确认假睫毛、胸贴等是否收费

### 道具与服务

- [ ] 确认鲜花、拍摄道具使用限制及是否存在加价
- [ ] 确认拍摄团队有哪些人、是否全程一对一
- [ ] 确认摄影师、化妆师是否可以指定
- [ ] 确认不满意是否可以更换团队
