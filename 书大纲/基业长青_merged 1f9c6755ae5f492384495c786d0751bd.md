- 马克·吐温名言：”我往往是在机会离去时，才明白这是机会。”

- 研究团队：詹姆斯·柯林斯（斯坦福商学院学者）、杰里·波勒斯（斯坦福大学副校长）

- 研究对象：18家高瞻远瞩公司 + 18家对照公司（选自《财富》500强工业企业和服务类公司）
- 研究目的：
    1. 找出高瞻远瞩公司共同拥有的基本特质和动力

### 3.1 大胆的愿景与行动（15:01-20:00）

### 四、市场营销的三个层次（30:01-40:00）

- 菲利普·科特勒理论：
    1. 回应式营销：满足现有需求（如洗衣机、洗碗机）

- 区别于”成功公司”：更强调制度延续性与价值观驱动

- 代表企业：3M、波音、迪士尼、惠普等18家1950年前创立的公司

### 2. 关键研究发现（无时间戳）

### 2.1 破除12个管理迷思（无时间戳）

| 迷思 | 事实 |
| --- | --- |
| 依赖魅力型领导者 | 内部培养管理者占比超对照公司6倍 |
| 追求利润最大化 | 核心价值驱动，利润为附属结果 |
| 需”正确”价值观组合 | 无统一标准，但执行一致性极强 |
| 战略规划决定成功 | 关键突破多来自试错与”进化论”式迭代 |

### 2.2 核心能力模型（无时间戳）

🔄 **逆境恢复力**

- 迪士尼（1939年）、波音（1970年代）等均经历濒临破产后重生

- 1926-1990年投资回报：高瞻远瞩公司股票基金收益为大盘指数15倍

### 3. 研究方法论（无时间戳）

### 3.1 研究设计（无时间戳）

- **对照组选择**：与高瞻远瞩公司同期创立、起点相似的优秀企业（如先力时vs摩托罗拉）
- **数据规模**：分析36家公司90年历史，3000+份文件，6万+页资料

### 3.2 研究步骤（无时间戳）

1. 确定研究对象（CEO问卷提名+1950年前创立筛选）
2. 历史对比分析（1700年CEO任职数据+财务年报追踪）
3. 跨学科验证（生物学进化论+组织行为学）

### 4. 实践启示（无时间戳）

✅ **企业建设关键点**

- 构建”保存核心+刺激进步”的双引擎机制

- 价值观践行比内容本身更重要

- 长期主义需兼顾稳定性与创新试错

### 一、核心观点：造钟 vs 报时（00:00-05:00）

🔧 **定义与区别**
- **报时**：依赖个人魅力或单一产品构想（如德州仪器的创业产品）
- **造钟**：构建可持续的组织制度与文化（如通用电气实验室制度）
- **关键结论**：高瞻远瞩公司的终极创造是组织本身，而非产品或领袖

### 二、破除迷思1：无需”伟大构想”（05:01-15:00）

| 公司 | 创业初期状态 | 成功关键 |
|————|—————————————|—————————|
| 惠普 | 500美元资本，生产保龄球指示器等产品 | 持续实验与组织进化 |
| 索尼 | 电饭煲失败，靠电热毯维持生计 | 组织学习能力 |
| 沃尔玛 | 无零售构想，从杂货店起步 | 制度创新（店中店模式） |
| 美利坚百货 | 初始乡村折扣构想清晰 | 依赖单一战略导致僵化 |

### 三、破除迷思2：魅力型领袖非必需（15:01-25:00）

- **对照组领袖**：
- 先力时 小尤金·麦克唐纳：个人集权，缺乏传承机制

原文：[基业长青_06_章节_6](https://www.yitulu.com/t/pdf/rPPiwjZf)

### 1. 核心理念的定义与重要性

- **核心理念=核心价值+目的**
    - 核心价值：组织长存的根本信条，如诚信、创新（不可因短期利益妥协）
    - 目的：赚钱之外的根本原因，如“保存和改善生命”（默克）、“科技造福大众”（索尼）
- **关键作用**：指导决策、凝聚员工、区分高瞻远瞩公司与对照公司

### 2. 利润与核心理念的关系

- **利润的定位**
    - 重要但非终极目标，而是“生存的必要条件”（默克、惠普）
    - 案例：默克免费赠送“美迪善”药物对抗河盲症，短期无利润但坚守“拯救生命”使命
- **对比案例**
    - 默克vs辉瑞：默克以“价值观与梦想”为百年史书名，辉瑞早期无类似理念
    - 福特vs通用：福特坚持“人员>产品>利润”，通用更侧重财务目标

### 3. 高瞻远瞩公司的实践案例

### 3.1 默克（制药）

- **核心理念**：“药品旨在救人，利润随之而来”（乔治·默克二世）
- **行动体现**
    - 二战后免费向日本引入链霉素消灭肺结核
    - 1989年《内部管理方针》强调“以改善生命为衡量标准”

### 3.2 索尼（科技）

- **核心理念**：“先锋精神”（探索未知、科技造福）
- **创业初期案例**
    - 1946年制定《公司说明书》，明确“推动科技进步”目标，尽管当时仅7名员工、1600美元资金

### 3.3 福特（汽车）

- **核心理念**：“人员第一，产品第二，利润第三”（唐·皮特森）
- **经典实践**
    - 1916年推出T型车，降低价格58%让大众买得起，同时提高工人日薪至5美元

### 4. 核心理念的落地与传承

- **实施方法**
    - 写入公司章程（如强生《我们的信念》）
    - 高管以身作则（如默克CEO魏吉罗强调“科学家士气优先”）
    - 持续培训与文化渗透（索尼“先锋精神”传承50年）
- **常见挑战**
    - 短期利润压力与长期理念的冲突（如德州仪器纯财务导向导致战略短视）

### 1. 核心理念与进步驱动的辩证关系（00:00-05:30）

🔑 **核心观点**

- 高瞻远瞩公司需同时保护核心价值与推动持续变革

- 核心理念（如惠普”尊重员工”、沃尔玛”超出顾客期望”）是恒定基础

- 非核心做法（如福利形式、服务方式）需灵活进化

### 2. 自我革新的本质与边界（05:31-12:15）

📈 **关键原则**

- 自我革新=用不同方式做事，但需坚守已验证的基本原则

- 案例：

- 3M”15%时间自由研发”政策（可调整的非核心做法）

- IBM因固守非核心策略（如大型计算机业务）导致衰落

### 3. 进步驱动力的特征（12:16-20:45）

🚀 **内在属性**

- 源自人类探索、创造、改善的深层冲动

- 表现为：

- 主动设定大胆目标（如迪士尼建乐园、波音造747）

- 持续自我批判与改进（如马利奥特”每天有价值”哲学）

- 容忍创造性失败（3M防水砂纸、便利贴的诞生）

### 4. 核心与进步的协同机制（20:46-30:20）

⚙️ **阴阳平衡模型**

| 核心理念（阴） | 进步驱动力（阳） |

|———————-|————————–|

| 提供稳定基础与方向 | 促进演化与实验 |

| 明确不可变的核心价值 | 推动战略/战术/产品迭代 |

| 内部培养契合理念的人才 | 外部引入新方法/技术 |

### 5. 落地执行的五大方法（30:21-40:10）

📋 **实践框架**

1. **胆大包天的目标**：如福特”汽车大众化”、索尼”晶体管商业化”

2. **宗教式文化渗透**：迪士尼大学培训、诺世全服务奖惩制度

3. **多方实验与试错**：3M内部创业基金、波音747风险项目

4. **内部晋升经理人**：确保核心价值观传承（宝洁、惠普案例）

5. **永远不够好机制**：六西格玛标准、持续改进程序

### 一、胆大包天目标的定义与价值（00:00-05:30）

🚀 **核心定义**

- 指通过设定远超当前能力范围的挑战性目标，激发组织潜能，推动突破性创新

- 西奥多·罗斯福（1899年）：”尝试伟大的事情，赢取光荣的胜利，即使遭遇失败，也远胜过与既不享受多少东西、也不承受多少痛苦的可怜虫为伍”

### 二、经典企业案例分析（05:31-25:00）

### 2.1 波音公司：从军工到民航的跨越（05:31-10:15）

- **关键决策**
    - 1952年：投入1500万美元研发波音707喷气式客机，赌上公司净值的1/4

### 1. 进化论与企业进化（00:00-05:30）

🔬 **核心观点**

- 达尔文《物种起源》核心：”增生、变化，最强者生存、最弱者死亡”（1859年）

- 企业进化类比生物进化：通过**随机变异（实验）** 和**自然选择（市场验证）** 实现迭代

- 与”胆大包天目标”的差异：

- 传统目标：明确方向的跳跃式进步

- 进化式进步：渐进式试错，意外涌现战略

### 2. 经典企业进化案例（05:31-20:00）

### 2.1 强生：意外的消费产品帝国（05:31-09:00）

- 1890年：因医生投诉药膏刺激皮肤，附赠滑石粉缓解不适
- 消费者直接购买滑石粉→推出”婴儿爽身粉”，占营收44%
- 核心策略：**小步骤试错+保留有效创新**（如”邦迪”创可贴源于员工应急发明）

### 2.2 马利奥特：从餐厅到航空餐巨头（09:01-13:00）

- 1937年：第8号餐厅发现旅客打包需求→主动拜访东方航空
- 创新业务：机场跑道餐盒配送，扩展至22班/日航班
- 关键动作：**将异常客户需求转化为新业务线**

### 2.3 3M：创新机器的进化之路（13:01-20:00）

- 转型关键：从采矿失败→砂纸→隔离胶带→思高胶带→如意贴
- 组织机制：
    - “15%自由时间”鼓励员工实验
    - “抽枝剪枝”原则：保留有效创新，淘汰失败项目
    - 对比诺顿公司：因缺乏实验机制，从行业领先沦为被收购

### 3. 进化式进步的核心机制（20:01-35:00）

### 3.1 变异：创造多样性（20:01-25:00）

- 来源：员工首创精神（如3M德鲁发明隔离胶带）、客户问题（如强生药膏投诉）
- 方法：**容忍”无目的的变化”**，如3M鼓励”科学的娱乐精神”

### 3.2 选择：市场验证（25:01-30:00）

- 标准：是否解决真实问题+符合核心理念
- 案例：3M如意贴因教堂唱诗班书签需求诞生，经15%自由时间实验成功

### 3.3 保留：制度化创新（30:01-35:00）

- 3M机制：
    - 设立实验室（1914年投入500美元建首个实验室）
    - 管理层承诺：”失败是最重要的产品”（小约翰逊）

### 4. 企业进化的五大教训（35:01-50:00）

1. **快速行动**：”试一试，而且要快！”（3M麦克奈特）
2. **接受失败**：强生107年无亏损，却经历多次产品失败（如可乐兴奋剂）
3. **小步骤迭代**：思高胶带从隔离胶带→防水包装→透明胶带渐进演变
4. **分权与空间**：3M允许员工”自由支配15%时间”，培养创新文化
5. **机制化创新**：如3M”30%新产品目标”强制创新，诺顿因缺乏机制衰落

### 5. 核心理念与进化平衡（50:01-60:00）

⚖️ **关键原则**

- 进化≠无方向多元化：3M坚守”解决人类真实问题”，淘汰偏离核心理念的创新

- 核心理念=遗传密码：如沃尔玛”顾客价值”、3M”创新改善生活”

- 案例对比：

- 成功：3M、强生（进化+核心理念双轨并行）

- 失败：诺顿、宝来（固守传统或失去核心）

2. **注重信号细节**：如惠普”剪锁链”事件传递尊重员工理念
3. **集中力量放大**：建立相互强化的机制（如默克的多维度研发支持）
4. **坚持自我道路**：敢于违背行业惯例（如惠普拒绝市场占有率优先）
5. **消除不协调点**：持续修正与核心理念冲突的制度/结构
6. **创新不忘守恒**：保存核心价值同时激励进步

🎯 **四大关键结论**（45:01-50:00）
- 做造钟师而非报时人：构建制度而非依赖个人
- 拥抱融合法：平衡保守与变革
- 保存核心+刺激进步：双元驱动
- 追求持续一致：长期坚持协调原则

- 火星小组模式：6-15人（12人最佳）的”遗传密码”代表组

### 二、组织协调与进步机制（15:31-35:45）

- 胆大包天目标（如波音模式）
- 多方尝试淘汰机制（如3M模式）
- 持续自我改进系统（如摩托罗拉模式）

### 2.2 组织协调的关键动作

- 消除不协调点：每年至少6项具体改变措施

### 2.3 中层管理者的实践路径

- 部门级核心理念建立：电脑公司制造部门的文化革新案例
- 15%规则移植：在部门内实施类似3M的创新激励机制

### 三、经典案例与21世纪适应性（35:46-55:20）

### 3.1 企业案例分析

- 核心不变性：默克、宝洁等百年企业的根本因素持续性
- 适应性策略：扁平化管理、员工自主权、多方尝试机制

### 四、常见问题解答（55:21-60:00）

### 4.1 非CEO角色的行动指南

- 成为”造钟者”：建立流程而非依赖个人英雄
