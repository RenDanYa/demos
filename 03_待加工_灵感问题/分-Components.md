# 任务打卡和笔记属性打卡的区别（2026-01-21）

| 字段 |                  任务属性                  |                         笔记属性                          |
|:----:|:------------------------------------------:|:---------------------------------------------------------:|
| **功能** |                    提醒                    |                        不需要提醒                         |
| **性质** |           主动规划事件，频率固定           |                     意外事件→被迫中断                     |
| **介入** |             事前规划，到点提醒             |                    事后记录，无法预判                     |
| **目标** | 记录数据，分析和目标的差距，尽量稳定、执行 | 记录数据，分析变化趋势，尽量减少/可控，或者一个周期内稳定 |
|**事件**|阅读，播客，运动，待办，模型，打字，午休，刷牙，洗澡，鱼油，周复盘|吃，做饭，洗澡，打电话，打扫|
