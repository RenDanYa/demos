| 特性        | 出链 (Outbound Links)                       | 反链 (Backlinks / Inbound Links)            |
| :-------- | :---------------------------------------- | :---------------------------------------- |
| **定义**    | **从当前网页主动指向外部其他网页的链接。** (你链出去)            | **从外部其他网页指向当前网页的链接。** (别人链向你)             |
| **视角**    | 站在**链接来源页面**的角度看 (即当前页面 A → 其他页面 B)       | 站在**链接目标页面**的角度看 (即外部页面 C → 当前页面 A)       |
| **相对性**   | 同一个链接，对**链接来源页**来说是**出链**。<br>(A→B：对A是出链) | 同一个链接，对**链接目标页**来说是**反链**。<br>(A→B：对B是反链) |
| **控制权**   | 当前页面的创建者**有完全控制权**（可以决定链向哪里）。             | 当前页面的创建者**没有直接控制权**（依赖他人主动链接）。            |
| **简单比喻**  | **当前页面伸出手指指向别人** (“看，他/它在那边”)             | **别人伸出手指指向当前页面** (“看，他/它值得关注”)            |
| **一句话核心** | **我链出去的**                                 | **别人链向我的**                                |
