## 一、准则一：工作要深入（系统化深潜设计）

### 1. 四大深度工作哲学流派（按需选择）


| 深度工作模式 | 核心特点 | 适配人群 |
| ------------ | -------- | -------- |
| **禁欲主义** | 彻底排除浮浅工作，极致专注核心目标 | 独立创作者 |
| **双峰哲学** | 明确划分深度/开放时段（可按年/周/日灵活划分） | 需要平衡深度思考与社交的知识工作者 |
| **节奏哲学** | 设定每日固定时段（比如清晨）启动深度工作，形成习惯链条，降低意志力消耗 | 大多数普通人，想要平稳养成深度习惯 |
| **记者哲学** | 利用一切碎片时间随时插入深度工作 | 日程紧凑、自控力极强的人士 |


### 2. 习惯化与仪式感（降低启动阻力）


| 深度专注策略 | 具体做法举例 |
| --- | --- |
| 设定固定场所、结构化工作流程 | 比如在专属空间工作、任务进行时禁用网络 |
| 提前准备支持系统 | 提前准备好提神咖啡、整理好所需资料 |
| 用「大手笔」强化重视 | 通过投资环境/工具（比如预定专注酒店）拉高任务 stakes，让大脑不敢轻视 |


### 3. 像经商一样执行（4DX框架）


| 步骤编号 | 核心原则 | 具体说明 |
| -------- | -------- | -------- |
| 1 | 聚焦极端重要目标（EIG） | 避免目标泛滥 |
| 2 | 抓住引领性指标 | 关注每日深度工作小时数等引领性指标，而非最终产出量等滞后性指标 |
| 3 | 建立视觉化计分板 | 即时反馈深度工作进度 |
| 4 | 保持自我问责 | 定期进行复盘仪式 |

### 4. 学会「图安逸」


| 策略 | 核心原理 | 具体实施 |
| ---- | -------- | -------- |
| 利用无意识思维 | 彻底放松让大脑后台处理复杂模糊的难题 | 基于「无意识思维理论」 |
| 执行停工仪式 | 切断「蔡加尼克效应」带来的注意力残留 | 例如梳理明日计划，保障夜间专注力彻底复原 |

## 二、准则二：拥抱无聊（重塑大脑专注回路）

| 策略 | 核心观点 | 具体做法 |
| ---- | -------- | -------- |
| 计划分心而非躲避分心 | 网络本身不完全损害专注，损害专注的是「稍感无聊/遇挑战就立刻转向高刺激活动」的条件反射 | 提前规划好网络使用时段，非规定时段必须忍受无聊，以此锻炼注意力控制力 |


| 训练方法 | 名称 | 操作说明 |
| -------- | ---- | -------- |
| 高强度训练法 | 罗斯福冲锋 | 锁定高优先级任务，设定极度严苛的截止时间，倒逼大脑快速进入极限专注状态 |
| 高强度训练法 | 有成果的冥想 | 在散步、淋浴这类无需消耗脑力的身体活动中，强制大脑聚焦解决一个具体专业难题，遵循固定操作路径：考察变量 → 确定具体子问题 → 巩固收获并循环训练 |
| 高强度训练法 | 记忆力训练 | 例如记忆扑克牌这类练习，通过构建空间联想完成高强度视觉聚焦训练，间接提升深度工作的能力 |

## 三、准则三：远离社交媒体（重建数字时代的注意力秩序）


| 策略 | 核心原则 | 具体做法 |
| ---- | -------- | -------- |
| 工具选择的哲学革命 | 摒弃「任何益处法」的妥协思维，拥抱「手艺人文主义」 | 主动评估工具，仅保留对核心目标产生巨大积极影响的工具 |
| 关键少数法则（80/20原则） | 识别个人职业/生活的2-3个绝对核心目标 | 严格审视现有工具是否直接服务这些目标 |
| 注意力重置实验 | 突然停用社交媒体30天 | 事后反思：生活质量真的受影响吗？是否有人真正在意你的缺席？ |
| 高质量娱乐替代 | 用需要专注的高质量休闲替代算法投喂的碎片化网络娱乐 | 例如阅读名著、深度面对面交流 |

## 四、准则四：摒弃浮浅工作（极致的时间与精力管理）


| 策略 | 核心方法 | 具体说明 |
| ---- | -------- | -------- |
| 全面接管时间（工作日程规划法） | 为一天的每一分钟安排具体任务 | 将时间划分为方格（每格≥30分钟）；主动预留「缓冲时间」「弹性方格」应对突发状况，保持动态调整 |
| 深度与浮浅边界测量 | 用「刚毕业大学生需多久学会这项任务」评估工作深度 | 需要数月以上为深度，数周以内为浮浅 |
| 严格预算与固定日程 | 控制浮浅工作时间比例，实施固定日程生产力模式 | 和上级/团队协商，把浮浅工作的时间比例严格控制在30%-50%以内；设定坚如磐石的每日下班时间，用截止期限倒逼自己对非核心事务说「不」 |

### 🎯 核心结论/黄金法则


| 法则名称 | 核心观点 | 具体说明 |
| -------- | -------- | -------- |
| 注意力体验法则 | 你所关注事物的概括，就是你的人生体验 | 深度工作不仅是提升经济产出的生产力工具，更是过滤负面琐事、获取「心流」与最优心理体验的核心人生策略 |
| 工具主奴法则 | 社交媒体本质是私人公司设计的注意力捕获器 | 深度工作者必须用「工具仆人」思维替代「工具奴隶」状态，做到「非必要不使用，必使用则限时」 |
| 专注力肌肉法则 | 专注力不是可以随时调取的态度，而是必须像肌肉一样持续训练的生理习惯 | 如果不同时戒断对分心的渴望，一切深度专注训练都将大打折扣 |

### 🛠️ 实操指南与行动步骤


| 步骤 | 策略 | 具体操作 |
| ---- | ---- | -------- |
| 1 | 实施每日规划与停工仪式 | 每天清晨用「时间方格表」规划全天每一分钟；每天傍晚执行严格的「停工清点」，在清单打勾并说服大脑「工作已追踪完毕」，随后彻底切断工作联系 |
| 2 | 建立专属深度计分板 | 在办公桌最显眼处张贴卡片，只记录一件事——「每日处于无干扰深度工作状态的小时数」，每周日晚做简短复盘 |
| 3 | 推行「断网隔离与定量供给」 | 工作时放一个记事本，规定下一个可以联网/看手机的具体时间（比如45分钟后），此时段内即便工作卡壳也绝对不碰手机，培养大脑对「无聊」的耐受力 |
